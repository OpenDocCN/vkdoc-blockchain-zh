# 12. 总结

在本节书中，我以简单的方式介绍了当今区块链最常见的用例。我的想法是保持简洁，直击要点。本书提供的所有示例旨在让您在学习道路上快速前进。作为一名开发者，我知道最佳学习方式就是编写代码，因此我希望与您分享这种学习方法。

学习如何成为一名区块链开发者充满挑战。这个领域日新月异，要作为一名专业人士跟上时代，需要花费大量时间和精力。不要让这吓倒您。我们都从零开始，通过一步步学习来掌握新技能。希望本书能为您提供入门所需的知识。

掌握了这些知识，您现在可以创建自己的加密货币，或铸造自己的 NFT 并在`OpenSea`上出售，甚至可以在`IPFS`上托管您的第一个站点并使用`ENS`的自定义域名。您还可以通过`Chainlink`预言机查询链下数据，或使用`Filecoin`以分布式方式保存数据。可能性是无限的。

请记住，区块链正越来越受欢迎，它已不再局限于比特币和以太坊。未来，我们将看到许多项目发展壮大，而另一些项目则会失去空间。但永远不要忘记，技术是为了解决实际问题而存在的。要应用你在本书中学到的知识，首先要思考你试图解决什么问题，以及这些问题是否更适合用链下解决方案来解决。归根结底，区块链将帮助你解决与信任相关的问题。中心化的数据源值得信赖吗？中心化的应用程序值得信赖吗？如果答案是否定的，那就选择区块链。

希望你喜欢阅读这本书；最后但同样重要的是，永远不要停止学习。期待不久后能见到你成为一名区块链开发者。
