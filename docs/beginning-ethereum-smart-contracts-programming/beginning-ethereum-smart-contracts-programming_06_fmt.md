# 工作量证明（PoW）

`PoW`消耗了巨大的计算资源。需要`GPU`（以及一些区块链的`ASIC`，即专用集成电路），而`CPU`速度并不重要。它还消耗大量电力，因为矿工们反复做同样的工作，即寻找`nonce`以使区块满足网络难度。

一个常见的问题是，为什么挖矿需要使用强大的`GPU`而不是`CPU`。简单比较一下：一个`CPU`内核每个时钟周期可以执行`4`条`32`位指令，而像`Radeon HD 5970`这样的`GPU`每个时钟周期可以执行`3200`条`32`位指令。简而言之，`CPU`擅长对少量数据进行复杂操作，而`GPU`擅长对大量数据进行简单操作。由于挖矿完全是执行哈希运算和寻找`nonce`，这是一项高度重复的任务，正是`GPU`所擅长的。

> **提示**
> 
> 当矿工成功挖出一个区块时，他们会获得挖矿奖励以及交易手续费。这激励着矿工投资于挖矿设备并使其全天候运行，从而产生高昂的电费账单。
