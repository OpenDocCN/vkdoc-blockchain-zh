# 5. 使用 MetaMask 加密货币钱包

在上一章中，你学习了如何使用 `Geth` 创建自己的私有以太坊测试网络，以便你可以尝试各种以太坊交易，例如向不同账户转账以及执行挖矿。你还学习了如何创建账户，以便持有自己的以太币。在本章中，你将学习如何使用一个名为 `MetaMask` 的 Chrome 扩展程序。`MetaMask` Chrome 扩展程序是一个以太坊加密货币钱包，它允许你持有你的以太坊账户；并且在接下来的几章中，它是帮助你开发和测试智能合约的重要工具。

## 什么是 MetaMask？

`MetaMask` 是一个非常有用的工具，在你进军区块链世界的过程中扮演着关键角色。与其试图用一段话来定义 `MetaMask` 究竟是什么，不如让我们在后续内容中讨论 `MetaMask` 所扮演的角色。

首先，`MetaMask` 是一个以太坊加密货币钱包。它允许你执行以下操作：

*   创建用于各种以太坊网络的账户
*   维护你账户的私钥，以便你可以导出私钥或导入新账户
*   在各种以太坊网络之间切换，以便你的账户能为每个网络显示正确的余额
*   在账户之间执行交易
*   从一个账户向另一个账户转移以太币
*   在你的 `MetaMask` 账户中持有代币
*   在区块链浏览器 Etherscan 上查看详细的交易信息
*   在去中心化交易所（DEX）上进行以太币和代币之间的兑换

**注意** 除了是以太坊钱包之外，`MetaMask` 还允许你通过注入一个由以太坊核心团队开发的名为 `web3.js` 的 JavaScript 库来与以太坊区块链进行交互。我将在下一章讨论智能合约时对此进行更详细的说明。

### MetaMask 的工作机制

在幕后，`MetaMask` 连接到一个名为 INFURA (`https://infura.io/`) 的服务器。INFURA 维护着连接到各自以太坊网络的以太坊节点。INFURA 并没有持有你的私钥（将私钥存放在由第三方持有的云端总是一种安全风险），而是允许 `MetaMask` 将你账户的私钥保存在你的本地计算机上，并且仅将交易中继到网络。图 5-1 展示了 `MetaMask`、INFURA 和以太坊网络之间的关系。

![](img/471881_2_En_5_Fig1_HTML.jpg)

一个以太坊网络的圆形示意图，其轨道上有一个 Infura 节点。它有一个 `MetaMask` 插件，该插件持有账户的私钥。

**图 5-1** `MetaMask` 的后台工作原理

当 `MetaMask` 连接到其中一个以太坊网络时，它会通过其 RPC URL 连接到相应的 INFURA 节点之一：

*   **主以太坊网络：** `https://mainnet.infura.io/v3/`
*   **Goerli 测试网络：** `https://goerli.infura.io/v3/`
*   **Sepolia 测试网络：** `https://sepolia.infura.io/v3/`

**警告** 在合并（以太坊从 PoW 切换到 PoS）之前，大多数开发者都使用 Ropsten 测试网络。然而，Ropsten 测试网络在合并后已被弃用。你应该改用 Goerli 测试网络。

系统将向您展示您的秘密恢复短语。秘密恢复短语是一组单词，允许您恢复和还原您的所有账户。这组单词需要您妥善保管。我个人建议您将其打印出来，并锁在安全的地方。

要显示您的秘密恢复短语，请点击锁图标（参见图 5-7）。

![](img/471881_2_En_5_Fig7_HTML.jpg)
*`MetaMask` 页面的截图。页面在"秘密恢复短语"下方显示了一条消息和提示。一个带有锁的方块被模糊处理，并伴有"稍后提醒我"和"下一步"按钮。*

**图 5-7** — 显示 12 个单词的秘密短语

您可以看到您的秘密恢复短语。请确保除了您自己之外，没人知道这 12 个单词。

> **提示**：这 12 个单词的秘密短语允许您在忘记密码时恢复您的账户。这个 12 个单词的秘密短语被称为 `BIP39`（比特币改进提案）*种子短语*。`MetaMask` 使用 `BIP39` 协议来计算如何使用 12 个单词的秘密短语来获得一个*种子*。这个种子用于为每种加密货币生成一个根密钥（就以太坊而言，是 ETH）。
>
> 然后，根密钥会被哈希处理以生成一个*私钥*，该私钥用于生成一个*公钥*，公钥又用于生成账户的*公共地址*。要生成第二个账户（以及第三个账户，以此类推），第一个私钥会被用来派生一组子密钥，然后这些子密钥被用来生成它们各自的公钥，依此类推。
>
> 使用这种方法，种子通过一个确定性算法为您的钱包生成无限数量的地址。
>
> 秘密短语中使用的单词列表可以在这里找到：`https://github.com/bitcoin/bips/blob/master/bip-0039/english.txt`。它基于一个包含 2048 个单词的列表，并按特定顺序选择了其中的 12 个单词。

系统将要求您确认您的秘密恢复短语（参见图 5-8）。按单词呈现的顺序点击页面底部的单词。所选单词的顺序很重要。如果顺序不正确，请拖拽单词重新排列。

![](img/471881_2_En_5_Fig8_HTML.jpg)
*`MetaMask` 页面的截图。页面在"确认秘密恢复短语"下方显示了一个空白框，用于选择每个短语。*

**图 5-8** — 确认秘密恢复短语

完成单词排列后，点击 **确认**。这样就完成了！您的账户现已设置完毕！

### 选择以太坊网络

默认情况下，`MetaMask` 连接到以太坊主网。这是一个真正的网络，使用真实的以太币进行交易、运行智能合约等。出于开发目的，您不希望连接到真实网络。相反，您应该连接到可用的测试网络之一：

*   Goerli 测试网络
*   Sepolia 测试网络

`MetaMask` 允许您连接到这些测试网络中的任意一个（参见图 5-12）。

![](img/471881_2_En_5_Fig12_HTML.jpg)
*以太坊主网页面覆盖在网络列表上的截图。以太坊主网已被选中。底部有一个"添加网络"按钮。*

**图 5-12** — 选择要连接的以太坊网络

> **提示**：默认情况下，`MetaMask` 会隐藏测试网络。要查看测试网络，请点击以太坊主网下拉列表旁边的图标，选择**设置 > 高级**，然后打开**显示测试网络**选项。

此外，`MetaMask` 还允许您连接到正在监听 `8545` 端口的本地以太坊节点。它还允许您连接到自定义的 RPC 主机。

在本章中，您将使用 Goerli 测试网络，因此在 `MetaMask` 中选择它。
