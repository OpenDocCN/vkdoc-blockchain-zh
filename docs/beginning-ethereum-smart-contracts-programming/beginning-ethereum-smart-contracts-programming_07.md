# 权益证明（PoS）

鉴于近期对可持续性的关注，如果区块链要成为去中心化解决方案的长期技术，`PoW`绝对不是可行之道。因此，许多新的区块链实现正在使用其他共识协议。特别是，`以太坊`已从使用`PoW`迁移到一种名为`权益证明（PoS）`的新协议。

在`PoS`中，以前所谓的矿工现在被称为*验证者*。这些验证者铸造（或锻造）新区块。要成为验证者，一个节点必须质押（因此得名`权益证明`）至少`32 个以太币`。质押的`以太币`数量将影响该节点被选为下一个验证者以铸造下一个区块的机会。

当一个节点成为验证者后，它将执行与矿工相同的任务：验证区块中的所有交易以确保其有效。当一切检查通过后，验证者签署该区块，将其添加到区块链，然后在整个网络中同步区块链。该验证者从其铸造的区块中获得交易手续费。

与`PoW`中每个矿工都在做相同工作相比，`PoS`看起来更高效，因为只需要一个验证者。但这引出了另一个问题：如果验证者不可信，并故意允许欺诈性交易被添加到区块中，该怎么办？这就是验证者所质押的权益发挥作用的地方。如果之后发现验证者允许了欺诈性交易被添加到区块中，验证者将失去其部分质押权益。只要质押金额高于验证者能赚取的交易手续费，我们就可以相信验证者会正确履行职责。如果验证者未能妥善履行职责，他们将损失超过收益的部分。

当验证者不再担任此角色时，所有交易手续费及其质押权益将在稍后日期返还给他们。这给了网络时间来惩罚验证者，以防后来发现某些铸造的区块是欺诈性的。

## PoW 与 PoS 的比较

现在您了解了`PoS`的工作原理，将其与`PoW`进行快速比较会很有帮助：

- 在`PoW`中，每个矿工节点都在挖矿！而在`PoS`中，只有被选中的节点才能锻造新区块。
- `PoS`更加去中心化。`PoW`利用矿池，人们联合起来以增加挖到新区块的机会。这些矿池控制着区块链（如`比特币`）的很大一部分，这是危险的。如果所有大型矿池合并，它们就可以开始批准欺诈性交易。
- 与`PoW`不同，`PoS`更容易锻造新区块，且不需要昂贵的硬件。这鼓励了更多验证者参与，从而增加了网络的安全性。
- `PoS`也是环保的，比`PoW`消耗的能量少得多。
