# 共享账本

过去时代传统的集中式云端服务器无处不在，无论是电子商务平台、媒体渠道还是电视行业。我们看到的广告和播放的节目都采用了一种更为集中、商业化的方式。

![../images/474923_1_En_2_Chapter/474923_1_En_2_Figd_HTML.jpg](img/474923_1_En_2_Figd_HTML.jpg) 冷知识

找出那些在大型市场上占据主导地位的高度集中化平台：

*   高度可篡改（可被任何人操控——黑客、干扰者等）
*   可能因故障、写入失败和服务器宕机而丢失数据
*   易受数据泄露和隐私问题影响
*   访问控制薄弱，且由解决方案提供商掌控

这引发了关于失踪船只和飞机的众多阴谋论。

`BBChain` 的概念用新时代去中心化、透明的共享账本彻底革新了传统的存储和托管方案。账本上仅可追加的数据作为唯一的事实来源值得真正信赖，能够确保安全，并与正确的利益相关者建立直接联系和可靠性。

让我们回到同样的问题：为什么用区块链？为什么要去中心化到共享账本？

在过去民主尚未完全成熟的年代，一些有影响力的个人或公司占据主导地位的现象更为普遍。从安装电话线到申请签证，一切都完全依赖于中心化权威机构。随着社会进步，民主结构在个体中得到改善，更多人群参与到决策、法律、权利以及我们的工作方式之中。

这一千禧一代可能并不完全了解在互联网时代之前获取某些机会所经历的痛苦。如今我们身处数字时代，互联网已成为普遍商品，构建我们使用的软件应用程序至关重要。例如，在一个社交媒体平台上，当某人上传一张照片并选择删除时，该照片的数据权限在上传至这一中心化平台时就完全由个人丧失（这一点可能因《通用数据保护条例》而改变，但执行和检查可能仍需手动）。因此，将没有民主的时代与当前时代相比，区块链带来了机会，让每个身份都能在数字时代发声、构建、投票并做出贡献。从技术角度来说，每台设备都成为访问控制触点，每台电脑都成为存储节点，每个用户都行使投票权，而共享账本上每个可自动化的策略都为该平台上的每一笔交易和每一次操作带来了透明度、数据去中心化、隐私性、安全性、不可篡改性和有效性。这正是本书希望向所有读者传达的理念。

以下是一些正在产生社会影响力的区块链（即共享账本）示例：

*   联合国世界粮食计划署于 2016 年启动了一个名为“Building Blocks”的项目，根据项目网站介绍，该项目使用以太坊区块链，使“世界粮食计划署日益增长的现金转移操作更快、更便宜、更安全”。该项目最初试点用于追踪对叙利亚难民的援助。
*   由联合国儿童基金会创新基金投资的区块链平台 Amply 和 IXO 为帮助南非教育的慈善机构的融资交易提供透明度。
*   布鲁克林微电网正在改变我们选择电力来源的方式，例如私人住宅产生的太阳能、传输中的能源等，通过共享账本实现生产与消费的直接供需连接，推动真正去中心化的经济。
*   M-Vision 是一个运行在区块链上的去中心化孵化器，它量化远程研究成果，并通过由研究人员、专业人士和实业家组成的账本网络将其推向商业化。

回到 BBChain，我们利用 BigChainDB 和 Cassandra 的原理以分布式方式存储数据，在图 2-4 和 2-5 中展示了共享账本上的公开数据。

以下是金融数据在中心化平台与去中心化平台上的存储模拟对比：

*   中心化/普通平台：
    *   作为加密记录存储在单个服务器的数据库中

```
X = "机密信息：100 万美元已转账至约翰·多伊博士"
```

*   去中心化平台/区块链：
    *   使得任何欺诈用户或网络干扰者难以解密并获取 X，因为黑客首先需要获得加密语句的完整形式，然后还需要解密密钥，而密钥也可能分布在链上。

```
Y = 加密(X)，将 X 分割成碎片并存储在账本的 N 个节点上
```

BBChain 在共享账本上以安全的方式将数据分布并锁定在链上。

根据以上信息，请识别在线平台的哪个主导方面可以在分布式账本上重构。

例如，在贸易金融中，金融机构是能够为大规模进出口活动开立信用证和银行保函的主导权威。这里的主导方面在于发行权，该权力由当局定义的规则和检查实现民主化。然而，这种发行有时可能受到人为偏见和影响，导致违约。当这种信任与权威方面在共享账本的智能合约上实现自动化时，就能更好地避免贸易违约。因此，本挑战要求读者识别自己希望重新编程的哪个主导部分，而本书将帮助您构建和编程该部分。
