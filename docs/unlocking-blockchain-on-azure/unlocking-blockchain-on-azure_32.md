# 贸易活动融资与追踪的新途径

随着加密货币的出现，多种代币经济模式应运而生。许多零售商开始接受此类支付方式。基于此类机制的微型借贷和融资成为可能。然而，由于缺乏监管，代币的可信度往往不符合国家法律框架。但自比特币问世以来，比特币和以太坊已成功获得了广泛关注。通过将工作和流程进行微量化，在链上进行激励无疑为利益相关方带来了优势。

再看另一个因区块链而得到更好发展的例子：一家领先的组织，它作为一个市场平台，帮助采购方在供应链中以全球市场的最优价格采购物料。全球贸易的百分之九十通过国际航运业实现。让我们深入探讨该行业的采购周期，以及区块链技术如何对其产生影响。

## 海事行业：保险、采购与贸易

让我们来看看最古老的行业之一，一个自丝绸之路时代就已存在的行业——海事行业。

鉴于该行业覆盖全球各个角落、货物运输量大且价值高、从业人员众多、各国法律法规复杂，以及之前讨论过的贸易金融问题，通过区块链干预能够改进的用例数不胜数。我们可以探讨以下几个案例：

*   海上保险
*   海事市场平台



### 海上保险

2017 年，四大会计师事务所之一的`EY`与软件安全公司`Guardtime`联合发布了全球首个针对海上保险领域的区块链平台。该平台的开发与推出是与`A.P. 穆勒-马士基集团`、`ACORD`、`微软`、`MS Amlin`、`韦莱韬悦`及`XL Catlin`通力合作的结果。这些合作伙伴在经过长达二十周的严格`概念验证`后共同采取了这一举措。

该平台基于`Microsoft Azure`全球云技术构建，是保险业首个此类平台，其分阶段部署旨在覆盖海运业端到端的应用场景。

这一全球化平台将海运客户、保险经纪人、保险公司及第三方连接到分布式共享账本，帮助获取关于身份、风险因素和风险敞口的数据。随后，平台将这些信息与保险合同整合。该平台能够创建并维护来自多方的资产数据，将数据关联至保单合同；接收并处理导致定价或业务流程变更的信息；连接客户资产、交易和支付；以及捕获并验证最新的首次通知或损失数据。

`EY`保险业务负责人`肖恩·克劳福德`的一段引述表明：

> *区块链改变保险生态系统的潜力始终清晰可见。我们所做的是将潜力变为现实。该解决方案首次将区块链的透明度、安全性和标准化应用于海上保险，并已准备投入商业使用。我们期待在整个海上保险行业部署这项技术，并正在探索这些成果与洞察将如何应用于其他专业保险市场及更广泛的领域。*

该平台部署于整个海上保险行业生态系统，其构建愿景是应对复杂系统带来的挑战——这些系统涉及多方参与、多重规则、工作重复、数据数字化采集缺失，以及大量对账问题。此前的这些问题导致数据访问不透明、数据无法与链条其他环节关联、保险公司承保风险评估不够精确，以及上述问题相关的模型优化困难。

`A.P. 穆勒-马士基集团`2017 年风险与保险负责人`拉尔斯·亨内贝格`的言论指出，风险理解与评估的问题与区块链技术具有契合度或解决潜力，从而让各方都受益。他的原话是：

> *利用技术简化和自动化我们与保险市场的互动是我们的优先事项。目前的保险交易过于繁琐且摩擦重重。风险与资本之间的距离实在太远了。区块链技术有望推动这一期待已久的理想发展。*

保险公司可以利用该区块链平台优化资本配置、提升效率，同时提高透明度，减少人工数据录入、对账及管理成本。该平台基于`无密钥签名基础设施`（即`KSI`平台）构建。

不仅是海运业，整个保险业都能从区块链技术中受益，因为它既能带来透明度，又能消除耗时的操作，同时降低成本。这一系列优势使风险评估更加精准、更具实时性。

引用`Microsoft Azure`首席技术官`马克·鲁西诺维奇`的话，我们可以看出：

> *微软认为区块链是一种变革性技术，能够显著降低商业摩擦，特别是简化跨组织的共享业务流程。海上保险正是可以通过区块链优化的复杂业务流程的典型例子。我们始终致力于将区块链引入企业，并很高兴与 EY、Guardtime 及其他行业专家合作，开发和部署由 Microsoft Azure 支持的区块链解决方案。*

目前，由于该技术带来的所有权属性和透明度，区块链在银行和金融服务领域的应用场景日益增多。贸易融资的首个案例已得到良好推广，如今，贷款及保险业也正在探索区块链的潜力。
