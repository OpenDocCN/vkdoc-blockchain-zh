# 柯达企业服务概述

这套名为 `Kodak Services for Business` 的套件，是一个旨在高效安全地存储和管理敏感性质文档的平台。柯达文档管理平台旨在为企业和政府提供一种设施，通过将文档安全存储并在区块链上管理访问权限来处理文档。

据柯达估计，通过工作流程自动化以及减少在内容、信息和文档方面的人工干预，可以节省 20% 到 40% 的成本。2018 年 1 月，一家第三方机构授权使用了柯达品牌名 `KODAKCoin`，该币种被设计为与一个为追踪在线图片版权而开发的区块链协同工作。

类似于供应链曾是区块链规模化的首要原因之一，文档管理及其安全流程在区块链领域正变得越来越受欢迎。并非只有柯达将其视为面向企业的潜在服务。

波兰-英国金融科技公司 `Billon` 也在开发一个基于区块链的文档管理系统平台。

数字时代中，考虑到纸质文档带来的冗余和对账工作浪费，文档数字化已促使企业探索数字化转型，并进一步论证了区块链作为最可行选项的合理性，尤其是在安全性和访问控制方面。
