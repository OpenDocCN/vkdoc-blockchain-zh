# 2. 区块链简介

`区块链`是一个可共享且不可篡改的账本，它使系统间的交易和计算更加简便、安全，并能在商业世界中追踪资产。

重要的是要理解，资产可以是有形的，也可以是无形的。这包括知识产权、专利、版权、商标等。区块链概念在商业领域被广泛应用。

区块链（可将其视为“一系列区块”）是一个可共享且不可更改的数据包。通常，它可以被定义为一个数字注册表，其中的条目被连接到一系列/链条的“区块”中，并按顺序串联起来，其完整性通过密码学技术得以确保。尽管其规模会随时间增长，但它是不可更改的，因为其内容一旦写入，通常无法在不使整个架构失效的情况下进行修改或删除。在区块链系统上追踪或交易资产可以降低交易双方的风险和成本。

请注意，这些概念属于更广泛的分布式账本范畴。（`分布式账本`是一个共享账本，它可以是一组公开或私有的数字信息，这些信息被同步并地理分布到多个国家、站点或机构，无需管理员或集中式数据管理。）这意味着这些系统基于一个可由网络中多个节点读取和更改的分布式账本。

此外，所涉及的节点无需了解彼此的行为或身份，也无需相互信任。为了确保不同副本之间的完整性和一致性，新区块的添加由全球统一的规则或协议管理。当区块从网络或系统获得批准后，每个节点都需要更新其自身的私有副本。因此，该数据结构保证了未来无法进行篡改。

采用区块链和分布式账本技术开发的系统具有数据数字化、去中心化、去中介化、转移可追溯性和透明性等特点。

凭借这些特性，你可以以集中、可监控的方式提供交易，并为每笔交易提供安全保障。

## 2.1 区块链简史

根据维基百科记载，第一个区块链由中本聪于 2008 年提出，并于次年实施，旨在为新兴数字货币比特币充当“账本”（所有交易的登记簿）。中本聪在原始文章中对`区块`和`链`这两个词的使用方式有所不同。2009 年，中本聪的创造物——比特币，首次被用于购买实物资产：一个披萨。

2014 年 8 月，随着比特币已在全球范围内声名鹊起，其区块链大小已达到 20 千兆字节。2018 年 3 月，它超过了 162 千兆字节。2014 年，“区块链 2.0”这一术语开始被用于指代一种使用区块链的新方式。其理念是让那些被当前货币体系排除在外的人能够获得可靠、安全的货币存储，保护他们的隐私，并实现其信息的货币化。根据一些作者的观点，它还有可能通过改变财富分配方式来解决社会不公问题。

2017 年，内华达州参议院完全放开了对区块链的限制。特别是参议院第 398 号法案的第 4 部分规定：

- 禁止地方当局对区块链征税。
- 禁止要求为使用区块链申请任何形式的许可证。
- 禁止对区块链的使用提出任何其他要求。
- 禁止地方当局对区块链征税。

2019 年 4 月，在国际手工艺品展览会上，展示了首款（意大利制造）产品，其生产步骤完全使用区块链技术进行追溯。

对数字文档进行时间戳标记的需求催生了区块链技术。例如，你可以给自己写一封信并通过邮件寄出。如果你不拆封，就能保证这封信是在信封盖戳之前写好的。企业在日常运营中采用日益复杂的流程，以提高其内部文件在日后被质疑时的可信度。在网络世界中，交易需要以安全的方式使用数字印章。可以使用比特币或区块链技术创建安全的时间戳数字签名。

所有数字签名都使用哈希函数。这些方法接受任何文件，并生成固定长度的指纹，这是一种唯一标识该文件的代码。任何有权访问该文件的人都可以生成该代码；然而，根据代码重构文件则非常困难。

## 2.2 区块链的应用

从长远来看，区块链具有改变商业战略和模式的巨大潜力。此外，区块链并非是一种颠覆传统商业模式的破坏性技术，而是一种赋能技术，能够为全球经济和社会环境创造新的技术和基础。

区块链的应用有望显著改善全球供应链、金融与营销交易、会计以及分布式社交网络。这项新技术可以整合到各个领域，其协议使企业更容易采用新方法来处理和管理数字交易。

例如，货币支付系统和促进众筹的数字货币、预测市场的实现以及通用治理工具。区块链可用于认证文件日期并验证其未被篡改。通过插入待认证文件的哈希值来实现的这项区块链应用被称为`公证`。一个例子是对从建筑工地运输泥土的卡车运行轨迹进行公证，以确保符合环境标准。

区块链可以满足对信托服务提供商的需求，并确保减少诉讼成本。它有潜力消除系统性风险和金融欺诈。它可以自动化许多以前需要长时间手动完成的流程，例如业务整合。

总的来说，大多数区块链应用都涉及诸如`Bitcoin`、`Blackcoin`、`Dash`和`Nxt`等加密货币。其中心账本系统存储所有交易，并使用去中心化的投票机制来理解交易。

这种新的分发方式也适用于保险和银行领域。这些领域对此新兴技术感兴趣，因为这些交易既快速又非常安全。

协作经济和物联网（`IoT`）技术也可以通过让许多志同道合的人参与在线协作投票应用，从而从区块链技术中受益。

区块链还可用于构建医疗人员信息系统，以提高其互操作性和安全性。多个区块链也已被开发用于存储数据、发布文本以及识别数字艺术的来源。例如，一家银行开设了专门研究区块链的研究中心，旨在研究这项新技术对金融服务产生的各种影响，以提高效率并降低成本。还有一项应用被提议用于共享无线网络。

## 2.3 为何区块链如此重要

企业依赖信息。在商业世界中，信息越快速、越准确就越好。区块链非常适合提供这类信息，因为它提供即时、可共享且完全透明的数据，并存储在不可更改的账本中。只有经过授权的个人才能查看这些数据。

此外，区块链网络可以追踪订单、货币交易、会计账本和生产流程等内容。由于成员共享包含在区块链中的公共信息，因此还可以查看端到端交易的所有数据（对所有参与方完全透明），这带来了更高的信任度，以及新的改进和未来应用。
