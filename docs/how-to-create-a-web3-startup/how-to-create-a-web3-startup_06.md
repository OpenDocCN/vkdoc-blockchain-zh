# 6. NFT、游戏与社交网络

## 消费者初创企业

`Mike Winkelmann`是一位平面设计师和艺术家，^(¹²⁶) 他更为人熟知的名字是`Beeple`。他一直是创新者，在艺术作品中运用虚拟现实(`VR`)和增强现实(`AR`)等技术。`Beeple`也是许多名人的首选合作对象，仅举几例包括`Nicki Minaj`、`Eminem`、`Justin Bieber`、`One Direction`和`Katy Perry`。

2020 年，`Beeple`开始探索非同质化代币(`NFTs`)。这是一种基于区块链的技术，允许人们拥有数字物品。

他开始用`NFTs`创作数字艺术，这些作品售价高达数千美元。但与他的政治漫画和电子游戏图像的数字拼贴画“`Everydays: The First 5,000 Days`”相比，这些都相形见绌。

他将这幅作品在`佳士得`拍卖，竞拍异常激烈（共有 353 次出价）。中标者最终为这件`NFT`支付了高达 6930 万美元。这使得它成为在世艺术家成交价第三高的作品，仅次于`杰夫·昆斯`和`大卫·霍克尼`。^(¹²⁷)

毫无疑问，这场拍卖引发了轰动——成为`NFT`市场巨大增长的催化剂。这也是首批成为主流的`Web3`消费者应用之一。

当然，还有其他应用。`Web3`游戏和社交网络也获得了大量关注。

在本章中，我们将探讨这些市场如何发展，以及为企业家带来的机遇。

## NFT 如何运作？

在`NFTs`出现之前，拥有数字物品是不可能的。毕竟，人们可以简单地复制粘贴，对吧？确实如此。这就是为什么音乐等内容产业在数字世界中举步维艰。

但`NFTs`将改变游戏规则。通过使用区块链，可以为数字物品提供唯一的代币。

第一个`NFT`诞生于 2017 年。一个名为`Larva Labs`的工作室开发了名为`CryptoPunks`的数字角色。^(¹²⁸) 该公司基本上实现了像传统收藏品一样交易它们。

`NFTs`通常构建在`Ethereum`区块链上，并使用`ERC-721`协议。这允许存储各种详细信息。所有这些都对公众透明。而且，`NFT`几乎可以代表任何事物。

毫不奇怪，人们对`NFTs`可能存在一些误解。请记住，它并不传达任何版权。版权仍然属于物理物品的所有者。例如，可能有`蒙娜丽莎`的`NFT`，但这并不意味着你拥有原画。

> **备注**
> 标志性电影“`低俗小说`”的编剧`昆汀·塔伦蒂诺`试图为七个独家场景创建`NFTs`。^(¹²⁹) 但制作该电影的制片公司`米拉麦克斯`提起了诉讼以阻止此事。结果如何尚不明朗。但此案可能对确立`NFTs`的权利具有关键意义。

其次，`NFT`不是加密货币。主要原因是加密货币是可互换的。例如，一个`比特币`与另一个`比特币`没有区别，你可以轻松地交换它们。然而，非同质化代币是不可转让的。没有一个是一样的。不过，你可以使用加密货币来获取`NFT`。

`NFTs`和加密货币依赖于相同的底层区块链技术。`NFT`市场也可能要求人们用加密货币购买`NFTs`。然而，加密货币和`NFTs`的创建和使用目的不同。

请注意，`NFT`市场实现了惊人的增长。2021 年，支出达到 176 亿美元，增长了约 21,000%。^(¹³⁰)

以下是一些其他有趣的事实：

*   `NFT`投资者获得了大约 54 亿美元的利润。
*   用于`NFT`交易的钱包有 220 万个。
*   超过 470 个钱包拥有价值超过 100 万美元的`NFTs`。

到目前为止，最受欢迎的类别是收藏品。但游戏和`Metaverse`领域也实现了强劲增长。

然而，到 2022 年，`NFT`市场将大不相同。随着加密货币市场承受巨大压力，增长率骤降。但活动水平仍远高于几年前。

> **备注**
> 虽然许多`NFTs`存储在`Ethereum`网络上，但当然也有其他选择。一些比较知名的包括`Flow`和`Tezos`。



### NFT 实例

音乐流媒体行业已成为庞大的全球性业务。例如，Spotify 拥有 1.8 亿付费用户和 4.06 亿月活跃用户。^(¹³¹) 该服务覆盖 183 个市场。

尽管如此，音乐人获得的回报却远谈不上丰厚。试想，Spotify 平台上有 800 万音乐人，但每年能赚到 5 万美元或以上的仅有约 1.4 万人。^(¹³²) 换言之，许多人无法通过这一渠道维持生计。

而这正是 NFT 能够大显身手之处。它为音乐人——或者任何创作者——提供了一种更好实现其艺术价值变现的方式。他们通常能保留超过 95% 的收入。

诚然，如果创作者不使用像 Spotify 这样的平台，他们就无法拥有庞大的规模。但在 Web3 世界里，你并不需要庞大的用户数量。相反，它更侧重于迎合你的忠实粉丝。他们很可能愿意花更多钱，例如通过使用 NFT。这也可能是创作者与粉丝建立更紧密联系的有效方式。

以乐队“里昂王族”为例。2021 年初，他们以 NFT 形式发行了新专辑。这是史无先例的举动。

该乐队提供了三种类型的代币。^(¹³³) 它们分别提供不同的额外福利，无论是 VIP 座位、独家艺术品等等。

该乐队通过发行 NFT 筹集了超过 200 万美元。^(¹³⁴) 其中约 50 万美元捐赠给了 Live Nation 旗下的“Crew Nation”项目，该项目在新冠疫情期间为音乐幕后工作人员提供了帮助。

这并非暗示从 NFT 赚钱很容易。事实并非如此。事实是，你仍然需要依靠营销，比如在 Instagram、Facebook 或 Twitter 上拥有粉丝。这些仍然是吸引超级粉丝非常有效的渠道。

只需看看布兰登·山德森的有趣案例。他是顶尖奇幻作家，创作了《迷雾之子》等系列作品。2022 年，他发起了一个 Kickstarter 众筹活动。该活动为读者设置了不同的档位，包括获取四部即将出版小说的权限以及月度订阅盒子。

嗯，这取得了巨大的成功。山德森总共筹集了 4170 万美元。^(¹³⁵)

这无疑凸显了众筹的力量。但也展示了像 Tor 这样的传统出版商的重要性。直面现实吧，正是这一切帮助山德森建立了一批忠实拥趸——自 2003 年开始出版作品以来，他一直在培养这批粉丝。

### 购买与创建 NFT

购买 NFT 可能很繁琐。这是该市场的缺点之一。不过，这个过程必然会变得更好。

购买 NFT 的第一步是获取用于交易的合适代币。这通常是以太币。你可以从像 Coinbase 这样的加密货币交易所获得它。

接下来，你需要将你的加密货币转账到一个钱包，一个常见的钱包是 MetaMask。然后你需要将你的钱包连接到一个 NFT 市场。以下是一些例子：

- OpenSea
- NBA Top Shot
- CryptoPunks
- Rarible
- Foundation
- 币安 NFT
- Nifty Gateway
- Mintable

你会在市场上进行搜索。当你找到喜欢的 NFT 时，就可以进行购买了。也可能会有拍卖。无论如何，NFT 会进入你的钱包，你可以将来出售它，例如在某个市场上。

好了，那么你如何创建一个 NFT 呢？当然，你需要指定它所代表的一个项目。同样，这可以是几乎任何东西。但该项目应该是你拥有权利的东西。如果不是，你可能违反版权法，并可能需要承担损害赔偿。

然后，你需要选择区块链的类型。通常是以太坊。接下来，你需要设置你的数字钱包，然后购买加密货币。然后选择 NFT 市场，并将其与你的钱包连接。你将上传数字文件，例如 GIF、PNG 或 MP3。

你需要决定如何出售你的 NFT。选项包括以下几种：

- 固定价格
- 定时拍卖
- 无限期拍卖。这意味着你可以随时结束拍卖

如果采用拍卖形式，你需要设定最低价格以及版税等级（这是针对二级市场的后续销售）。然而，费用可能很高，例如以太坊的 Gas 费。正因如此，在 NFT 销售中亏损实际上很常见。

### NFT 的价值

与加密货币的情况一样，对 NFT 进行估值很困难。这通常是一个主观过程。

然而，有一些指导原则值得考虑。以下是一些要点：

- **稀有性**：稀缺可以创造价值。NFT 具有唯一性，这当然有助益。但当然，如果公众对某个数字项目兴趣寥寥，这一点帮助甚微。这就是为什么有价值的 NFT——尤其是在艺术和娱乐领域——通常是独特或与众不同的东西。Beeple 的案例正是如此。

- **实用性**：这通常适用于非艺术类的数字项目。基本上，实用性意味着你可以用 NFT 做某些事情。一个例子是允许你进入音乐会的 NFT（尽管，这样的 NFT 可能价值有限，因为通常会有时间限制）。另一个例子是你可以用它来在游戏或元宇宙中进行购买。

- **有形性**：这是指 NFT 与现实世界有联系。房地产就是一个例子。代币可以代表部分所有权。这有助于改善记录保存并减少欺诈的可能性。

- **来源**：这是一份关于谁曾拥有该 NFT 的历史记录。这可以提供关于该物品价值的见解。毕竟，前任持有人可能在 NFT 方面有良好的记录。

- **流动性**：查看区块链，该 NFT 是否持续获得买家的兴趣？类似的数字项目需求如何？

- **社会认同**：这描述的是有名人支持该 NFT。诚然，这并不能保证创造价值。但它仍然是一个非常重要的考量因素。

- **互操作性**：这意味着你可以在其他平台上使用该 NFT。例如，你可以在一个游戏中购买 NFT，但在另一个游戏中也能使用它。



### NFT 的问题

希娜·埃斯塔维是总部位于马来西亚的区块链公司 `Bridge Oracle` 的首席执行官。2021 年，她同意支付 290 万美元从杰克·多尔西那里购买了 NFT（她将这笔钱捐给了慈善机构）。这个 NFT 是 Twitter 上的第一条推文，内容是“just setting up my twttr。”

这笔收购引起了轰动——并进一步激发了人们对 NFT 市场的兴趣。

但不幸的是，对埃斯塔维来说，情况并不太乐观。在不到一年的时间里，她将 NFT 挂牌拍卖——但市场反应冷淡。出价甚至没有超过 14,000 美元。^(¹³⁶)

这样的结果并不罕见。现实情况是，NFT 市场波动性极大——甚至比加密货币还要剧烈。

实际上，其价值往往受到投机行为的影响。此外，还有潮流风尚的影响。

诚然，你仍然可以从 NFT 中赚到很多钱，但你可能需要采取更短期的策略。成功或许更像是日内交易。

然而，NFT 市场还存在其他令人困扰的问题：

*   **普及性**：正如本章前面所述，购买和创建 NFT 的过程远非直观。但如果这个市场要变得更加主流，就必须有显著的改进。本质上，它需要像亚马逊一样实现一键交易。
*   **拉地毯**：这是指 Web3 开发者放弃项目并卷款跑路。那些被骗的人几乎没有追索的途径。
*   **白名单**：这是指邀请特定群体参与一个项目。当项目向公众开放时，他们将获得可观的收益。这类似于低价股的“拉高出货”。根据 `Chainalysis` 的研究，白名单用户的收益率为 75%，而普通用户仅为 20%。^(¹³⁷) 不幸的是，这是一种常见做法，并且很难对此采取任何措施。
*   **虚假交易**：用户在同一笔交易中既是卖方又是买方。这是可能的，因为你可以轻松创建两个钱包，并且无需提供任何身份信息。通过虚假交易，可以人为地制造某个 NFT 的交易量，从而吸引关注。
*   **版权侵权**：某个人将他人知识产权制作成 NFT 并不困难。这不仅欺骗了真正的所有者，也欺骗了进行购买的买家。

### 新的市场进入策略

2010 年，凯文·斯特罗姆和迈克·克里格创建了一个名为 `Burbn` 的应用。它允许用户签到位置并发布消息和照片。但有一个问题：它几乎无人问津。这个应用很笨拙——这在早期的 iOS 应用中很常见——并且没什么实际用途。

但斯特罗姆和克里格注意到了一件有趣的事。`Burbn` 的用户非常喜欢照片分享功能。于是，两位创始人只专注于这一点，并创建了 Instagram。^(¹³⁸) 它还具有编辑照片的功能。不过，也许最重要的功能是只需点击三次就能发布一张照片。

在一个月内，Instagram 获得了超过一百万的用户。^(¹³⁹) 这期间没有任何广告或营销。增长完全靠口碑。

两年之内，创始人以 10 亿美元的价格将 Instagram 卖给了 Facebook。

这无疑又是一个标志性的硅谷故事。但这也是一个极其罕见的成功案例。在大多数情况下，初创公司通常需要大量的营销。这是风险投资大部分资金的去向。

那么，有了 Web3，情况会有所不同吗？对风险资本的需求会减少吗？

或许会。原因是，Web3 对代币的聚焦可以作为一种融资机制。它也可以是吸引用户的有效方式。嘿，谁不想赚钱呢，对吧？

但有了 Web3，初创公司的典型演变路径通常不同。你可以在没有产品的情况下开始。相反，你会先提出一个想法，然后从拥有代币的社区那里获得反馈。这有助于实现产品与市场的契合。

但要实现这一点，需要有一个令人信服的愿景。什么能鼓励人们加入并押注你的项目？你打算解决什么问题？

通过回答这些问题，你将更有可能创建一个强大的社区。他们也将有更明确的方向来提供反馈，以完善这个愿景。

就市场进入策略而言，目前涌现出多种有趣的模式。以下是一些例子：

*   **开发者资助**：这是一种鼓励程序员加入项目的方式。资助形式可以是代币或传统货币。无论如何，通常需要某种类型的激励来引起兴趣。例如，`Compound` 在 2022 年 3 月 12 日至 9 月 12 日期间设立了一个开发者资助计划，并提供其自有代币 `COMP` 中的 5,000 枚作为资金。^(¹⁴⁰) 根据其网站的说法：“若没有‘胡萝卜’来激励社区成员提出变革，协议就无法在充满活力和竞争的市场中以应有的速度进行创新。” 该计划不仅仅用于开发新功能。它还用于代码审计，以及为发现漏洞提供赏金。
*   **模因**：这些可以帮助用户基础实现病毒式增长。不过，创造模因并非易事。创始人可能也不是做这件事的最佳人选。相反，这更适合那些有营销背景的人。在传统社交媒体渠道上拥有大量粉丝也很有帮助。还需要记住另一点：模因的影响往往是暂时的。换句话说，它们的创造很可能需要持续进行。
*   **空投**：这是指 Web3 项目免费分发代币。也可以空投 NFT。这通常是为了鼓励人们注册项目。但它也可以用于测试项目的技术。一个著名的 NFT 空投案例是 `Bored Ape Yacht Club`。该项目创建了一个名为 `Mutant Ape Yacht Club` 的新系列。一次空投让所有现有用户都能创建 10,000 个被称为“变异猿”的猿猴，另外还有 10,000 个可供新用户获取。这对现有用户来说是一种有趣的奖励。同时，这也是吸引新用户的有效方式。

### Web3 的社会身份

在 Web3 世界中，用户拥有自己的头像很常见。它们也可以作为社交媒体的头像。这些通常是复古、像素化的视觉图像，比如朋克或无聊猿。例如，Jay-Z 和 Snoop Dogg 就在他们的 Twitter 和 Instagram 账户上使用这些头像。

Web3 头像背后的理念是，它们不应该被修饰打磨。相反，重要的是与社区的互动。

请记住，许多头像都是 NFT。通过这样做，它们可以成为下一代版本的简历。由于 NFT 在区块链上，用户可以公开查看该用户的活动情况。



### Web3 社交网络

截至 2021 年底，Facebook 的用户基数已达 28.2 亿，同比增长 8%。^(¹⁴¹) 当时，全球人口接近 80 亿。

Facebook 在变现方面确实做得非常出色。2021 年营收高达 1179 亿美元，利润达 393 亿美元。

但对于那些为 Facebook 贡献内容的用户，平台支付了多少报酬呢？基本微乎其微。原因在于公司的大部分收入都来自广告。

这本身并无不妥。许多人喜欢使用 Facebook。这并非为了赚钱。

然而，对于 Web3 创业者来说，他们正试图颠覆 Facebook 的模式。区块链意味着用户将对自己的数据拥有更多控制权，并获得赚取代币的机会。

Web3 社交网络很可能与传统平台截然不同。首先，其界面可能使用开源工具，例如用于聊天功能。或者它也可能基于像 `Discord` 这样的专有系统。

其次，Web3 社交网络的规模很可能远小于 `Twitter`、`Snap` 或 `Facebook`。原因是它将专注于某个特定社区。部分原因在于它很可能涉及一个 `DAO`。为此，会有一个特定的目标。否则，可能难以吸引合适的人选来让这个社交网络富有活力。

现在，Web3 社交网络的一个关键优势是可移植性。如果你想离开，不会放弃你的社交图谱和内容。它们属于你，因为它们在区块链上，你通过哈希所有权来维护数据。然后你可以将其转移到另一个平台。

相比之下，传统社交网络是一个封闭的花园。当你离开时，无法转移你的用户或内容。除非你删除账户或将其关闭，否则它仍然属于该社交网络。

这长期以来一直是社交网络的巨大权力来源。正因如此，即使出现侵犯隐私或其他重大问题，往往对用户数量影响甚微。用户基本上被牢牢锁定。

因此，社交网络拥有巨大的权力，例如可以改变政策。如果 `Twitter` 不再希望其平台上存在某些内容，它可以直接禁止。或者，如果它想收取费用，也可以这样做。

大型在线市场也是如此。看看 `iOS` 就知道了。如果你想在平台上发布应用，需要经过审查流程。如果失败，对创业者来说可能是一个重大挫折。

接着是费用问题。`Apple` 通常对应用产生的收入抽成 30%。总而言之，这会让创业者很难赚钱。即使是最大的科技公司也是如此。请注意，游戏开发商 `Epic` 正在就其 `Appstore` 政策起诉 `Apple`。

Web3 社交网络仍处于起步阶段。目前还没有哪个网络取得突破性发展。但这肯定会改变——而且可能很快。

以下是该领域一些有趣的初创公司：

*   `APPICS`：创始人 **Uma Hagenguth**，常驻瑞士。她 16 岁时创办了第一家公司，专注于营销和网页设计。^(¹⁴²) 至于加密货币，她在 2013 年开始涉足。她对 `APPICS` 的构想是创建一个界面时尚、基于奖励的社交网络。用户可以通过创作获得特定点赞数和评论数的内容来赚取代币。^(¹⁴³) 因此，激励基于参与度。`APPICS` 构建在 `Telos` 区块链上。该应用可在 `iOS` 和 `Google Play` 上下载。

*   `Aave`：这是另一家位于瑞士的初创公司。创始人 **Stani Kulechov** 于 2017 年推出该公司。^(¹⁴⁴) 他是在赫尔辛基攻读法律学位期间创办的。该公司是 `DeFi` 加密货币（基于 `Ethereum` 网络）的早期开发者之一。但 `Aave` 也在进军社交媒体领域。该项目名为 `Lens Protocol`，基于 `Polygon` 区块链构建。^(¹⁴⁵) 它允许创建一个作为 `NFT` 的用户资料，其中包含所有社交媒体内容。用户可决定如何将其变现。有趣的是，`Aave` 正在探索将个人资料转变为 `DAO` 的想法。

*   `Yup`：这是一个 Web3 平台，允许用户策管来自 `OpenSea`、`Twitter`、`YouTube` 和 `Mirror` 等平台的内容。`Yup` 根据用户的参与度向其支付报酬。2022 年初，该公司从 `Distributed Global` 筹集了 350 万美元。^(¹⁴⁶)



### Web3 游戏

Web3 中一个早期的成功案例是游戏领域。该类别通常被称为 `GameFi` 或加密游戏。

根据 `DappRadar` 与 `BGA Games` 联合发布的一份报告，2022 年第一季度的市场增长高达 2000%^(¹⁴⁷)。大约 52% 的区块链活动归因于游戏，约有 122 万个钱包参与其中。

风险投资正在加速涌入。2022 年第一季度，投资额约为 25 亿美元^(¹⁴⁸)，预计全年总额可能达到 100 亿美元。

其中一笔引人注目的交易涉及总部位于澳大利亚墨尔本的初创公司 `Animoca Brands`。该公司以超过 50 亿美元的投前估值筹集了 3.59 亿美元^(¹⁴⁹)，领投方是 `Liberty City Ventures`。

`Animoca Brands` 开发了多种中心化和去中心化游戏，这些游戏也适用于手机、游戏机、PC 和网页端。该公司是利用区块链实现游戏内奖励和货币化的先驱。

Web3 游戏的发展势头并不令人意外。许多游戏玩家已经熟悉区块链和加密货币。此外，游戏行业一直是允许购买虚拟物品的早期采用者。

在未来几年，游戏很可能是大量人群初步接触 Web3 的主要方式。随着用户引导流程变得越来越容易，这一进程将会加速。

然而，传统游戏行业在微交易方面一直面临挑战。简而言之，利用传统金融服务基础设施来让微交易变得经济实惠一直非常困难。

但区块链和加密货币应该会改变这一点。它将带来更高效的方法。游戏玩家也能从中受益，能够自己控制代币。这意味着他们可以在其他地方使用这些代币。这使得游戏玩家参与其中的吸引力更大。

> **注意**
>
> 新兴的游戏市场如 `Tegro` 正在涌现。它们专注于为游戏玩家提供买卖虚拟物品的途径。

相比之下，在传统游戏中，存在丢失虚拟资产的风险。原因是存在一个本质上拥有一切的中枢机构。当游戏关闭时，游戏玩家很可能会失去他们的资产。这种情况在诸如《我的世界：地球》和《哈利波特：巫师联盟》等游戏中就曾发生过。

Web3 游戏的另一个有趣方面是社区参与。通过建立 `DAO`，用户可以决定游戏的规则和演变方向。这可以进一步增加参与度和趣味性。然而，对于游戏开发者来说，重要的是不要过于激进。如果决策过多，可能会拖慢游戏进程。需要找到一个平衡点。

Web3 社区的一个有趣方面是排他性的概念。例如，你可以展示你是某款游戏的原始玩家之一。在社区中，这无疑可以赋予一定的地位。

Web3 游戏也有机会创造持久的品牌。它们还可以在用户的帮助下不断发展。所有这些最终都可能意味着 Web3 游戏将催生下一代娱乐公司，它们最终能与漫威等传统巨头相媲美。

对于 Web3 游戏初创公司来说，其初始产品甚至可能不是从游戏开始的。它可以是一个博客、一个 `NFT` 集合或一个内容网站。这会有一个有趣的故事线，引起用户的兴趣。在此基础上，初创公司可以围绕它打造一款游戏。在此过程中，社区将始终是世界构建环节的一部分。

但是，在 Web3 游戏中，某些因素不会改变。毕竟，一款游戏需要出类拔萃。现实情况是，市场会迅速饱和，从喧嚣中脱颖而出的难度会更大。这意味着需要在游戏上投入更多资源——同时也需要大量的创造力，而这并不容易实现。

另一个问题是游戏行业容易受到潮流影响。一款火爆的游戏可能很快就会变得无人问津。能持续数年的游戏非常罕见。但有了加密机制，这一点就变得至关重要。这正是建立真正价值的方式。

即使是成功的游戏公司也可能停滞不前。`Zynga` 就是一个例子。2012 年，该公司进行了高调的首次公开募股 (`IPO`)，股价为 10 美元。但业务面临压力。随后 `Zynga` 斥资数十亿美元进行收购，但这并未带来实质性的改变。2022 年初，`Zynga` 被 `Take-Two Interactive` 收购^(¹⁵⁰)。投资者得到了什么？股价是 9.86 美元。

对于 Web3，理解经济学至关重要。创业者甚至可能想聘请一位经济学家！为什么？原因是 Web3 游戏将是一个真正的经济系统，存在代币的供需关系。然而，如果结构不当，游戏很容易分崩离析。在某些情况下，它可能更像是一个庞氏骗局或多层次营销策略。在这种情况下，早期用户的收益开始减少，因为没有新成员被引入生态系统。当这种情况发生时，用户可能想要离开，因为他们的代币价值已经大幅贬值。

更重要的是，Web3 游戏的经济学可能意味着玩一款游戏需要花费数千美元。当然，这对许多用户来说可能是不可承受的。为解决这个问题，出现了汇集资源以允许玩游戏的公会。另一种选择是提供试用期。

Web3 一种流行的商业模式是“边玩边赚”。这种方法的创新者是总部位于越南的 `Sky Mavis`。该公司是《Axie Infinity》游戏的开发者。

其工作原理如下：玩家繁殖被称为 `Axies` 的角色。然后他们可以参与战斗和冒险。但其中也存在一个经济体。例如，玩家可以获取土地、制作药水等等。为了处理这些交易，游戏允许交易代表虚拟物品所有权的 `NFT` 和以太币。`Axie Infinity` 的原生代币是 `AXS`。

市场上的交易额已达到 36 亿美元，为单个 `Axie` 支付的最高价格是 82 万美元^(¹⁵¹)。每日活跃玩家有 280 万。

> **注意**
>
> 一些最受欢迎的“边玩边赚”游戏包括 `Splinterlands`、`Alien Worlds` 和 `Decentraland`。

`Andreessen Horowitz` 是 `Sky Mavis` 的投资者。在一篇博客文章中，作者指出：“`Axie Infinity` 解锁的能力是，你可以玩你喜欢的游戏，享受乐趣，同时还能参与社区的财务收益。这是一个非常强大的想法：这意味着游戏不再是纯粹的娱乐，而是跨界进入了工作领域。`Axie Infinity` 正在改变世界各地的社区，并取代传统的就业形式，因为游戏玩家互相帮助成功，从而改变了人们的生活。”^(¹⁵²)

Web3 游戏的另一种商业模式是“边玩边拥有”。这与“边玩边赚”类似，因为两种方法都涉及使用加密货币和代币提供激励。

那么，区别是什么？这更多是关于程度。对于“边玩边赚”，通常有更高的经济回报。有些人实际上可以靠游戏谋生。

另一方面，“边玩边拥有”游戏更注重乐趣和参与感。赚钱只是一个不错的额外奖励。

一款流行的“边玩边拥有”游戏是 `Crypto Raiders`。这是一个地牢探险游戏，玩家可以使用加密货币购买角色、怪物、任务等。该平台构建在 `Polygon` 区块链之上。

2022 年 3 月，`Crypto Raiders` 筹集了 600 万美元。领投方包括 `DeFiance Capital` 和 `Delphi Digital`。



Crypto Raiders 的 CEO 兼联合创始人尼古拉斯·克努珀表示：“我们打造 Crypto Raiders 的灵感，源于我们成长过程中玩过的 RPG 游戏。这些游戏是将游戏与区块链结合起来的完美用例。”^(¹⁵³)

最后，还有一种“边跑边赚”模式。它聚焦于现实世界，并倡导健康生活。这是如何实现的呢？来看看市场上的顶级运营商：STEPN。要玩这个游戏，你需要购买基于 Solana 区块链的 **NFT**。然后，你用智能手机上的 GPS 追踪自己的运动轨迹。通过这样做，你就能获得应用内代币奖励。归根结底：这是一种促进更健康生活方式的方式。

2022 年，STEPN 宣布获得了`500 万美元`的种子轮融资。^(¹⁵⁴) 领投方包括红杉资本印度和 Folius Ventures。

## 品牌与 NFT

由于隐私法规，企业通过数字平台与客户建立联系变得越来越困难。以 Cookie 的使用为例。这些技术可以追踪用户行为，也使得广告重定向成为可能。但 Cookie 正在被淘汰。谷歌的 Chrome 浏览器将在 2023 年弃用 Cookie。火狐和 Safari 浏览器也是如此。

或者看看苹果的新隐私要求。对于 iOS 应用，它们要求用户主动选择“允许跟踪”。毫不意外的是，这对在线营销工作产生了不利影响。Facebook 指出，这将在 2022 年减少其超过`100 亿美元`的销售额。^(¹⁵⁵)

有鉴于此，广告商们正在寻找替代方案。有趣的是，**NFT** 可能会帮上大忙。这可以通过使用现有或新的内容来实现。这些内容随后可以包含激励措施、福利和折扣。

事实上，这种策略可以让公司对其在线领域拥有更多控制权。我们得面对现实，当依赖 Facebook、YouTube、TikTok 或 Twitter 这样的平台时，算法变化可能会带来不利影响。这可能会降低广告效果，或意味着广告商需要支付更多费用。

然而，也需谨慎行事。在 Web3 世界中，用户通常不希望看到赤裸裸的营销推销。他们更喜欢引人入胜的内容，以及能提升公司产品或服务的沉浸式体验。这也可能包括为某项事业提供支持，例如向非营利组织捐款。

以下是一些例子：

*   在梅西百货感恩节大游行 95 周年之际，该公司拍卖了十件**NFT**。^(¹⁵⁶) 每件都包含历次活动中的一个游行气球。拍卖所得捐给了“许愿基金会”。该公司还向注册用户赠送了`9500 件 NFT`。
*   在麦当劳 McRib 烤肉三明治 40 周年之际，该公司赠送了限量版**NFT**。^(¹⁵⁷) 要想获得它，人们需要在 Twitter 上关注`@McDonalds` 并转发抽奖邀请。
*   美泰推出了风火轮**NFT** 车库。这包括为这些标志性玩具车打造的 40 个系列**NFT**。^(¹⁵⁸) 同时还有一项促销活动，有机会赢得限量版压铸车模。

作为对这块市场价值的验证，可以看看耐克。2021 年底，耐克同意收购 RTFKT，这是一家创建加密收藏品的 NFT 工作室。这家初创公司是 CloneX 的合作伙伴，后者成功创建了一个名为 CryptoSlam 的**NFT**，产生了超过`6500 万美元`的交易额。^(¹⁵⁹)

在关于这笔交易的新闻稿中，耐克 CEO 约翰·多纳霍表示：“这次收购是加速耐克数字化转型的又一步，使我们能够在运动、创意、游戏和文化的交汇点上服务于运动员和创作者。”^(¹⁶⁰)

好了，接下来在本章剩余部分，我们将看看 NFT 领域其他一些有趣的初创公司。

### Novel

罗杰·比曼是一位软件工程师，曾在高盛和 Shutterstock 工作过。^(¹⁶¹) 2021 年，他联合创立了 Novel。其目标是让企业更容易地创建、购买和销售 **NFT**，类似于使用 Shopify 这样的平台。比曼后来为他的初创公司筹集了`600 万美元`。^(¹⁶²)

他在 Shutterstock 和高盛的经历被证明非常有帮助。“这两份工作与 NFT/Web3 世界的关联度比你想象的要直接得多，”比曼说。“在 Shutterstock，我从事浏览器内图像编辑器的开发，并推动了浏览器内体验的可能性，这直接为我们帮助创作者使用 Novel 制作 **NFT** 的先进工具提供了灵感。在高盛，我负责构建复杂金融产品的市场架构，正是这种思维方式帮助我描绘了 **NFT** 市场及其对 Novel 在这其中扮演角色的定位。”^(¹⁶³)

Novel 的灵感来源于比曼和他的联合创始人——布莱恩·舒格和安娜·梅尔齐——开始看到电子商务广告获取成本不断上升的影响。“我们问自己，是否有更可持续、由社区驱动的品牌建设方式？”他说。“这就是 **NFT** 进入营销人员工具箱的地方。许多品牌都拥有一个等待被激活的潜在社区。Web3 拉近了品牌与其客户的距离，也让客户之间的联系更加紧密。”

不过，当时存在一个问题。许多品牌根本不具备创建和管理 **NFT** 的技术知识。因此，比曼着手利用“无代码”技术。

这意味着使用易于填写的表单和拖放操作。无需任何技术技能，也无需深入了解加密货币和区块链系统。

以下是 Novel 的工作原理：

*   第 1 步 – 创建：Novel Generator 创作艺术品，设定稀有度，并确定版税费率。系统随后生成智能合约的底层代码。
*   第 2 步 – 销售：只需按几个按钮，你就可以将你的 **NFT** 上架到 Shopify 平台。然后客户可以使用借记卡或信用卡进行购买。无需加密钱包。
*   第 3 步 – 铸造：购买完成后，系统会进行代币铸造。Novel 还为所有客户承担 Gas 费。

“我们受到启发，要创建一个能够通过消除许多准入门槛（例如需要了解加密货币和区块链技术）来普及 Web3 关系构建组件的平台，”比曼说。

Novel 也不仅仅是一项技术。该公司还提供策略方面的协助。考虑到许多品牌仍然对 **NFT** 缺乏深入了解，这一点无疑非常重要。

其定价基于每月`99 美元`的软件订阅费。Novel 对 NFT 销售收取 10% 的费用，对二级市场的交易收取 1% 到 2% 的费用。^(¹⁶⁴)

至于比曼给创业者的建议：“这个领域噪音很大，很容易迷失其中。要在这个领域成功构建，你需要穿透噪音，找到自己的真理。这在商业上普遍适用，但在这里尤其如此。你业务核心的敏锐洞察，很可能与当下对这个领域的炒作和媒体报道大相径庭。”



### Fractal

贾斯汀·坎是互联网视频的先驱。2007 年，年仅 20 岁出头的他，将一个网络摄像头绑在头上，直播自己的生活，这个节目托管在 `Justin.tv` 上。这引发了“生活直播”的潮流。^(¹⁶⁵)

坎坚持了大约八个月。没错，这为他带来了大量曝光，也帮助提升了 `Justin.tv` 的流量。

然而，坎从中看到了巨大的商机。他帮助将 `Justin.tv` 转变为一个视频中心。随后，他又协助创立了 `Twitch`。这开启了一股新风潮——即电子游戏的实时直播。这最终催生了电子竞技产业。接着在 2014 年，`Twitch` 以 9.7 亿美元的价格卖给了亚马逊。^(¹⁶⁶) 当时，该平台拥有超过 5500 万用户。

需要注意的是，坎并未停止寻找下一个重大商机。他相信这与 NFT 有关。为此，他在 2021 年联合创立了 `Fractal`。这是一个基于 `Solana` 区块链的市场，用户可以在其中直接从游戏公司购买 NFT。坎已经为这项事业筹集了 3500 万美元。^(¹⁶⁷)

`Fractal` 的一个关键部分是 `Launchpad`。这是一项帮助游戏公司销售其 NFT 系列的服务。

但并非任何人都能在 `Fractal` 市场上进行销售。这里有严格的审核流程，有助于提升品质和安全性（该公司仅接受大约 5%的提交申请）。坎认为 NFT 的未来需要策展。

`Fractal` 上的一些游戏包括 `Yaku`（赛车）、`House`（实时策略）和 `Tiny Colony`（多模式游戏）。该市场上的所有游戏的 NFT 系列都已售罄。

### Magic Eden

2021 年，Sidney Zhang、Jack Lu、Zhuoxun Yin 和 Zhuojie Zhou 联合创立了 `Magic Eden`。他们都在科技界拥有深厚根基，曾在 `Uber Eats`、`Facebook AI`、`FTX` 和 `Coinbase` 等公司任职。`Solana` 区块链是这项事业的一大吸引力。当 Lu 在 `FTX` 工作时，他亲身体验了这项技术。

`Magic Eden` 是一个优先考虑 Web3 的 NFT 市场（Lu 为公司起这个名字，是因为它蕴含着无限可能的感觉）。它是第一个允许竞标、提供稀有度指数，并建立连接二级市场的铸造系统的平台。它也已迅速成为最大的基于 Solana 的平台。以下是一些数据指标：^(¹⁶⁸)

- 每月超过 1000 万独立访客
- 超过 4000 个系列
- 超过 10 万个钱包连接
- 二级市场交易量达 10 亿美元

该公司已筹集了 2700 万美元的风险投资。

### OpenSea

2015 年，软件工程师 Devin Finzer 联合创立了 `Claimdog`。这是一项可以查找是否有企业欠你钱的服务。一年后，他将公司卖给了 `Credit Karma`。在那里，Finzer 了解了区块链技术，并将其视为他下一个创业公司的沃土。

他的想法是 `WifiCoin`。它允许使用代币来共享无线路由器。Finzer 为这项事业获得了 `Y Combinator` 的支持。然而，几个月后，他改变了业务方向。由于 `CryptoKitties` 的流行，他看到创建一个 NFT 市场有着更大的机会。于是，`OpenSea` 诞生了。

虽然公司在成长，但直到 2021 年才迎来转折点。结果，Finzer 毫不费力地从精英风投公司那里筹集到了巨额资金。到 2022 年初，他以惊人的 133 亿美元估值筹集了 3 亿美元。^(¹⁶⁹) 主要投资者包括 `Paradigm` 和 `Coatue Management`。

有了如此强大的支持，`OpenSea` 得以招募到顶尖人才。该公司成功聘请了前 `Lyft` 首席财务官 Brian Roberts 和前 `Meta` 商务副总裁 Shiva Rajaraman。

在融资时，`OpenSea` 当月的交易量超过 24 亿美元。这同比增长了超过 600 倍。^(¹⁷⁰)

根据 Finzer 的一篇博文：“我们专注于通过引入功能和简化流程来抽象化区块链的复杂性，从而降低 NFT 的准入门槛。我们也在加速多链支持，并优先进行改进，帮助人们通过更好的工具、分析和展示方式，来发现、管理和展示他们的 NFT。”^(¹⁷¹)

### Rarible

Alex Salnikov 拥有计算机科学学士学位和数据科学硕士学位。这为他在加密货币领域的职业生涯奠定了良好基础。他接着在相关领域创办了多家公司，其中一家成功售出。

他最新的创业项目是 `Rarible`，他在其中担任产品主管。他也过着游牧式的生活。也就是说，他没有固定住所，主要生活在世界各地的爱彼迎（Airbnb）民宿中。

他于 2019 年夏天联合创立了 `Rarible`。“我们不断遇到三类人，”他说，“一类认为加密货币太复杂。另一类认为这项技术没有必要，他们只需要信用卡和借记卡。还有一类是死忠粉。”^(¹⁷²)

对于 Alex 来说，他看到了机遇。他认为，为那些认为该技术太难的人构建一个加密货币市场的时机已经成熟。

那时，钱包的使用已经变得更简单。“但加密货币能带来的乐趣并不多，”Alex 说，“这就是我们专注于 NFT 的原因。”

他的团队能够构建一个平台，让用户可以轻松创建和销售 NFT。“用户只需点击几下就行，”Alex 说。

他们还创建了一个 API。这使得任何人都能创建自己的 Web3 市场。

目标市场是加密艺术。为了建立用户群，Alex 和他的团队通过自主推广来积累用户基础，例如在推特上联系相关社群。

这个策略奏效了，`Rarible` 开始成长。然而，市场很快发生了转变。用户对个人资料 NFT 更感兴趣了。“我们不得不改变公司的方向，”Alex 说，“但这也是初创公司的优势所在。”

在建立成功的 Web3 公司方面，他认为创始团队必须是“加密原生”的。这意味着他们必须对技术有深刻理解，比如阅读关键的白皮书。如果对 Web3 的兴趣主要是为了跟上下一波热潮，他认为这是行不通的。“你需要相信 Web3 的核心原则，比如去中心化，”Alex 说，“这必须是你的基因的一部分。”

2021 年，`Rarible` 宣布了其 A 轮融资。该公司从 `Venrock Capital`、`CoinFund` 和 `01 Advisors` 筹集了 1420 万美元。^(¹⁷³) 当时，公司的销售额已达到 1.5 亿美元。



### Pastel Network

安东尼·乔治亚德斯和杰夫·伊曼纽尔于 2018 年共同创立了 Pastel Network。他们认为 NFT 市场即将迎来爆发。然而，他们也看到了其中固有的问题：NFT 平台主要依赖现有的 Layer1 区块链，而这些区块链并非为 NFT 的特殊需求而构建。

通过 Pastel Network，创始人创建了一条特定于应用的 Layer-1 区块链。它降低了高昂的交易费用，提高了安全性并减少了低效问题。此外，还添加了一些功能，用于提供防伪保护、真实性认证和永久存储。

“从数字收藏品到媒体、文件、应用程序，用户和开发者能够认证资产的稀有性，并真正实现数据的永久存储，”乔治亚德斯说。“通过可互操作的开放 API（例如`Sense`和`Cascade`）提供的轻量级协议，可以轻松集成到现有网络中。广泛的 Web3 应用可以直接在 Pastel Network 上构建，使开发者能够享受可扩展的注册功能、存储流程以及更广泛生态系统的安全性。”^(¹⁷⁴)

在安全基础设施方面，它采用了多层技术协议用于共识机制。所有网络节点都使用`工作量证明（PoW）`，而 Pastel 的去中心化`超级节点`网络目前则使用一种创新的`权益证明（PoS）`形式。

其商业模式涉及适度的交易费用。其中一小部分会被销毁。这意味着这部分代币从供应中移除，以此作为避免代币通胀的一种方式。

集成合作伙伴无需支付许可费用。这对于该技术的推广至关重要。

对于给 Web3 创业者的建议，乔治亚德斯表示：“首先也是最重要的，创业者需要带着长期的眼光和目标进入这个领域。技术、产品开发和用户采用可能需要多年时间才能见效。人们很容易陷入短期的兴奋之中。然而，在构建新技术和新公司时，保持谨慎和耐心至关重要。考虑到这一点，创业者确实应该对基础知识有很好的理解——比如什么是区块链或私钥，智能合约如何工作，以及 Web3 究竟意味着什么？在此基础上，你才能开始更好地了解生态系统中的各种参与者，他们在 Web3 大格局中的位置，以及在哪里可以最大化创新。”

他还认为，创业者需要将安全作为重中之重，并且必须从第一天就开始。“Web3 本质上不是一个危险或高风险的空间，但它仍处于早期阶段，恶意用户正利用这一点来诈骗和攻击项目及消费者，”乔治亚德斯说。“最大化安全性的最重要方法之一是优先考虑基础设施和去中心化。大多数失败都源于过度中心化的网络，由一小部分利益相关者管理，这类网络更容易受到攻击。为了避免中心化故障点，充分分散参与达成共识和执行网络操作的参与者至关重要。节点也应分布在不同用户和实体之间，同样是为了避免单个实体持有过多节点，从而拥有过多控制权。”

### Rally

罗布·佩特罗佐表示，他共同创立 Rally 是因为错过了机会。对他而言，这源于对篮球巨星迈克尔·乔丹的喜爱。

青少年时期，他就知道乔丹在特定比赛中穿了哪双运动鞋，开了哪辆车去球馆，以及几乎每一项数据。“即使在很小的时候，我就能告诉你一些显而易见的事，比如‘迈克尔·乔丹的新秀卡有一天会价值六位数’，”他说。^(¹⁷⁵)

但对佩特罗佐来说，他既没有渠道接触收藏品，也没有资源去购买。

这就是 Rally 诞生的原因。它于 2016 年推出，是一个快速发展的市场，你可以在其中买卖超稀有资产（包括 NFT）的股权。这类似于你使用像`Robinhood`这样的应用购买股票。“在 Rally，我们希望为那些能够预见未来、更愿意投资于自己真正关心和痴迷的物品，但却不知从何入手的人们解决这个问题，”佩特罗佐说。“能够将那份怀旧或那些抱负转化为股权，并允许人们以自己感觉舒适的任意价格做到这一点，这是在兑现我对 10 岁的自己许下的‘总有一天我会拥有它’的承诺。”

在创办公司之前，他作为一名数字设计师有着成功的职业生涯。他曾与音乐、出版和生物技术等行业的客户合作。这段经历对于创建一个有吸引力的市场至关重要。

诚然，`OpenSea`在 NFT 二次交易中占据着明显的统治地位。但佩特罗佐意识到还有其他吸引人的类别可以瞄准。

“我们能够让用户接触到最昂贵、最知名的 NFT，而无需让他们接受一个全新的、通常令人生畏的支付流程，”他说。“这使我们能够承接像 Bored Ape Yacht Club 和 CryptoPunks 这样的项目，并以每股约 10 美元的价格进行首次公开募股（IPO），通常能让单个 NFT 的投资者数量超过该项目所有 NFT 的总持有者数量。我们的第一个 CryptoPunk NFT 首次公开发售迅速售罄，被近 4000 名独立投资者购买。而当时，所有 10000 个 CryptoPunks 的总独立持有者数量也只有大约 3800 人。”

他给 Web3 创业者的建议很简单。有趣的是，这与耐克标志性的口号相似：直接开始创业。

“Web3 空间是创建最小可行产品并在社区成长过程中与之共建的真正体现，”佩特罗佐说。“完美是敌人，许多大型项目的传奇故事仍在书写中。如果你能找到方法为大众带来任何独特的东西，或者为未被充分服务的受众构建产品，那么进入 Web3 框架的途径从未如此之多。根据你所服务社区的反馈来构建、发布和迭代。”

2021 年底，Rally 宣布完成一轮 1500 万美元的融资。^(¹⁷⁶) 领投方是`Wheelhouse`。总体而言，该公司已筹集超过 6500 万美元。



## 结论

在本章中，我们介绍了面向消费者的 Web3 应用程序的各种方法。首先，我们了解了 `NFT`（非同质化代币）。它们是一种代表数字资产所有权的方式。

一个关键优势在于，`NFT` 的创建者可以获得更高比例的收入。同时，这也有助于与用户建立更紧密的联系。

但 `NFT` 的价值通常是主观且不稳定的。此外，还存在黑客攻击和欺诈的风险。不过，企业正在投入资金寻求缓解这些问题的方法。

尽管如此，`NFT` 很可能会继续流行下去。好消息是，在处理这些问题方面已经取得了更多进展。

`NFT` 也很可能成为 Web3 风格社交网络和游戏平台兴起的关键。它们将提供一种视觉上很有趣的方式来拥有数字物品并进行交易。它们还将允许用户将 `NFT` 转移到其他平台，这应该能带来更多价值。

就 Web3 游戏而言，它们已经成为早期的成功案例之一。话虽如此，该行业在数字物品交易和部署创新商业模式方面一直处于领先地位。Web3 游戏也很可能成为新用户参与加密货币领域的入口。

品牌也正在研究利用 `NFT` 和 Web3 来与客户建立联系。这可能是应对隐私法规问题的有效方式。
