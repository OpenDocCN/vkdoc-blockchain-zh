# 总锁仓价值（TVL）

DeFi 的一个关键指标是总锁仓价值（TVL）。这显示了一个协议或网络的总资产量。

虽然这确实能很好地反映成功程度，但 TVL 远非完美。面对现实吧，这个价值可能会突然下跌甚至崩溃。一个项目的用户转移到另一个项目的情况并不少见。维护一个 Web3 社区绝非易事。

因此，查看其他指标也很重要。其中一些包括以下内容：

*   社区的参与度。
*   协议与其他应用程序、交易所和钱包的集成。
*   代币持有者数量的增长。这可能意味着积极的口碑营销。
*   开发者数量的增加。

好了，至于本章的剩余部分，我们将看看 DeFi 领域一些有趣的初创公司。

### Tribal

阿姆尔·沙迪和穆罕默德·埃尔卡斯塔维是埃及顶尖的创业者。但他们亲身经历了在中东地区经营业务的挑战——尤其是在获取融资方面。文书工作和监管通常既昂贵又耗时。对于规模较小的企业来说，融资标准往往过于苛刻。

但沙迪和埃尔卡斯塔维看到了利用区块链创建新型平台的机会。^(¹⁸⁶) 这家名为 Tribal 的公司于 2019 年底推出。

Tribal 的系统（包括一张企业信用卡）使用了一套复杂的人工智能审批流程。^(¹⁸⁷) 这极大地简化了流程。Tribal 还提供各种服务，如本地和国际电汇以及支出管理。其客户遍布 22 个国家。

2022 年初，该公司在 B 轮融资中筹集了 6000 万美元。^(¹⁸⁸) 投资者包括软银拉丁美洲基金、BECO Capital、QED Investors 和 Rising Tide。总融资额已超过 1.4 亿美元。

软银拉丁美洲基金的管理合伙人舒·尼亚塔表示：“Tribal 正在利用加密货币从根本上改变支付和借贷的游戏规则。很少有公司能够以如此创新且无缝的方式连接传统金融和 DeFi。”^(¹⁸⁹)

### Valora

21 世纪初，杰基·博纳从加州大学洛杉矶分校毕业，获得了国际关系和历史的文学士学位。然而，她很快成为了一名成功的营销专家。^(¹⁹⁰) 她在谷歌工作了七年多，随后又去了推特和声破天。在这些公司，她学到了如何实现增长黑客以及如何开发易于使用的应用程序。在此期间，她曾在遍布四大洲的九个国家生活过。

到 2021 年，博纳成为了一名科技创始人。她推出了 Valora，专注于为无国界支付提供移动优先的加密钱包。她是首席执行官，并已筹集了约 2000 万美元。

该应用程序适用于新手以及有经验的加密货币交易者。它在 100 多个国家可用，费用低廉，仅为 0.001 美元。是的，使用它几乎免费。这不仅有助于推动采用，也使其在交易金额通常较小的发展中国家变得实用。

该应用程序的简洁性是另一个关键特性。您无需使用 42 个字符的钱包地址。相反，您只需一个电话号码即可汇款。

Valora 拥有一个利润丰厚的奖励系统，奖励金额为余额的 25%。这有助于鼓励更多交易。

Valora 使用 Celo 区块链，该区块链源于以太坊。然而，它们彼此独立。这意味着 Celo 钱包无法与以太坊网络连接。

Celo 专注于稳定币，例如基于美元、里亚尔和欧元的稳定币。其次，完全支持 ERC-20 代币。这允许移植去中心化应用。例如，以下是 Valora 提供的一些应用：

*   Celo Tracker：管理您的 NFT 和其他数字资产组合。
*   Bidali：使用流行品牌的礼品卡。
*   ChiSpend – 超级应用：您可以使用加密货币购买现实世界的产品和服务。
*   Celo Tax：提供您的交易税务分析。

Celo 也环保。请注意，它拥有碳负排放的基础设施。

### TrustToken

拉斐尔·科斯曼曾在 Palantir、Kernel 和谷歌等公司担任机器学习工程师。但他认为加密货币是一个巨大的机遇，并于 2017 年创立了 TrustToken。^(¹⁹¹) 他想创建一家下一代金融服务公司。科斯曼为他的初创公司筹集了约 3400 万美元。

TrustToken 有两个主要部分。首先是 TrueFX，这是一套针对美元、英镑、加拿大元等的稳定币。其次是 TrueFi，这是一个建立在以太坊区块链上的 DeFi 平台。其重点主要是提供无抵押贷款。

请记住，这相当独特。原因是加密货币借贷通常都是超额抵押的。虽然这提供了更高的安全性，但它确实限制了市场的潜力。请注意，这个机遇是巨大的。全球范围内的传统无担保贷款约为 11 万亿美元。^(¹⁹²)

TrustToken 的贷款还具有为贷款人提供更高回报的优势。但需要有办法为信用评分提供标准。这类似于传统贷款中的 FICO 信用评分。

其工作原理如下：TrustToken 为贷款人提供了一个易于存储资金的系统。然后借款人发送信贷请求。利率和限额基于借款人的信用评分。但社区必须对每笔贷款进行投票。本质上，这是一种基于声誉的贷款分配方式。不过，如果借款人不偿还贷款，将面临潜在的法律后果。

截至 2022 年初，TrustToken 已处理超过 5.24 亿美元的贷款，并支付了超过 1750 万美元的利息。^(¹⁹³)



### Ox Labs

`Ox Labs` 是首批 DeFi 协议之一，也是加密货币交易所 `Matcha` 的运营方。据首席执行官兼联合创始人 Amir Bandeali 表示：“大约六年前我们创办 0x 时，就有一个论断，即一切有价值的东西最终都会被代币化。”^(¹⁹⁴)

`Ox Labs` 的核心业务是集成服务。借助这些服务，公司能够创建自己的 Web3 平台，例如允许使用代币。举例来说，该技术为 `Coinbase` 的 NFT 市场提供了动力。

`Ox Labs` 支持七条大型区块链，包括 `Ethereum`、`Polygon`、`Fantom`、`Avalanche`、`Optimism`、`BNB Chain` 和 `Celo`。此外，还与顶级钱包和应用程序进行了集成，例如 `MetaMask`、`Coinbase Wallet`、`Polygon Wallet`、`Brave`、`Matcha`、`dYdX`、`Zapper`、`Zerion` 和 `Shapeshift`。`Ox Labs` 已对超过 1570 亿美元的价值进行了代币化。^(¹⁹⁵)

2022 年 4 月，该公司筹集了 7000 万美元。在其投资者之一 `Greylock` 的一篇博文中，有如下表述：“在 `Greylock`，我们特别乐于支持那些武装建设者（开发者和创业者）的公司。`0x Labs` 就是这样的公司之一。虽然有些人可能认为去中心化交易所不可行，用户会倾向于少数几家已经领先的交易所和钱包，但我们持相反观点。我们相信，我们正处于在新的轨道上重建金融基础设施的非常早期阶段，未来的最低交易量将囊括所有现有的传统金融资产，而预期的最终状态将包括跨社交网络、游戏、其他现实世界资产等的所有权经济。”^(¹⁹⁶)

## 结论

DeFi 的规模仍然相对较小，但对创业者而言潜力巨大。该技术可以实现更简化、更快速、更便宜的金融交易。DeFi 还可以极大地扩展金融服务的可及性，尤其是在发展中国家。

但当然也存在风险。由于监管较少，已经出现了安全漏洞和故障问题。然而，这意味着创业者可以构建具有更好安全性和稳定性的系统。

机会不仅限于 B2C 应用。B2B 领域实际上可能是一个更大的市场。一些全球最大的金融服务公司一直在投资 DeFi。
