在前几章中，我们专注于部署智能合约、去中心化应用和令牌。在本章中，我们将简要讨论作为数据库的区块链，以便更深入地了解链本身是如何部署的。

## 私有链与许可链

`私有链`本质上是一个通过点对点的以太坊协议实现的云数据库：它是一个你可以控制并可授予他人访问权限的独立仓库。

这应与`许可区块链`形成对比，后者像企业软件应用一样，具有可被中央管理员设置的、带权限的明确定义角色。

总体来看，私有链在本质上绝不比云数据库更优越。在实践中，以太坊协议的效用源于将不同的群体聚集在一起，共享安全的基础设施，而不是重复劳动。目前，以太坊网络已全面运行，但其规模尚未达到现有 Web 应用提供商可以迁移的程度。但这仅仅是在经过两年的开发之后，正如你将在第 11 章中看到的，未来的里程碑相当令人难以置信，并且正按计划实现。

相比之下，HTTP Web 自 1989 年以来一直在开发中。¹ 去中心化云存储、命名空间和 HTTP Web 的其他常见元素尚未在以太坊 Web 中得到重现，但很快将会实现。让我们继续前进，这样你就可以创建自己的定制区块链，以更好地理解它们的工作原理。
