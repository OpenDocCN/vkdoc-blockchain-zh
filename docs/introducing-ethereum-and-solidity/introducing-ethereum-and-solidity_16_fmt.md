# 3. 以太坊虚拟机

## 以太坊虚拟机（EVM）是一台全球性计算机，任何人都可使用，只需支付少量费用（以以太币计价）

Chris Dannen (1)美国纽约州布鲁克林

EVM 是一个单一的、全局性的 256 位“计算机”，其中所有交易在网络中的每个节点上都是本地的，并相对同步地执行。它是一个全球可访问的虚拟机，由许多较小的计算机组成。

这台任何人都可以通过节点或钱包应用程序访问的巨大计算机，使得近乎即时地移动任意大额价值（金钱）变得简单。尽管任何人都可以使用这个全球虚拟机，但任何人都无法在其中制造假币，或未经许可转移资金。

如果让整个 EVM（所有那些节点）复制相同的交易并在数千台独立计算机之间忠实地维护相同的状态似乎很浪费，那么有必要建立一个适当的比较基准来了解当今金融服务 IT 的工作方式。相比之下，EVM 简直是简洁高效的典范！更重要的是，所有这些工作并非徒劳。事实上，正如您将在本章中看到的，正是这些工作的证据才真正保障了网络的安全。
