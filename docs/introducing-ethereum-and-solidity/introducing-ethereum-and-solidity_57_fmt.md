# 软件开发

毫无疑问，以太坊最具颠覆性的潜力在于其承载软件和服务的能力。

- **云计算**：随着数据存储功能于 2017 年底引入 `EVM`，该网络将最终开始成为一个功能完备的 web 应用托管环境。分布式共识协议因其`无需信任`的架构而成为优秀的云计算平台：无需担心复杂的网络配置来保障数据安全或处理高流量。这类系统可能并非适用于所有类型的应用，但某些易于并行化的软件则可以胜任。
- **长期应用托管**：一些金融合约的编写如同时间胶囊。但如何确保一个计算机程序在 50 年或 100 年后仍然存在并能执行？一种方法是将其开发为公共服务；个人可以托管文档，并确信即使他们早已不在，网络仍将运行。
- **廉价、弹性且无需审查的公共文档托管**：重要文档，如出生证明、纳税申报表、法院传票、移民表格、健康记录及其他非结构化数据，可以轻松加密并存储在区块链中，供第三方进行特定检索。如今，大多数背景调查活动和信用报告由私人组织负责。这至少可以说是有问题的；公有链可能提供一个“永久网络”，让这些文档能够为后代而托管。
