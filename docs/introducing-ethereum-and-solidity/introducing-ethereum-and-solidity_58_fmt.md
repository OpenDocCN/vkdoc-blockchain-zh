# 游戏、博彩与投资

区块链开发者已经推出了一些可证明公平的机会游戏，以展示网络的能力。未来，这类应用可能会扩展到以下场景：

-   **点对点赌博**：撇开法律不谈，建立覆盖广泛地理区域的赌博网络非常困难，因为很少有人信任他们的庄家会持有大笔资金。在以太坊中，通过纯软件（一种对智能合约的通俗描述）创建赌注是极其容易的。例如，设想一个简单的赌博合约，其赌注是非随机数（`nonce`）的值，或者其他类似在链维持共识过程中自然发生的随机事件。
-   **预测市场**：预测市场试图利用大规模的博彩市场来确定事件的现实结果。一个基于预测市场来做决策，以期实现自动化并提高效率的政府，被称为`未来学统治`（`futarchy`）。
-   **稳定价值的加密资产**：加密货币以其剧烈波动而闻名。由于它们的交易不受任何第三方中介，因此没有退款机制，这使得它成为快速流动的做市商吞噬经验不足交易者的完美媒介。创建一个人们会持有、储蓄甚至传给子孙后代的稳定资产，是迄今为止没有任何金融机构敢于挑战的难题。

以上是对你可以在以太坊网络上用智能合约和 `DApp` 构建的内容的一个良好概述，但这远不是一份详尽的清单。
