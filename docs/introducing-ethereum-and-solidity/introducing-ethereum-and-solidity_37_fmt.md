# 熟练掌握命令行

在 Ubuntu 上安装 `Geth`（具体步骤见后文）后，你将直接进入一些练习环节。这些练习假设你使用的是 macOS 或 Ubuntu 的终端应用程序。本文不会讨论 Windows 系统中的 Geth 命令，但你可以在 [`clients.eth.guide`](http://clients.eth.guide) 找到相关说明。

以下指南是为首次使用命令行的用户编写的。如果你符合这种情况，请立即注意以下几点。

当你首次打开终端应用程序（macOS 和 Ubuntu 中位于应用程序文件夹内）时，会看到一个闪烁的光标。这表示计算机已准备好接收指令。

> **注意：** 在此界面中，计算机一次只能处理一件事。当你输入一条命令后，可能需要几秒钟才能完成。在此期间，屏幕上会有文本飞速滚动。不要惊慌；这是正常现象。随意尝试 `Geth` 或命令行并不会损坏你的计算机。
