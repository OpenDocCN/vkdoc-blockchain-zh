# 11. 高级概念

## 以太坊协议的未来与起源

Chris Dannen¹  
(1)美国纽约州布鲁克林

一本介绍以太坊和 Solidity 的书如果不提到围绕以太坊发明者、同时也是其他几个知名区块链项目合作者的维塔利克·布特林（Vitalik Buterin）所形成的新兴个人崇拜，那将是不完整的。

## 谁在引领软件开发者走向去中心化？

也许对布特林的最佳描述来自于作家摩根·佩克（Morgan Peck）于 2014 年春季在网络杂志 Backchannel 上发表的一篇关于他的文章。¹ 文章描述了作者与这位以太坊联合创始人的初次会面：

> 布特林是唯一醒着的人。他坐在外面的躺椅上，全神贯注地工作。我没有打扰他，他也没打招呼。但我还记得他当时给我留下的印象。这个骨瘦如柴、四肢修长的 19 岁男孩，像螳螂一样悬垂在他的笔记本电脑上方，以惊人的速度对其进行着灵活而致命的重击。
>
> 结果，布特林正是大家当初来到这里的原因。两个月前，他发表了一份白皮书，描述了一项难以想象的雄心勃勃的技术——这项技术超越了比特币实现不可阻挡、无中介数字支付的使命，构想了所有类型自主软件的运行平台。

在这个自由开源的网络协议运动的核心，布特林的智力领导力是独一无二的。也许最令人震惊的事情，除了该项目取代 HTTP Web 的惊人野心之外，就是其速度之快。

以太坊项目于 2014 年启动，2015 年开始运行，到 2016 年成为仅次于比特币的第二大加密货币网络。所有以太坊基金会成员的当前名单可在 [www.ethereum.org/foundation](http://www.ethereum.org/foundation) 找到。

2017 年，核心开发团队计划推出其他组件，使以太坊达到与我们今天所熟知和喜爱的 web 应用程序同等的水平——但具有前几章所述的那些惊人新能力。

本章的其余部分涉及以太坊路线图，以及一些尚未解决和尚未构建的组件。

如果你想在此停下，深入研究当今以太坊网络背后的数学、经济学和商业原理，那么以太坊博客是阅读长篇、深度文章的最佳去处，布特林在其中阐述了他对协议一些核心概念的思考。
