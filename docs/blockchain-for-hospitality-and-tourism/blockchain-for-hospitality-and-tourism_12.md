# 4. 踏入 Web3

咨询公司和行业专家一直在密切关注区块链技术的崛起，因为它有潜力颠覆众多不同的行业。区块链甚至被称为我们这个时代最具颠覆性的技术，因为它有可能彻底改变当前的经济结构。

区块链能够在金融、供应链管理或医疗保健等不同领域实现去中心化、透明且安全的交易，这为革新传统行业提供了可能。通过消除中间商并允许用户之间直接互动，区块链有潜力提高效率、降低价格并增强业务交互的安全性。

区块链也被视为一项基础性技术。它可以建立新型的信任、价值交换和治理模式，从而催生更强大的经济和社会机构。凭借其去中心化的特性，该技术可用于创建新型的去中心化应用、DeFi（去中心化金融基础设施）、身份识别系统等。

从本章开始，我们将重点关注区块链所带来的具体应用和发展。你可能会感到惊讶，但这项技术确实是当今许多热门创新的基础。事实上，区块链是 Web3 的支柱——这是建立在去中心化账本之上的新一代互联网。

我们将首先定义 Web3，并简要介绍其发展历史。然后，我们将从宏观角度审视 Web3 如何应用于酒店和旅游业，着重强调它所带来的益处和机遇。请将本书的这一部分视为一个引言、一个概览。许多 Web3 元素，例如加密货币支付或 NFT，将在后续章节中单独介绍，因为该领域已有诸多发展，值得深入探讨。

## 什么是 Web3

互联网的兴起和演变为酒店和旅游业带来了巨大的变化。在过去二十年里，该行业见证了客户与企业互动以及进行旅行预订方式的转变。随着 Web3 的到来，该行业必须为又一次巨大的变革做好准备。Web3 以去中心化、透明化和安全性为特征。它为酒店和旅游组织提供了各种机会，从实现安全交易到创造独特且个性化的消费者体验。

顾名思义，Web3 是互联网的第三代。第一代互联网，Web 1.0，始于 20 世纪 90 年代初万维网问世之时。你可能还记得它：笨重、静态、只读。你可以消费信息，但无法与之互动。

情况在 21 世纪发生了巨大变化，随着社交媒体、移动设备和云计算的兴起。Web 2.0 以交互性、用户生成内容和个性化为特征，并由 Facebook、谷歌或亚马逊等中心化服务和平台主导。我们大多数人真的无法想象没有这些公司提供的便捷强大的在线工具的生活。然而，围绕它们从用户那里收集的数据量、对隐私和审查的担忧，以及权力集中在少数大型企业手中，也存在着诸多争议。

Web3 有望改变这一切，创造公平的竞争环境。《哈佛商业评论》的一篇文章^(¹⁵)将 Web3 定义为“一个去中心化的互联网，用户在其中拥有并控制自己的数据，数字资产在点对点之间直接交易。”用户可以对自己的数据和在线体验拥有更大的控制权。基于区块链的社交网络和市场使用户能够直接相互交流，无需中间商或中心化权威机构。这种模式也提供了更好的隐私和安全性，因为消费者可以控制自己的数据，并保护自己免受中心化公司窃取或滥用数据的风险。

以太坊的联合创始人之一加文·伍德博士在他 2014 年的白皮书^(¹⁶)《以太坊：下一代智能合约和去中心化应用平台》中首次描述了 Web3 的概念。

伍德设想了一种新型的互联网，一种允许开发去中心化应用（dApps）的互联网，这些应用运行在区块链网络上，而不是中心化服务器上。伍德的想法是构建一个互联网，用户可以在其中无需中间商就能相互交易，并利用自动执行交易条件的智能合约。

自伍德的白皮书发表以来，Web3 一直是科技界的热词。根据德勤的分析^(¹⁷)，Web3 有潜力颠覆包括金融、医疗保健和政府在内的众多行业，因为它能够实现新的商业模式并提高效率。有些人声称它不仅仅是一项新技术，更是我们作为个人和客户与互联网互动的一种全新方式。Web3 的主要规则是用户控制——你拥有并管理自己的数据。你可以决定谁能访问它。你可以将自己的数据和数字资产货币化。在 Web3 中，用户参与去中心化治理模式，并对系统和应用的开发与运营方式拥有发言权。这与 Web2 截然不同，在 Web2 中，我们仅仅扮演内容消费者的角色，对我们的数据和在线体验几乎没有或根本没有控制权。

这种转变是由构成互联网新世界的几个核心关键元素促成的：

*   **去中心化技术**，例如区块链，消除了对中间商和中心化权威的需求，使点对点交易成为可能
*   **互操作性协议**，例如以太坊或波卡（Polkadot），使得不同区块链网络之间能够通信
*   **密码学**，包括数字签名和密钥加密（参见前几章），用以保护数据和交易安全

Web3 在很大程度上依赖于人工智能、3D 图形以及虚拟和增强现实等新技术。我们将在接下来的章节中讨论它们的重要性和应用。

现在，让我们来看看引入 Web3 对酒店和旅游业可能产生的影响。

## Web3 为酒店和旅游业带来的机遇

此时，你可能会想，我们为什么要讨论 Web3，以及这个话题与酒店和旅游业有何关联。让我们在此稍作停顿，思考一下互联网的重要性以及过去二十年它给这个行业带来的变化。我们都看到了企业运营和与客户互动方式的重大转变。一切都转移到了“线上”，世界突然变得小得多，也更易于访问。你无法想象一家没有网站、在线预订平台、在线支付和社交媒体账户的酒店。

现在，随着我们逐步迈向 Web3，酒店业需要为另一次巨大的变革做好准备。基于区块链的创新为酒店和旅游业面临的许多挑战提供了解决方案。然而，其中一些具有改变游戏规则的潜力，可能会对该行业的发展方式产生深远影响。

现在是时候开始将技术转化为商业和用例了。在表 4-1 中，你将看到几个区块链和 Web3 的应用。请注意，此列表并未详尽无遗，并可能会随着技术的成熟而演变。
