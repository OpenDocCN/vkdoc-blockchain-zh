# ABC 零售

ABC 零售早在 2014 年就意识到通过 `Instagram` 进行销售的机遇，并且该公司不断围绕移动端和社交媒体增加更多功能以提升客户体验。他们利用 `物联网`、大数据和分析技术分析大量客户数据，打造个性化客户体验。其供应链不仅延伸至自有门店，还覆盖合作伙伴。这得益于他们构建的能力，即所选择的客户互动与业务生产力提升这两大商业战略的效力。

## 总结

本章讨论了企业为在市场中保持相关性而应采用的三项商业战略：第一是客户互动战略，第二是业务转型战略，第三是业务生产力提升战略：

*   **客户互动战略**，也称为市场进入战略，重点完全在于加强与客户的关系。
*   **业务转型战略**意味着企业不应仅局限于所销售的产品和服务。他们需要考虑客户的需求与问题，并通过重新定义业务来寻求解决方案，必要时还需进入新领域或业务范畴。
*   **业务生产力提升战略**意味着在企业任何环节寻找提升效率的机会，例如通过优化现有运营或生产线来削减成本，或通过减少人工干预来改善当前运营。

一旦选定商业战略，企业便会应用 `物联网` 来实现业务目标。`物联网` 路线图通过识别特定业务流程来制定，这些流程要么能让企业在市场中（与竞争对手）形成差异化，要么能改善企业与客户的互动，或帮助企业提高效率。

下一章，我们将讨论业务转型模型，利用该模型，将选择那些对业务战略产生影响的业务流程，并运用 `物联网` 进行转型。
