# 反思债务型资本主义

希望本章前文提供的论据和事实能帮助我们厘清为何需要以更深刻的自省态度看待债务。债务本身重要且有用，但过高的债务水平会拖累经济增长，压低工资水平，降低生活标准，并在遭受危机或突发事件时使借贷变得更加困难。这不仅适用于家庭，也同样适用于国家。不断膨胀的债务水平的净效应，便是繁荣与崩溃的周期性循环，随后便是经济衰退。我们投资房地产的倾向、全球失衡以及社会内部日益加剧的不平等，是过去几十年来信贷密集型周期持续存在的根本原因。

继 2008 年危机之后，监管机构近年来的改革正在逆转历史的车轮。在里根-撒切尔时代，金融市场摆脱了大量限制。这一举措在九十年代的克林顿-布莱尔时代再次重演。虽然布莱尔政府于 1997 年将设定利率的责任从财政大臣移交给了独立的`英格兰银行`，但 1999 年美国《格拉斯-斯蒂格尔法案》的废除实际上与这一举措遥相呼应。《沃尔克规则》的颁布限制了美国银行进行某些对客户无益的投机性投资，这标志着回归到更古老的银行监管形式——那时存款不能用于银行的自营交易。

尽管这些规定以及 LIBOR 丑闻后更严格的监管措施，确实对商业银行参与投机活动的能力施加了新的限制，但它们也与量化宽松（QE）以及量化与质化宽松（QQE）等非常规措施相伴随。这并非是说监管机构和央行行长们做错了。在 2008 年危机事件之后，以超低利率向经济注入资金总比不采取任何行动要好。但这再次重演了与债务共舞的戏码，因为由于当前的`行事方式`，政府必须为它们注入经济的新增货币向中央银行借款并支付利息。结果，实施量化宽松的收益最终成为央行的收入，并可计入利润。

廉价资金的供给也加剧了社会内部的不平等，因为用于银行救助和刺激经济的公共资金，为那些拥有金融资产的人推动了繁荣。当量化宽松通过增加货币供应量从而推高资产价格来刺激市场时，得益最大的是那些拥有最多资产的人。除了使基于债务的货币循环永久化之外，由于资产所有权的不平等，其收益也向富人倾斜。资产价格的上涨也意味着，那些愿意贷款购买这些资产的家庭，现在需要承担比量化宽松前更高的债务水平。因此，公共债务的减少被私人债务的增加所抵消。

要打破这一循环，需要从根本层面上对资本主义进行反思。我们不仅需要思考一种更少依赖信贷的增长模式，还需要重新审视我们今天使用的理论和模型，因为它们相对于当今金融市场的复杂性和多样性而言，已经明显过时了。此外，需要采取措施来解决债务发行的根本原因、不成比例的问题，以及由部分准备金银行和影子银行所助长的随之而来的不稳定性。尽管已经采取了许多措施来解决其中一些问题，但这些措施都是被动反应而非主动预防。鉴于自动化水平不断提高以及工作被机器人取代的威胁，这种做法无异于试图用创可贴而不是止血带止住流血不止的动脉。

与此同时，金融科技和基于区块链的解决方案日益壮大，为我们提供了一系列在技术层面和管理层面的可能解决方案。但由去中心化应用和代币化假名身份提供的解决方案，也需要重新思考监管规则手册。本书的后续部分将阐述重新思考规则手册为何以及如何对社会有益，以及如何利用这些技术来更好地衡量和管理我们的金融体系和经济，同时为未来的问题提供解决方案。
