# 宏观经济影响与中央银行

本书的第三和第四部分从更宏观的经济层面审视了这项技术的影响。在简要讨论货币政策和财政政策之后，对中央银行可能受到的影响进行了回顾。我们还将分析多种货币、去中心化账本和加密控制系统对中央银行意味着什么后果。这为理解需要掌握哪些措施、工具和理论以创建新的货币经济学框架奠定了基础。在此，读者还将接触到无现金经济的概念及其兴起。除了描述无现金系统在控制过度债务和经济污染方面的影响外，读者还将了解到哪些新的科学分支将帮助我们衡量和管理这一系统。

虽然经济学是一门古老的学科，但用于理解这些多层面生态系统的方法，并未充分尊重其交织的网格结构所带来的复杂性。在当今数据丰富的环境中，我们需要的是一种能够比当前经济模型具有更高数学精确度和更高概率识别系统性风险的方法。随着世界在本质上日益数字化，迫切需要一种观察和衡量经济系统的新方式。不仅我们的技术已经过时，这些技术所依据的理论同样如此。
