# 脚注

[1] 皇家学会是一个由世界上许多最杰出的科学家组成的团体，也是历史最悠久且持续存在的科学院。

[2] 参见《技术新颖性特征与发明的未来影响》，Kim 等人，（2016），《EPJ Data Science》。

[3] “组合进化”一词由科学理论家 W. Brian Arthur 创造，他也是复杂经济学的创始人之一。与托马斯·库恩的《科学革命的结构》类似，亚瑟的著作《技术的本质：它是什么以及它如何进化》解释了技术基于相互作用，并被组合成可以生长的模块化组件系统。由于具有模块化特性，它们相互结合，当一项技术达到组件和接口的临界质量时，它会进化进入新的领域，并根据与之相互作用的新自然现象而发生改变。总而言之，亚瑟的组合进化包含了发明、生物进化、行为模型、社会科学、技术变革、创新和社会学等概念。

[4] 即使是进化也离不开组合方法。以进化科学闻名的查尔斯·达尔文，其分类系统建立在卡尔·林奈（1707-1778）——分类学之父的工作之上。

[5] 微分分析仪由多个由电动机驱动的旋转圆盘和圆柱体组成，这些圆盘和圆柱体通过金属杆连接，需要手动设置（有时需要长达两天）来求解任何微分方程问题。
