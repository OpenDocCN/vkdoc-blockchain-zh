# 新兴研究综述

随着越来越多来自不同学科的学者开始思考类似问题，本书的另一个目的是为读者提供该研究领域进展的概要。任何试图涵盖融合所有这些主题和发展项目的人，都只能通过对众多研究人员、学者和政策制定者的工作进行广泛研究才能做到，这些人正在不同学科中重新思考这些主题。因此，尽管作者旨在为读者提供一个新的视角，但这种对资本主义的新描绘，正是那些正在与现状搏斗的思想家们智慧的结晶，没有他们的努力，本书便不可能存在。就此而言，本书有时可能看起来像一份冗长的学术报告，因为它融入了许多杰出新思想家的著作。作者在绘制这一蓝图过程中所面临的任何不可避免的批评，均由作者本人承担。



在阅读本书之前，必须记住，**区块链**不应被视为解决所有经济问题的万能药。目前还有许多其他技术也在改变金融和经济学领域。但由于技术具有相互促进的特性，区块链作为一种基础设施技术，能够与其他技术结合并放大其效果。因此，本书的最终目标是澄清当今数字世界中信息轰炸所造成的迷雾，让读者能够自行串联起线索，见证技术的融合。最重要的是，它试图确定区块链如何被用来为我们沉迷于债务的货币体系找到解药。

这项技术为我们提供了一个前排座位，让我们见证历史在眼前展开，并可能留下对未来的记忆。毕竟，历史不仅仅是过去，也是我们改变现在以影响未来的方式。
