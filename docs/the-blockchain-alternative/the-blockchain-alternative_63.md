# 遗传算法

遗传算法（GA）是一种启发式搜索方法，受达尔文进化论启发，用于人工智能和计算领域。它基于自然选择和进化生物学的理论（即选择、变异、遗传和重组）来寻找搜索问题的最优解。

遗传算法基于将染色体视为一串基因的经典观点。R.A. 费舍尔利用这一观点创立了数理遗传学，提供了指定特定基因在种群中传播速率的数学公式（Fisher, 1958）。费舍尔公式的关键要素包括（Holland, 2012）：

1.  为每个基因指定一组备选方案，从而确定可以进化的允许的基因字符串。
2.  逐代演化的观点，其中在每个阶段，个体种群会产生一组构成下一代的后代。
3.  一个适应度函数，为每个备选方案的字符串分配该染色体携带者将贡献给下一代的后代数量。
4.  一组遗传算子，特别是在费舍尔公式中的变异，它们修改个体的后代，使得下一代与当前代不同。

在解决约束和无约束优化问题时，经典算法在每次迭代中生成单个数据点。数据点序列随后逼近最优解。另一方面，遗传算法使用类似于自然选择的过程——它反复修改个体解的种群，并在每一步中，从当前种群中随机选择数据点，并将它们用作“父代”来产生下一代的子代。在连续几代中，种群“进化”到最优解（Holland, 2012）。

尽管是随机化的，但遗传算法绝非随机，相反，它们利用历史信息将搜索引导到搜索空间中性能更好的区域。遗传算法模拟个体在连续几代中适者生存的过程来解决问题。每一代由一组字符串组成，这些字符串类似于我们在 DNA 中看到的染色体。每个个体代表搜索空间中的一个点和一个可能的解。然后，种群中的个体经历一个进化过程。

遗传算法用于搜索庞大而复杂的数据集，因为它们能够通过解决无约束和约束优化问题，为复杂问题找到合理的解决方案。它们用于解决不适合标准优化算法的问题，包括目标函数不连续、不可微、随机或高度非线性的问题。遗传算法在混合（连续和离散）组合问题上也特别有效，因为它们比经典梯度搜索方法更不容易“陷入”局部最优。然而，它们的计算成本往往很高。
