# 区块链在金融领域的过渡性角色

本书的第二部分从区块链在金融领域所扮演的过渡性角色这一视角，对其进行了深入探讨。2008 年金融危机之后，金融领域一直处于动荡之中。一方面，政府和监管机构如今要求金融创新、税收和跨境交易方面具有更高的透明度。另一方面，技术进步正在解构金融领域，使得现有企业面临科技公司的挑战。虽然当前的讨论将区块链视为一种独立的技术，但本书的这一部分试图阐明，当它与当前正在解构金融领域的其他技术并列时，它所具备的聚合功能。通过将区块链视为一种能够利用其他金融学科（如今普遍称为金融科技）进展的工具，我们能够更全面地理解这项技术的角色。

在理解了金融领域如何在技术、债务和货币创造的背景下被解构之后，本书的第三部分试图确定这些范式转变对社会货币体系意味着什么。虽然人们常将金融领域出现这些变化的原因视为独立的波动，但它们实际上是相互关联的，而这种相互作用所产生的沉淀物，催生了重新定义经济学的需求。本节试图通过让读者理解包括区块链在内的不同技术如何改变金融领域并创造新的资本主义范式，来阐明这一定义。
