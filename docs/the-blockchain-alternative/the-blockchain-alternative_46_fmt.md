# 4. 复杂经济学：见证资本主义的新方式

> 如果我们假设均衡，我们就为能在经济中观察到的东西设置了一个非常强大的过滤器。在均衡状态下，根据定义，没有改进或进一步调整的空间，没有探索的空间，没有创造的空间，没有暂时现象的空间，因此，经济中任何需要调整——适应、创新、结构变化、历史本身——的东西都必须被理论绕过或排除在外。结果是一个优美的结构，但却是缺乏真实性、活力和创造性的结构。——W.B. 亚瑟，《复杂性与经济》，(2014 年)。

2008 年底，伊丽莎白女王访问了伦敦政治经济学院经济系。在访问期间，她向在场的经济学家们提出了几个简单直接的问题：“为什么没有人预见到（危机）来临？”“如果这些事情如此重大，为什么大家都错过了？”（皮尔斯，2008）。虽然回答在深度和广度上各不相同，但有两个事实是明确的：首先，对市场是金融体系最佳管理者这一信条毫不妥协的错误自信，导致了金融化的加剧和不可持续的债务积累。其次，这种对市场能力的信念是基于学术经济学和政策制定中僵化的正统观念。这些理论不仅未能预见到即将到来的危机，还将金融体系削弱到政府被迫救助那些本应是问题根源的银行的地步。可悲的是，尽管正在采取一些措施来解决这些缺陷，但机构中正在教授和实践的根本理论仍有待彻底质疑，更不用说完全修改了。

效仿女王问题的直率，前三章揭示了三个直截了当的事实——首先，日益增长的债务和经济金融化带来了系统性风险，并且目前是国家经济面临的最大挑战之一。其次，金融市场处于持续波动状态，新技术的加速涌现必将加剧这种变化状态的动荡。第三，当前管理市场、法规和政策的框架，在识别泡沫和崩盘的征兆方面显得笨拙且无能为力。本质上，经济思想中的主流流派，即有效市场假说和理性预期理论，正变得越来越不适合衡量和理解金融与宏观经济稳定。

随着技术使金融体系更加复杂，这些问题将变得更加频繁。因此，与其专注于维持现状的方法，我们需要的是从根本上重新思考我们如何研究经济学，以及如何理解和衡量资本主义。如果不采用新的思维方式，我们将被迫承认永无止境的繁荣、萧条和危机循环的威胁。简而言之，我们不能期望用对过去工具和理论的顽固信仰来解决今天的问题，更不用说解决明天的问题了。
