# 康德拉季耶夫周期与经济周期

在经济学中，康德拉季耶夫周期（以苏联经济学家尼古拉·康德拉季耶夫命名）是一种周期性现象，它将技术发明、扩张和最终替代的周期与其经济效应联系起来。尽管尼古拉·康德拉季耶夫于 1925 年率先研究了技术对价格、工资、利率、工业生产及消费的经济影响，但约瑟夫·熊彼特才是将其引入学术界的关键人物。

泰勒规则是一套指导中央银行如何根据经济状况变化调整利率的准则。该规则由经济学家约翰·泰勒提出，旨在调整并设定审慎的利率，以实现经济的短期稳定，同时维持长期增长。该规则基于三个因素：(i) 目标通胀水平与实际通胀水平；(ii) 充分就业水平与实际就业水平；(iii) 与充分就业相适应的短期利率水平（Investopedia）。其数学表达式为：`r = p + 0.5y + 0.5(p - 2) + 2`，其中`r`代表联邦基金利率，`p`代表通货膨胀率，`y`代表实际 GDP 偏离目标的百分比（Bernanke, 2015）。

**第一福利定理**：每一个瓦尔拉斯均衡配置都是帕累托有效的。  
**第二福利定理**：每一个帕累托有效的配置都可以被支持为一种瓦尔拉斯均衡。

第一和第二福利定理是福利经济学的基本定理。第一定理指出，任何竞争性均衡，或瓦尔拉斯均衡，都会导致资源的帕累托有效配置。第二定理则阐述了其逆命题，即任何有效的配置都可以通过竞争性均衡来维持。
