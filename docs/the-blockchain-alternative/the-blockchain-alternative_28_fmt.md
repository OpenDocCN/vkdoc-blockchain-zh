# 重新审视资本主义的现行定义

让我们首先提问：资本主义的定义是什么？对我而言，鉴于货币在我们生活方方面面中扮演的亲密角色，资本主义既是一种经济范式，也是一种文化建构。根据实证研究和哲学解释，资本主义社会是这样一种社会：市场由多个参与者组成，他们都在稀缺资源最优配置的前提下进行高效的价值交换操作。在此类市场中，法规和政策充当杠杆，需根据清晰规则恰当运用，以实现整个社会的繁荣成果。

然而，前两章分析所沉淀出的结果表明，社会的日益金融化已玷污了这一描述。如今，资本主义已从一个利用市场实现某些共同繁荣目标的社会，转变为一个一切皆可出售的社会，连风险也不例外。几乎可以说，我们在资本主义的道路上走过了头，过度使用了资本主义的三大治理杠杆——市场、法规和政策——基于对效率的误解或不健全理解，制造了不平等的矩阵。因此，发现没有任何历史定义能涵盖所有这三项杠杆，也就不足为奇了。

因此，要构建一个更新的资本主义定义，我们需要清晰且有辨析地理解这三项杠杆当前的局限性及其所扮演的角色。在重新定义市场角色时，我们需要引入不平等的概念，并扪心自问：由市场主导的收入和财富分配是否构成问题？不平等本身并非谬误；它是一种激励工具，能促进技能和知识的发展，进而将边缘化人群提升至此前未知的繁荣与幸福水平（有时被称为著名的“中产阶级”）。因此，不平等可被视为启动转型过程的催化剂，使强健经济体得以发展；过去三十年中国的发展，便是市场开放与消除不平等相结合如何带来变革性成果的绝佳例证。的确，过去三十年间，日益强劲的全球化势头使得利用社会不平等作为催化剂，将近十亿人从贫困中解救出来成为可能（Olinto & Uematsu, 2013）。但不平等也可能过犹不及。当不平等达到某个阈值时，会导致社会流动性降低，并影响未来经济增长，因为它使社会中的贫困阶层无法获得与创新密切相关的知识和技能（Chetty et al., 2015）。因此，尽管资本主义的市场杠杆解决了许多贫困问题，但不平等这一子因素却引发了新的问题。今天，我们目睹了社会各阶层民众的权利被剥夺。当不平等本身削弱了市场杠杆时，我们必须承认问题已经存在。

法规杠杆则是一个更为复杂的九头蛇，它在行业和社会层面呈现出多种变体。有一种论调认为，正是制约性的法规导致了当前社会的停滞，并侵蚀了生产力浪潮。虽然有证据支持这一说法，但它掩盖了法规的本质。法规允许社会利用市场机制来达成结果，但其方式应当对整体社会有利，并确保市场参与者之间的公平竞争。法规的过程在于确定：对于我们所期望的社会目标，以及市场为实现这些目标所需采取的行动之间，理想的成本效益权衡是什么。这涵盖了从打破垄断到保护环境等一系列广泛的社会关切。毋庸置疑，法规至关重要。但正如我们所看到的，当前的监管结构导致了银行业寡头垄断的形成，并限制了区块链等技术的进步。由于法规在不断演变，但与技术浪潮加速的频率相比存在显著的时间滞后，因此需要努力弥合这一鸿沟。这是一项复杂而艰巨的任务，在本章稍后部分，我们将讨论一种看待法规的替代方式。

由于市场和法规相互交织，构成了政策的经纬，资本主义的最后一项杠杆，即政策，是引导社会发展的基本工具。从宏观视角来看，可以说当今的政策在执行上是低效的，因为它们未能涵盖技术循环往复的行为特征。如文本框 3 和 4 所示，技术进步最快的分支是软件。软件擅长循环行为。通过使用代码执行重复性任务，软件使得之前需要熟练工人的任务得以大规模自动化。过去，技术进步对体力劳动岗位构成威胁。但随着机器学习、神经网络和人工智能持续取得重大而迅速的进展，如今受雇从事认知性任务的技能人员也面临威胁（请参见侧栏 3-1）。正如马克·安德森所说，“软件正在吞噬世界。”如今，软件日益威胁着任何其工作职能涉及重复性框架的人。随着自动化和金融化对工业化经济体构成最大风险，需要出台政策应对生产力的变化。因此，政策制定者必须明白，我们这个自我组织的社会已经出现问题，需要更多引导来对抗和适应未来的技术变革。为此，政策制定者需要思考与无现金社会发展、技术性失业、人类寿命延长及其宏观经济影响相关的举措。
