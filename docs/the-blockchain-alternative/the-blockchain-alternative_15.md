# 2. 金融的碎片化

自危机爆发以来，两个严酷的现实被推到了公众讨论的前沿。首先，银行因其巨大的规模和影响力而被允许对经济产生如此深远的影响。其次，由于信息集中和运作不透明带来的权力集中，它们已偏离了民主的一般定义。

在了解了债务和货币在当代金融体系中是如何被创造出来之后，需要回答的问题是：“银行和金融机构是如何被允许发展到如此规模并以这种方式运作的？”是哪些情势导致了金融渗透到我们生活的方方面面？如果货币仅是一种旨在辅助价值交换的虚拟表现，它为何能对社会各个方面施加如此巨大的影响和控制？对于这个问题的回答，很大一部分带有文化内涵。为了追溯这种文化演变的根源，我们将目光投向全球金融中心之一——英国。
