# 账本

在区块链语境中，账本是一种数字记录系统，用于记录网络上发生的所有交易。该账本维护着所有交易的永久且防篡改的记录。由于区块链通常用于记录交易，因此常被称为账本，有时也称为分布式账本。

账本是去中心化的；没有中央机构控制它。相反，网络上的所有节点都拥有一份账本的副本。只要产生新交易，该副本就会实时更新。所有节点都可以查看这些交易及其相关数据。

账本通常使用密码学算法（如哈希运算）来维护，以确保数据的完整性和安全性。区块链上的每个区块都包含前一个区块的哈希值，形成一条相互链接的区块链。这就创建了一个不可篡改的账本，未经网络共识，任何修改或篡改都无法进行。任何人都可以运行节点，这意味着区块链可以以分布式方式存储，这种账本称为分布式账本。这使得攻击者很难得手，因为他们必须修改网络上的多个副本。由于不存在单点故障，这有助于防止拒绝服务（DoS）攻击。

账本也可以是许可型或无许可型的。在许可型区块链中，只有经过授权的参与者才能访问账本；而在无许可型区块链中，任何人都可以参与网络并查看账本。

该账本为网络上的所有交易提供了透明、安全且防篡改的记录。
