# 3.9.1 挖矿

`挖矿`是通过解决复杂的数学谜题来向区块链添加新交易并保障网络安全的流程。网络参与者（称为`矿工`）会竞相破解一种名为`工作量证明`（`PoW`）的密码学难题。

`矿工`将有效交易收集到区块中，然后对包含前一个区块引用的区块数据应用数学函数（`哈希运算`）。他们会修改一小段称为`随机数`的数据，直到找到一个满足特定条件（例如具有特定数量的前导零）的哈希值。这需要大量的计算能力和能源。

第一个发现满足条件的有效哈希值的矿工，会将这个新挖出的区块广播到网络。随后，其他矿工会验证该区块中交易的有效性。如果就区块有效达成共识，它就会被添加到区块链中，而该矿工将获得新铸造的`比特币`以及区块内包含的交易手续费作为奖励。

`比特币`挖矿主要有以下两个目的：

**交易验证：** 矿工通过验证新块并将有效交易纳入其中，确保`比特币`账本的完整性和准确性。

**区块链安全：** 作为一种安全机制，挖矿所需的计算工作（`PoW`）充当了安全措施。`比特币`共识是指网络参与者就区块链状态和交易合法性达成的一致。以`比特币`匿名发明者`中本聪`命名的`中本聪`共识，是`比特币`所采用的共识机制。`中本聪`共识是通过多数矿工达成一致来实现。矿工共同决定哪条链被视为“最长”，因此也是合法的。这由累计`PoW`最多的链决定，这表示在其创建过程中投入了最大的计算工作。

这使得恶意行为者操纵区块链变得极其困难和消耗资源。
