# 17. 区块链的正式研究与教育

本章基于维克拉姆·迪隆与托里·塞纳吉的一次访谈，内容涉及托里在大多数工作仍以非正式方式进行并主要通过开发者博客传播时，如何着手创建正式的区块链研究教育空间。《当今医疗中的区块链》是当今区块链领域一本知名的同行评审期刊，其编辑委员会拥有非常广泛的从业经验。

**VD**：是什么启发了你创办这本期刊？是什么促使你思考需要用严谨的科学流程来评估区块链技术？

**TC**：我的整个职业生涯都专注于或包含了循证研究成果输出、出版规划、医学出版、研究咨询、医学教育以及研究数据的传播。我曾负责管理一个涵盖多个国际专业期刊和学会同行评审出版物的项目组合。对我来说，出版与传播一直是为全新突破性创新提供创造性表达的渠道。通过一本值得信赖的循证期刊进行交流，既能提升研究工作本身，也能促进成果和/或发现的传播。

我从一位朋友那里了解到加密领域。他当时正在启动一项奖励太阳能使用的资产，我也开始参与挖矿。随着对这项技术了解的加深，我开始思考它在医疗领域的潜在应用。

当时，金融市场就像一个乌烟瘴气的污水池，充斥着更多关心如何欺诈公众、快速捞钱，而不是致力于创造一个能建立监管和新世界市场的安全环境的邪恶参与者。

鉴于我的背景是医疗领域并专注于出版，创办一本新的同行评审期刊，为区块链技术在医疗领域的应用开拓市场，对我来说是顺理成章的事。

作为创始人和出版人，我选择创办一本传统的学术同行评审期刊，预见到了编辑使命将涉足一个不熟悉的领域。整个医疗行业需要逐步建立对这项技术的熟悉度和信心，并理解其广泛的应用前景。《当今医疗中的区块链》这本同行评审期刊提供了最广泛、最快速的采纳途径，用以规模化地掌握这项技术，提升领导层、早期采用者以及最终所有利益相关者的熟练度和灵活性，以填补知识空白、了解更多。

我的想法是，在初期遵循传统学术出版渠道，将有助于更快地促成对期刊本身以及该技术应用的接受。那些理解并从事研究工作的人明白这一点。而那些刚进入这个市场的人则在沟通和撰写合格论文方面遇到了困难。当然，也存在一些傲慢的怀疑论者，但这本期刊已经证明了其对市场的价值：它为区块链技术在医疗领域的应用提供了验证，通过一本同行评审期刊赋予了其可信度。如今，它既塑造也反映了市场的增长与创新。我们的读者遍布七十多个国家，并且刚刚在海湾合作委员会（`GCC`）和埃及推出了期刊的授权版本。我对目前的发展轨迹相当满意。想了解更多关于期刊创刊号的信息，你可以随时阅读我的出版人信函，网址是 [`https://doi.org/10.30953/bhty.v1.25`](https://doi.org/10.30953/bhty.v1.25)。

**VD**：创办这本期刊需要什么样的资源？我知道你过去曾参与过加密社区；跟我们谈谈你过去的经验，以及这些经验如何帮助你创办了这本期刊。

**TC**：从一开始，我就远离加密货币，不会发表任何与之相关的内容，只在会议上偶尔提及，因为当时围绕着它争议颇多。我一直认为，除非它具备真实的货币价值，否则它很可能，而且也经常，被用作我们市场中的一种行为矫正工具。

我偶尔会问一位医生对区块链技术的看法。我得到的回答往往与投资机会有关，所以我知道巨大的认知鸿沟依然存在。这也是为什么我们为医生和药剂师开发了一个新的医学教育项目（`CME`），目前正在寻找赞助商。

**VD**：你成功组建了如此多元化的编辑/审稿人团队。在组建这个团队时，你在寻找什么样的专业背景？你是如何确定找到正确的人选的？

**TC**：我们编委会的水平是一流的，这是因为所有这些人都对改变医疗及传统系统充满了极大的热情。他们是充满好奇心的行家和专家，敢于挑战极限，无所畏惧，并且乐于承担风险。我们编委会的编辑们将自己的名字与一本当时还相对不知名的期刊（`BHTY`）联系在一起，他们都是在拿自己的声誉冒险。对此我心知肚明。《`BHTY`》的目标是引领市场，成为该领域最重要、最卓越的期刊。

我一直相信每个人都能带来不同的观点。随着《`BHTY`》在市场上的成熟，我从世界各地寻找最优秀的人才，为期刊和我们的读者带来新的视角。这是一个志愿者职位，他们投入的时间是期刊质量的体现。这是一项团队合作，我们都为集体的努力感到自豪。

**VD**：你最期待收到哪种类型的研究发表在期刊上？为什么？过去一段时间里，创办期刊的经历如何改变了你对区块链技术发展方向的想法？

**TC**：我们通常会在年度“区块链预测”文章中总结市场上看到的趋势，邀请编委会成员和会议演讲者参与，分享他们对这项技术短期和长期的预测。这是一篇很受欢迎的文章，受到了很多媒体关注。

我很兴奋地看到区块链技术正在被各个行业采纳，而不仅仅是医疗领域。我们看到了美国政府相关举措的增加，并且已经发表了反映这些举措的论文。我希望看到的是广泛的采用，以及大众对这项技术的理解——它是在新技术时代连接不同部分的底层连接机制和平台。但我仍在等待其为美国市场带来成本效益的实现。我们在海外看到了这种情况。此外，普遍存在一种假设，认为区块链成本高昂且由于编码问题难以实施。这种误解也必须要克服。

**VD**：作为一本期刊，在内容方面，你们正在探索哪些新的途径（或垂直领域）？作为一本与区块链这类尖端技术紧密相连的期刊，是否给编辑流程留出了一些实验空间？在不久的将来，是否有任何基于区块链的编辑活动计划？

**TC**：我可以分享的是，我们是第一本在区块链上为文章加盖时间戳，并与`ARTiFACTS`合作提供基于区块链的研究记录账本的健康科学期刊。我们为作者提供了一个定制化的仪表盘，用于分享和推广工作，并确保科学界的研究来源和可重复性。我们告诉作者要使用存储库来存放研究成果，并且还在为一个全球性的研究社区倡议而努力。我们已经成立了大学理事会，在全球设立了大使分会，并且一直在寻找新的、声誉良好的合作伙伴进行合作。欢迎随时联系我们。
