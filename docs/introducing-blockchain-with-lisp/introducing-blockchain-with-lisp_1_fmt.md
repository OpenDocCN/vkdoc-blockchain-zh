# 前言

本书旨在让具备一定计算机基础操作经验、并且至少有`一些`编程经验的初学者能够轻松理解。严格来说，会编程并非必需，但它可能有助于理解某些概念。本书还假设读者具备一些高中数学知识，例如函数。

撰写本书的原因之一，是我找不到一本能够从头开始解释如何实现区块链、并涵盖所有细节的书。我不得不在互联网上进行了大量研究，才得以全面了解区块链是什么、如何实现以及它的用途。

由于信息量庞大容易让人分心，每一章都包含一个总结，以便回顾。在章节之间反复翻阅，直至信息牢记于心，有助于加深理解。

本书的推荐阅读方式是在讲解过程中跟随并编写代码，而非被动阅读。这样才能最大程度地吸收内容。所有源代码均可从 [`https://github.com/bor0/racket-coin`](https://github.com/bor0/racket-coin) 下载。
