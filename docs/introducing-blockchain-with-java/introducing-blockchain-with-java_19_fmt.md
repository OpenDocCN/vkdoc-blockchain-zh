# 第 4 章 构建用户界面

## 4.4 小结

在本章中，我们介绍了如何创建前端用户界面及其对应的控制器。我们学习了如何使用 `Scene Builder` 快速创建场景的模型，并为后端创建引用。我们还学习了如何创建控制器类以及如何恰当地设计它们。

以下是我们目前已涵盖概念的简要回顾：

-   使用 `Scene Builder` 创建 `FXML` 文件
-   在应用的前端和后端之间创建对象和方法引用
-   使用 `java` 和 `javafx` 创建控制器类
-   搭建并使用 `Base64` 外部库编码器
-   在考虑关注点分离的情况下设计控制器类
-   在 `javafx` 中创建对话窗格
