# 15. 资金与合作伙伴

与传统公司类似，加密货币项目也需要资金来支付费用，以推动产品或服务的增长、扩大用户群、保障安全、维持日常运营，以及支付团队成员和区块链世界内各类专家的薪酬。因此，没有资金，大多数项目都无法生存。

资金来源有多种形式，从大型机构和私人投资者，到像 ICO 这样的公开代币销售。本章将资金问题简化为通俗易懂的术语，包括资金来源、项目获得资金的原因以及融资的好处，同时还提供了一个易于遵循的评估框架，用于分析散户投资者面临的风险，并判断出资实体的可信度和声誉。此外，本章还为参与公开发行的投资者提供了一个重要检查清单，为降低风险和避免潜在骗局或损失提供了指导。

本章还讨论了加密货币领域的合作伙伴关系。合作伙伴关系可以是“加密项目对加密项目”或“加密项目对企业”的形式，即区块链项目与传统公司合作。这两种类型的合作伙伴关系都很重要，它们允许公司整合资源、技术专长和技术，从而提升项目的价值输出，并有助于推动采用和创新。例如，加密项目之间的合作通常侧重于互操作性、流动性或跨链功能，而加密项目与企业合作则将区块链解决方案带入现实世界的产业。然而，投资者有责任核实项目团队声称的合作伙伴关系是否强大、有效且可信——本章提供了一套评估程序来帮助实现这一目标。

理解资金与合作伙伴对加密货币项目的影响对投资者至关重要。实体和私人投资者的充足资金有助于确保项目能够达成开发里程碑。同时，资金来源可以极大地增强投资者信心，并提升项目的整体长期潜力。那些得到信誉良好的机构或强大战略合作伙伴支持的项目，往往比依赖未经核实来源的项目更值得信赖。花时间审查出资实体的可信度以及真正增值的合作伙伴关系，能为投资者更清晰地描绘出项目的潜在未来价值，更重要的是，能揭示其中涉及的风险。

请注意，本章是为普通投资者编写的；因此，无需具备金融行业经验也能理解。相反，这种直截了当的方法为投资者概述了易于遵循的评估步骤。

**本章讨论的基本要素：**

-   机构投资
-   私人融资
-   公开融资
-   合作伙伴关系

事实

超过 50% 的区块链项目——就像早期的科技初创公司一样——会在第一年内因资金耗尽而失败。通过了解资金的来源和用途，你已经领先于大多数投资者了。
