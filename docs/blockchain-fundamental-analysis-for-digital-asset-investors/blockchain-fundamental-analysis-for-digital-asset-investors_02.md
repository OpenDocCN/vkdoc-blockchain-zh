# 本书使用的约定

为方便阅读，本书通篇使用了若干术语和短语，具体如下：

1. 为简洁易读，基于区块链的项目，无论是实际的区块链基础设施还是基于代币的去中心化应用（dApps），均称为“项目”或更常见的“加密”项目。

2. “红旗”一词是指警示信号或潜在问题、风险、隐患的迹象，提示需要深入研究或应完全避免投资——具体含义取决于该术语在上下文中的使用方式。

3. 每节开头设有“评估目标”，详细说明作为投资者分析一部分所需评估的基本目标。实现评估目标的具体行动步骤则详述于每节末尾。

4. *缩写“DYOR”意为*“Do Your Own Research”（自行研究）。这表示投资者应在投资前自行调研。责任在于投资者本人，而非其他任何人。

5. 技术发展迅速，示例项目中所引用的一些技术可能已更新或替换。财务数据也可能频繁变动，参考链接可能更改或迁移。然而，本书概述的基本评估技术历久弥新，不应受到这些变化的显著影响。

6. 本书中，你将看到专门用于突出重要信息的提示，这些提示包括：
   1. **专业建议** — 有助于你做出更明智投资决策的实用建议或可操作见解。
   2. **事实** — 每位投资者都应了解的关键信息或重要细节。
   3. **警告** — 评估项目时，应注意潜在风险、红旗或问题的警示。
