# 证券型代币发行（STO）

**示例：** [Polymath](https://polymath.network/)、[Securitize](https://securitize.io/) 和 [Tokensoft](https://www.tokensoft.io/)

证券型代币发行（STO）是指通过数字代币代表的金融工具，向通过 KYC（合格）投资者出售现实世界资产的程序。证券型代币本质上是对基础现实世界资产的数字表示，这些资产可能包括艺术品、股票、债券、黄金、衍生品或房地产。因此，证券型代币发行服务使得许多企业能够实现资产代币化。此外，证券型代币还赋予投资者参与利润分配、分红以及重大管理决策投票的同等权利。购买证券型代币的投资者主要目标是通过现金流或资产增值来获取利润。公司可以按照向公众发行金融工具的标准程序来发行证券型代币。与传统市场类似，所有证券型代币持有者都会获得所有权证书记录。但与传统的资产所有权记录方式一样，证券型代币的所有权记录在区块链上，而非作为正式所有权证书录入文件。

证券型代币发行（STO）是通过特定平台出售证券型代币的过程。这些证券在一个已注册（或以其他方式合规）适用证券市场法规的平台上发行——例如美国 SEC 注册的另类交易系统（ATS）——类似于传统证券交易所，并通过证券型代币发行（STO）的方式进行销售。与 ICO 类似，STO 使创始人能够为新项目筹集资金。但与 ICO 不同的是，这些筹资活动伴随着严格的政府监管。尽管如此，与传统的“证券”融资流程相比，获得 STO 批准是一个相对快速且成本效益较高的过程。STO 独特地将区块链技术与受监管证券市场的要求相结合，以促进资产流动性和融资可及性。智能合约自动化推动并融入了证券监管披露目标、市场诚信、公平、创新和效率。这反过来使区块链领域内外的投资者能够以代币化形式投资于受监管的金融证券——类似于传统金融工具。

尽管 STO 仍处于起步阶段，但它们是一种快速兴起的筹资流程，允许中小型企业以较低的成本影响进行融资。此外，由于 STO 受到高水平的监管和审查，它们被认为比 ICO、IDO 和 IEO 风险更低。STO 平台旨在简化代币生成流程并使其更易于访问。投资者可以通过诸如 [Polymath](https://polymath.network/)、[Securitize](https://securitize.io/) 和 [Tokensoft](https://www.tokensoft.io/) 等网站查找各种 STO 发行清单的信息。

SEC 在正在或近期的执法行动中指控为未注册证券的数字资产包括 Cardano (ADA)、Solana (SOL)、Polygon (MATIC)、Cosmos (ATOM)、NEAR Protocol (NEAR)、Binance Coin (BNB)、Filecoin (FIL) 和 Algorand (ALGO)。
