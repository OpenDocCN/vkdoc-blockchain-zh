# 1. NFT 解决的问题

非同质化代币（NFT）是一种数字资产，可以以艺术、音乐、游戏内物品、视频等形式呈现，如今似乎无处不在。从艺术和音乐到墨西哥卷饼和卫生纸，这些数字资产正像 17 世纪风靡一时的异域荷兰郁金香一样热销——其中一些甚至能卖出数百万美元。

因此，当你创造出新事物时，你理应获得与所创造价值相称的回报。为了实现这一点，我们需要将物理世界的约束复刻到数字世界中。也就是说，我们需要引入稀缺性、明确所有权、验证真实性、追溯来源，并实现版税的收取——这正是本章的基础。

## 引言

技术创新的速率正以前所未有的速度加速发展，这已是不言而喻的事实。一项新技术不仅能解决一系列现有问题，还为下一代创新工具的诞生铺平了道路。最明显的例子就是个人电脑。最初问世时，PC 是一种帮助人们更高效地完成已有事务的小工具：打字时无需再使用需要纸张、墨水和涂改液（还时不时会出故障）的打字机；计算时无需计算器；制作资产负债表时无需铅笔、钢笔、纸张和尺子；学习时无需去图书馆；玩游戏时也无需去游戏厅。

但很快，PC 本身就成为了一个平台，催生了进一步的创新。BASIC、FORTRAN、COBOL 和 Pascal 等高级编程语言，让任何愿意学习的人都能编写新软件，为现有问题提供新的解决方案。此类软件（例如 AutoCAD）使其用户能够以更低的成本更快地进行试验。这反过来又催生了更新的解决方案，这些解决方案本身又成为了下一代的创新引擎。

自 20 世纪 70 年代末第一代 PC 问世以来，我们已经进入了第五个十年。从那时起，我们见证了好几波创新浪潮，每一波都开辟了新的可能性前景，同时也带来了市场采用这些可能性转化成的产品和解决方案而产生的挑战生态系统：互联网、万维网、Wi-Fi 连接、手机、云、智能手机及其运行的移动应用，以及最近的社交网络。

但第五个十年的新现实在于，我们几乎日复一日、在一天中几乎每一个清醒的时刻（甚至当我们睡觉时，如果包括在睡眠中监测和收集我们生命体征的设备），所做的每一件事都涉及使用数字设备——无论是台式机、笔记本电脑、平板、智能手机、智能音箱还是可穿戴设备。我们与同胞互动、购物、规划旅行、组织政治运动、听音乐、工作、点餐、学习、娱乐，以及其他一切可以想象的事情，几乎都完全在数字世界中进行。

确实，从 Radio Shack 的 TRS-80、Commodore PET 和 Atari 800 时代以来，无论从硬件、软件，还是支撑我们世界运转的基础设施来看，我们都已取得了长足的进步。但数字生态系统的一些核心属性与支撑我们经济价值链交付体系的基本支柱之间的错位，至今仍未得到任何有意义的解决。

数字世界的基本特征是近乎零成本和即时复制：我可以随时随地以几乎零成本无限次地复制一份文件。我可以以任何我想的方式分发这些文件。收到这些文件的人也可以这样做：他们可以以任何他们希望的方式复制和分发这些文件。在这种情况下，稀缺性、所有权、真实性、来源追溯和版税收取要么成为无意义的概念，要么几乎无法实现。在财富分配不再依赖劳动之前（而且没有人会屏息以待这样一个体系能很快出现），人类必须采用一种机制，使我们经济所依赖的价值交换能够适应我们日益数字化的现实。



### 稀缺性

稀缺性是指资源相对于需求而言有限可用或短缺的概念。在经济学中，稀缺性是影响个人、企业和政府如何有效分配资源决策的基本概念。

在数字世界中，稀缺性常常是一个问题，因为数字商品可以轻易地无限复制。这意味着一旦创建了原始数字资产，就可以无限复制，导致稀缺性缺失，价值也随之降低。

NFT 通过创建独特、不可互换的数字资产来提供数字稀缺性。这意味着每个 NFT 都是独一无二的，无法复制，这有助于确立其价值。NFT 可用于代表多种数字资产，包括艺术、音乐、电子游戏等。例如，艺术家可以创作一件独特的数字艺术品，并将其作为 NFT 出售，确保该艺术品在世界上仅存一份真实的副本。是的，复制品会存在——正如在物理世界中一样——但与物理世界不同，数字世界可以解决判断某物是原作还是复制品的问题。

一个取得显著价值的 NFT 例子是 `CryptoPunks` 系列。`CryptoPunks` 是在 `Ethereum` 区块链上创建的 10,000 个独特的数字角色系列。每个 `CryptoPunk` 都是独一无二的，其中一些售价高达数百万美元。

另一个例子是数字艺术家 `Beeple`，他于 2021 年 3 月在佳士得拍卖会上以 6900 万美元售出了一件 NFT 艺术品。这次销售展示了独特、不可互换的数字资产的潜在价值，以及 NFT 在数字世界中创造稀缺性的能力。

第三个例子是 `NBA Top Shot`，这是一个基于区块链的平台，允许用户以 NBA 篮球比赛视频精彩片段的形式购买、出售和交易 NFT。每个精彩片段都是一个独特的 NFT，并且通过限制每场比赛可用的瞬间数量来确保其稀缺性。`NBA Top Shot` 大受欢迎，部分精彩片段售价高达数十万美元。

还有一个例子是 `Decentraland`，这是一个建立在 `Ethereum` 区块链上的虚拟现实平台，允许用户将虚拟土地地块作为 NFT 进行购买、出售和交易。每个土地地块都是独一无二的，无法复制，从而在虚拟世界中创造了稀缺性。用户可以利用自己的土地地块构建虚拟体验，例如游戏、艺术画廊等，并通过将其作为 NFT 出售来实现创收。每个土地地块的价值由其位置、大小以及其上构建的内容决定。

### 所有权

所有权是指个人或实体对有形或无形资产主张控制权和拥有权的法律和道德权利。传统上，所有权与房屋、汽车或土地等实物相关联。然而，在数字世界中，所有权通常指控制和主张对数字资产（如图像、视频、音乐、播客、软件及其他类型的数字内容）拥有权的能力。

数字世界在所有权方面带来了许多挑战。与实物资产不同，数字资产可以轻易地被复制、共享和分发。这使得个人和企业难以主张对其数字资产的所有权，并难以控制它们的使用和分发方式。数字复制的便利性也导致了版权侵权和知识产权盗窃问题，使艺术家和创作者难以通过作品盈利并保护自身权益。

NFT 作为数字世界中所有权问题的解决方案应运而生。通过为特定的数字资产创建一个独一无二的代币，NFT 不仅能创造数字稀缺性，还能实现数字资产的独有所有权。

NFT 通过提供一种证明数字资产所有权和真实性的方式，解决了数字世界中的所有权问题。NFT 为艺术家、音乐家和创作者提供了一种将其数字创作作为独一无二的资产进行销售和变现的方式。NFT 也为收藏家提供了一种主张对数字收藏品（如交易卡、限量版收藏品和稀有游戏内物品）拥有权的方式。

以下是 NFT 如何解决数字世界中所有权问题的五个示例：

- **数字艺术**：NFT 允许艺术家证明其数字艺术品的所有权和真实性，提供了一种将其数字创作作为独特、独一无二的资产进行销售和交易的方式。例如，艺术家 `Beeple` 以 NFT 形式售出了一件名为“每一天：前 5000 天”的数字艺术品，售价为 6900 万美元。

- **收藏品**：NFT 允许收藏家主张对数字收藏品的拥有权，例如交易卡、限量版物品和稀有游戏内物品。例如，NBA 创建了 `NBA Top Shot`，这是一个以 NFT 形式买卖数字收藏品的平台。

- **音乐**：NFT 为音乐家和音乐制作人提供了一种主张其音乐所有权、销售独特体验以及控制其音乐分发的方式。例如，音乐人 `Grimes` 出售了一个 NFT 系列，其中包括未发布的音乐和艺术品，售价近 600 万美元。

- **虚拟房地产**：NFT 允许拥有在线游戏和虚拟世界中的虚拟房地产，为玩家提供了一种拥有和交易虚拟资产的方式。例如，虚拟世界 `Decentraland` 允许玩家以 NFT 形式买卖虚拟土地。

- **域名**：NFT 允许域名的拥有和转让，为个人和企业提供了一种将独特域名作为独一无二的资产进行主张和出售的方式。例如，域名 `eth.com` 曾以 200 万美元的价格作为 NFT 售出。



### 真实性

真实性是一个在艺术、文学和哲学等多个领域被广泛讨论的概念。其核心是指一种品质，即真实的、原汁原味的，或忠于自我及原始来源。在数字世界中，由于数字内容的创建、复制和分享变得十分容易，真实性问题便应运而生。这导致了虚假或伪造数字内容的泛滥，使得区分真伪变得困难。近年来，非同质化代币（NFT）的出现彻底改变了数字世界中真实性的感知和管理方式。

`NFT` 是用来代表对独特或稀有资产或内容所有权的数字代币。`NFT` 旨在通过提供一种独一无二且不可篡改的所有权记录来解决数字世界中的真实性问题，这种记录无法被复制或仿造。

以下五个例子说明了 NFT 如何解决数字世界中的真实性问题：

-   **数字艺术**：过去，数字艺术家很难证明其作品的所有权，也难以因创作获得合理报酬。借助 NFT，数字艺术家可以为每件作品创建独特的代币，提供一个安全且不可复制的所有权记录。
-   **音乐**：NFT 也被用于音乐行业，为稀有或独家音乐曲目提供安全的所有权记录。NFT 可以用来代表原始录音的所有权，为艺术家提供了将其作品变现并控制其发行的机会。
-   **游戏**：在游戏行业，NFT 被用来代表游戏内物品和虚拟资产的所有权。玩家可以买卖稀有或专属物品的 NFT，这提供了真实所有权的安全记录，并为数字收藏品创造了新市场。
-   **体育**：NFT 也被用于体育行业，代表稀有或独家纪念品（如比赛穿过的球衣或签名物品）的所有权。NFT 提供了安全且可验证的所有权记录，使粉丝能够在透明且安全的市场上买卖独特的体育纪念品。
-   **房地产**：NFT 也被用于房地产行业，代表独特或专属房产的所有权。NFT 可以提供安全且不可篡改的所有权记录，使买卖双方更容易转移高价值房产的所有权。

### 来源出处

来源出处指的是特定物品或物件（包括其起源、保管链条以及随时间发生的任何变化）的所有权与保管历史。例如，在艺术界，来源出处用于确立一件艺术品的真实性、所有权和历史，这对于确定其价值至关重要。在数字世界中，来源出处指的是数字资产（如文档、图像或视频）的所有权及更改历史。它对于确定数字资产的真实性、完整性和所有权至关重要，随着数字资产变得更有价值和无处不在，这一点也愈发重要。

数字世界中的来源出处问题之所以产生，是因为数字资产易于复制和篡改，并且难以确定数字资产的原创性和所有权。这使得人们很容易创建和分发伪造或篡改的数字资产，这些资产可能被用于欺诈目的，例如身份盗用、知识产权盗窃和金融诈骗。此外，追踪数字资产的所有权和历史也很困难，这可能导致所有权和控制权的纠纷。

NFT 已成为解决数字世界来源出处问题的一种方案。NFT 是代表特定数字资产（如图像、视频或音乐文件）所有权的独特数字代币。每个 NFT 都是独一无二的，无法复制或仿造，这意味着很容易确定数字资产的原创性和所有权。NFT 存储在区块链上，区块链是一种分布式账本，记录着数字资产所有权的所有交易和变更。这使得追踪数字资产的所有权和历史变得容易，从而可以确立其来源出处。

以下是五个例子，说明 NFT 如何解决数字世界中的来源出处问题：

-   **艺术**：NFT 在艺术界变得流行，用于确立数字艺术的所有权和来源出处。NFT 可以用来验证数字艺术品的真实性，而区块链可用于追踪其所有权及随时间发生的任何变更。例如，艺术家 Beeple 在 2021 年 3 月以 6900 万美元出售了一件数字艺术品，用于代表该艺术品的 NFT 确立了该作品的所有权和真实性。
-   **音乐**：NFT 可用于确立音乐文件的所有权和来源出处，音乐文件经常遭受盗版和知识产权盗窃。NFT 可以用来验证音乐文件的所有权，而区块链可用于追踪其使用情况及随时间发生的任何变更。例如，2021 年 3 月，摇滚乐队 Kings of Leon 成为第一支以 NFT 形式发行专辑的乐队，此举确立了该专辑的所有权和真实性。
-   **游戏**：NFT 在游戏行业日益流行，可用于确立游戏内资产（如虚拟房地产、角色和物品）的所有权和来源出处。NFT 可以用来验证游戏内资产的所有权，而区块链可用于追踪其使用情况及随时间发生的任何变更。例如，游戏`Axie Infinity`允许玩家拥有并交易代表游戏内资产的 NFT，从而确立了这些资产的所有权和真实性。
-   **房地产**：NFT 可用于确立房地产资产（如地契和合同）的所有权和来源出处。NFT 可以用来验证房地产资产的所有权，而区块链可用于追踪其所有权及随时间发生的任何变更。
-   **体育纪念品**：NFT 可用于确立体育纪念品（如球星卡、球衣和装备）的所有权和来源出处。NFT 可以用来验证体育纪念品物品的真实性，而区块链可用于追踪其所有权及随时间发生的任何变更。例如，美国国家篮球协会（NBA）与一家名为`NBA Top Shot`的公司合作，销售代表数字收藏品（如比赛集锦和球员卡）的 NFT，这些 NFT 确立了收藏品的所有权和真实性。



### 版税

版税是向知识产权所有者支付的一种费用形式，通常作为使用该权利的交换。例如，在音乐领域，当歌曲以多种方式被使用时，例如在电台播放、在线流媒体播放，或用于电影、电视节目中，需要向词曲作者和出版商支付版税。

在数字世界中，版税的概念同样重要，甚至更为关键。随着数字媒体的兴起，人们比以往任何时候都更容易创作和分发自己的内容，无论是音乐、视频、图像，还是其他形式的数字艺术。然而，这种便捷的分发方式也给创作者带来了新的挑战，他们现在必须应对盗版、作品未经授权使用，以及在追踪和执行知识产权方面遇到的困难。

创作者应对这些挑战的一种方式是使用 NFT。NFT 是一种数字资产，存储在区块链上——这是一种以安全、透明方式记录交易的去中心化账本。当有人购买 NFT 时，他们本质上是在购买一个独特的数字代币，代表对特定内容（例如歌曲、视频或艺术品）的所有权。

NFT 旨在解决长期困扰数字内容世界的诸多问题，其中就包括版税问题。由于 NFT 存储在区块链上，它们使创作者能够以一种透明且易于验证的方式确立对其作品的所有权。这意味着如果有人想使用与某个 NFT 关联的内容，他们必须获得所有者的许可，并支付所需的版税。

为了说明版税的概念以及 NFT 如何帮助解决这一问题，我们来看五个不同的例子：

- **音乐**：一位音乐人创作了一首新歌，并在 Spotify 等流媒体平台上发布。由于这首歌与一个 NFT 相关联，音乐人可以确保从歌曲的流媒体播放收入中获得分成，即使歌曲被加入播放列表或在其他场景中被使用而未经其直接参与。
- **艺术**：一位艺术家创作了一件数字艺术品，并以 NFT 形式出售。由于艺术品的所有权记录在区块链上，艺术家可以确保从该艺术品的任何后续销售中获得分成，即使它被多次转售或交易。
- **电子游戏**：一位游戏开发者开发了一款新游戏，并通过 Steam 等在线平台发布。通过将游戏与 NFT 关联，开发者可以确保从游戏产生的收入中获得分成，即使游戏通过其他渠道销售或分发。
- **摄影**：一位摄影师拍摄了一张精彩的照片，并授权给网站或杂志使用。通过使用 NFT 确立对照片的所有权，摄影师可以确保每次使用该照片时都能获得版税，无论是一次性使用还是多次使用。
- **写作**：一位作者写了一本新书，并在亚马逊等平台上自行出版。通过将书与 NFT 关联，作者可以确保从书籍销售产生的收入中获得分成，即使书籍通过其他渠道销售或分发。

## 总结
