# 3. 创建与出售 NFT

既然我们已经涵盖了区块链技术如何工作的具体细节，特别是它如何为 NFT 提供动力，那么下一个最重要的主题就是这些 NFT 的创建、出售或分发。为了让 NFT 发挥任何作用，它们必须被创建出来并推向世界，以便人们能够使用它们。

虽然在此过程中需要考虑许多因素，但最重要的因素之一是你打算创建的 NFT 的特性。同样重要的是，要考虑这些 NFT 的最终持有者将如何实际持有它们，这意味着要考虑钱包，以及如何将这些 NFT 交到他们手中，这通常是通过出售的方式。

## 特性

在创建 NFT 时，第一步是确定使你的 NFT 独特且有价值的方面。这些可以是任何东西，从一件艺术品或收藏卡到一条推文或虚拟房地产。关键在于确保 NFT 具有使其区别于其他数字资产的品质，并且这些品质在其设计和元数据中得以体现。

NFT 的方面或变量可能会因所代表的数字资产类型而有很大差异。

### 变量

一般来说，NFT 的变量可分为两大类型：内在变量和外在变量。

#### 内在变量

这些变量指的是 NFT 本身的固有品质。例如，就一件艺术品而言，内在变量可能包括艺术家的签名、创作日期、使用的媒介、尺寸，以及任何特殊的技术或材料。对于虚拟房地产，内在变量可能包括房产的位置、大小，以及任何特殊的功能或设施。

从内在变量中获取价值的 NFT 的一个例子是流行的 CryptoPunks NFT 系列；每个 NFT 都有独特的 8 位像素化设计，使其易于识别和收藏。另一个例子是 Bored Ape Yacht Club NFT 系列，其中每个 NFT 都有一幅不同的卡通猿猴插图，并带有独特的配饰和背景元素。

#### 外在变量

另一方面，这些变量指的是有助于提升 NFT 价值或独特性的外部因素。例如，如果艺术品是由知名艺术家创作的，这可能会增加其价值。同样，如果虚拟房地产位于一个受欢迎的虚拟世界中，这也可能会增加其价值和吸引力。

例如，在 NBA Top Shot NFT 系列中，每个 NFT 代表一个特定的篮球瞬间，如扣篮或三分球，并有一个表示其稀有性的限量编号。就 Beeple Everyday 系列而言，其外在价值源于它是第一个在主要拍卖行佳士得售出的 NFT 系列，并且它以创纪录的 6900 万美元成交。

一些 NFT 系列试图通过增加外在价值来随时间提升其价值。Bored Ape Yacht Club 就是其中一个例子——Bored Ape 的创造者 Yuga Labs 不断开发新方式，让 Bored Ape 的所有者可以使用它们，例如在他们的 Otherside 元宇宙中作为头像，以及作为特殊活动的入场通行证。



### 实用性

另一个需要考虑的变量是 NFT 的实用性。虽然有些 NFT 纯粹是收藏品，但许多其他 NFT 在特定生态系统或平台内具有功能性用途。例如，一些游戏 NFT 可用作游戏内物品或货币，而另一些则可用作访问虚拟世界中特定区域或功能的密钥。创作者不妨考虑如何为自己的 NFT 增加功能性用途，以提升其价值和吸引力。

`实用性`可以被视为 NFT 的外部变量。这是因为 NFT 的实用性并非资产本身固有的，而是源于其在特定生态系统或平台内的使用目的或用途。换言之，具有功能性用途的 NFT 的价值取决于外部因素，以及它如何在特定背景下被整合和利用。

例如，一个游戏 NFT 可能在特定游戏或虚拟世界中具有功能性用途，但这种用途依赖于其所处的游戏或世界。脱离了该游戏或世界的背景，NFT 本身并不具备内在的功能性用途。同样，一个提供访问平台内特定功能或内容的 NFT，其用途仅限于该平台，而这种用途并非 NFT 本身所固有。

以功能性 NFT 为例，我们可以再次审视 NBA 的 Top Shot，其中 NFT 以篮球比赛精彩片段的形式代表数字收藏品。这些 NFT 兼具内在价值和外在价值，其功能性用途在于能够参与梦幻体育游戏，并访问独家内容和活动。

另一个例子是将 NFT 用作访问虚拟世界中独家内容或活动的门票。例如，建立在以太坊区块链上的虚拟世界 Decentraland，允许用户买卖作为独家虚拟活动和体验门票的 NFT。

此外，一些 NFT 可用作游戏内物品或货币，为收藏品资产增添了功能性用途。这在许多基于区块链的游戏中很常见，例如 Axie Infinity 和 The Sandbox，在这些游戏中，NFT 可用作游戏内资产，并在去中心化市场上进行交易。

在 NFT 中加入功能性用途可以提升其吸引力和价值，因为它提供了超越单纯拥有收藏品资产的实际益处。有志于创作 NFT 的人应仔细考量其 NFT 的潜在用例，以及如何增加功能性用途以吸引更广泛的买家，并提升 NFT 的长期价值。

### 其他变量功能

NFT 的其他方面或变量可能包括其稀有性、历史记录和出处。

`稀有性`指的是 NFT 的稀缺程度，通常认为稀缺性会提升其价值。历史记录和出处指的是 NFT 的所有权链条和真伪认证，这通常也被认为能提升价值并确保真实性。不过，这并非绝对保证；糟糕的出处记录肯定会*降低*价值。

稀有性或稀缺性是 NFT 创作者试图提升其价值时最常使用的因素之一。它营造了一种专属感。为了实现这一点，创作者可以限制一个系列中 NFT 的数量，或者创作独一无二的 NFT。这通常被称为“孤品”。稀缺性也可以通过限量版发行来实现，每个版本都有固定数量的 NFT 可供购买。

几乎每一个 NFT 系列都试图利用稀有性的概念来提升价值。例子包括 CryptoKitties NFT 系列，其中的每个 NFT 代表一只拥有自身属性（如毛色和眼型）的独特虚拟猫。这些猫中的一些比其他的更稀有，收藏家们愿意为此支付溢价。以 World of Women NFT 系列为例，每个 NFT 代表一幅独特的女性角色插画，其中一些因其独特的特征或背景故事而比其他更稀有。

请记住，与出处一样，稀有性并不会自动提升价值。如果一个 NFT 除了稀有之外没有其他价值，那并不意味着它本身就有价值。许多创作者忘记了这一点，创作出以稀有性为卖点的系列，以为这会让它们更值钱，结果却发现没人关心，因为缺乏其他价值。

这引出了 NFT 的另一个潜在价值方面：历史意义。例如，推特 CEO 杰克·多西发布的第一条推文被铸成了 NFT，并以 290 万美元的价格售出，该推文的历史意义为其增加了价值。另一个例子是有史以来第一个被创建的 NFT，因其历史意义及其在建立 NFT 市场中的作用，以 150 万美元的价格售出。

不过请记住，这种价值很大程度上取决于市场和买家的情绪。杰克·多西的第一条推文 NFT 在 2021 年 3 月可能卖出了 290 万美元，但在 2022 年 4 月再次挂牌出售时，最高出价仅为 277 美元。

也有多个实例表明，NFT 的铸造还借鉴了数字领域之外具有历史意义的作品或事件。何塞·拿破仑·杜阿尔特系列代表了这位萨尔瓦多前总统的作品全集，每个 NFT 代表对其遗产中杜阿尔特作品、演讲等经认证复制品的所有权。

创建 NFT 时的另一个重要考虑因素是它的交互潜力。与传统的实体艺术品不同，NFT 可以设计融入互动元素，例如动画、音效和其他多媒体组件。这可以为 NFT 增加价值和独特性，同时为收藏家和买家提供更具吸引力和沉浸感的体验。Bored Apes 就是包含互动元素的 NFT 的一个好例子——Snoop Dogg 和 Eminem 甚至制作了一个以他们的 Bored Apes 为主角的音乐视频。

不过，创作者在基于 NFT 创作衍生作品或二次创作时必须谨慎：NFT 的创建和拥有并不必然赋予其底层艺术品或资产的版权或知识产权。版权法仍然适用，在创建和销售 NFT 之前，确保已获得所有必要的许可至关重要。在销售和分发 NFT 时，考虑潜在的剽窃或版权侵权问题也很重要。

当 Yuga Labs 创建 Bored Ape Yacht Club 系列时，他们将每只猿猴的完整知识产权赋予了其所有者。这是一种不常见的做法，但对于任何正在创建 NFT 系列的人来说，这值得作为一种潜在的方面进行考虑，以使每个 NFT 更具价值。

不幸的是，许多变量都受限于一个简单的限制：文件大小。



### 文件大小

文件大小问题在 NFT 领域是一个重要的考量，尤其是在区块链上存储和传输它们时。由于 NFT 通常由数字文件（例如图片或视频）组成，这些文件的体积可能相当大，特别是对于高质量或高分辨率的内容而言。这给存储容量和带宽有限的区块链网络带来了挑战。

解决这个问题的一种方法是使用压缩技术，在不牺牲质量的情况下减小文件大小。然而，文件大小和质量之间存在权衡，在某些情况下，压缩可能导致细节丢失或视觉伪影，从而对 NFT 的价值产生负面影响。

最常见的方法是将文件存储在链下，而不是存储在区块链本身。这涉及将文件存储在单独的服务器或云存储平台上，并通过唯一的标识符或哈希值从 NFT 链接到该文件。这种方法有助于减小区块链上 NFT 的体积，同时在需要时仍能访问原始文件。然而，这也引入了额外的复杂性和潜在的故障点——链下存储平台必须保持维护和可访问，以便访问该文件。

这种方法在 NFT 社区中也一直是争议点，有些人指责其是骗局。链下存储要求用于链接文件的唯一标识符或哈希值保持安全且可访问，这本身就是一个挑战。此外，维持存储平台的在线状态也可能出现问题，如果平台宕机或被黑客攻击，可能会危及 NFT 的真实性。

这也造成了一个漏洞，即创建 NFT 和链下存储的人可以修改存储在链下的文件，从而从远处有效销毁该 NFT。这个漏洞在 2021 年由加密短信应用 Signal 的创始人展示，他出售的 NFT 指向一个文件，后来他将该文件替换为便便表情符号，以此作为一个警示故事。

### 存储考量

为了解决这些问题，一些 NFT 创作者转向了去中心化的存储解决方案，例如 `IPFS`（星际文件系统）或 `Arweave`。这些平台提供了一种更安全、更去中心化的文件存储方法，有助于减轻与链下存储相关的潜在风险。这些解决方案并非完美，但随着时间的推移在不断改进。

尽管存在这些担忧，链下存储仍然是许多 NFT 创作者青睐的方法，因为它允许他们创建和销售 NFT，而无需担心链上存储的限制。然而，NFT 买家在购买前应了解相关风险，并仔细评估 NFT 的真实性和价值。

`BSV`（比特币中本聪愿景）区块链是原始比特币区块链的一个分叉，它对 NFT 文件大小问题采取了不同的方法。与其他限制单个交易大小的区块链不同，`BSV` 区块链允许更大的交易，因为它对区块大小没有上限。这是因为 `BSV` 区块链专注于支持大规模数据应用。

通过允许更大的交易，`BSV` 区块链能够在链上存储更多信息，包括更大的 NFT 文件。这种方法消除了对链下存储的需求。此外，它还可以降低创建和转移 NFT 相关的交易成本，因为其他区块链上的大额交易可能更昂贵。

然而，`BSV` 区块链更大的交易大小限制也带来了潜在的安全风险。大额交易在网络中传播以及被节点验证所需的时间可能更长，这可能会使其容易受到攻击。此外，较大的交易也可能被用来膨胀区块链，消耗更多的存储和处理资源。

文件大小对 NFT 创作者和收藏者来说是一个重大挑战，尤其是在对高质量和高分辨率内容的需求持续增长的情况下。在文件大小、质量和可访问性之间找到平衡，对于确保 NFT 作为一种数字资产类别的长期价值和可行性至关重要。

最终，NFT 的各个方面或变量将取决于具体的用例和目标市场。例如，一张收藏卡可能具有与一块虚拟房地产不同的内在和外在变量。仔细考虑 NFT 的各个方面和变量，以创建独特且有价值的数字资产，这一点至关重要。



### 内容与区块链选择

在确定好 NFT 的各个属性后，下一步就是创作构成 NFT 的数字内容。这可能包括创作艺术品或其他数字媒体，也可能涉及将现有内容汇编整理成可代币化为 NFT 的格式。请确保数字内容具有高质量且获得适当授权，或至少注明原创作者。

创建好 NFT 的数字文件后，下一步是准备元数据。这些信息包括 NFT 的名称、艺术品描述以及艺术家姓名。此外，元数据还可以包含与 NFT 相关的任何附加属性，例如稀有度或版本编号。

这些信息与数字文件一同存储在区块链上，从而创建 NFT 特征的永久记录。元数据还用于提供关于 NFT 来源、所有权及其他相关细节的信息。元数据应经过精心策划，并准确反映 NFT 的独特品质与特征。

为 NFT 创建好数字内容和元数据后，下一步是选择铸造 NFT 的区块链。

这通常涉及选择一个支持 NFT 的区块链，并选用兼容的智能合约进行铸造。确保所选用的区块链和智能合约安全可靠且维护良好，因为这有助于维护 NFT 的长期可行性及价值。

以太坊是目前用于创建和交易 NFT 最广泛的区块链。专门用于创建和管理 NFT 的`ERC-721`和`ERC-1155`智能合约，得到众多钱包、市场及其他 NFT 相关工具的支持。这些合约已被用于创建涵盖艺术、音乐、游戏和收藏品等广泛用例的 NFT。

对于艺术、音乐和收藏品，`ERC-721`合约是热门选择，因为它允许创建独一无二、可用于代表单件内容的 NFT。对于涉及创建大量相同 NFT 的游戏及其他用例，则常使用`ERC-1155`合约，因为它允许创建半同质化 NFT（有时称为 SFT），可用于代表同一内容的多个副本。

除以太坊外，其他区块链也在被用于创建和交易 NFT。例如，币安智能链和 Polygon 网络因其较低的交易费用和更快的交易速度，已成为以太坊的热门替代方案。币安智能链支持使用`BEP-721`和`BEP-1155`智能合约创建 NFT，而 Polygon 仍沿用`ERC-1155`和`ERC-721`合约。

其他支持 NFT 的区块链包括 Flow 区块链——专门为在游戏和其他应用中创建 NFT 及其他数字资产而设计；以及 Tezos 区块链，它通过`FA2`代币标准支持 NFT。Polkadot 区块链也通过 Substrate 框架支持 NFT，该框架允许创建自定义智能合约。

针对特定用例，某些区块链和智能合约可能更具优势。例如，WAX 区块链在收藏品领域备受青睐，因其专注于为创建和交易 NFT 打造用户友好的体验。WAX 区块链使用`AtomicAssets`标准，该标准专为在游戏、收藏品及其他应用中创建 NFT 而设计。

在音乐行业，已涌现多个帮助艺术家创建和销售 NFT 的平台。这些平台包括使用 Solana 区块链的 Audius，以及使用以太坊区块链的 Royal。这些平台专门帮助音乐人创建和销售代表其音乐作品的 NFT，并常包含版税追踪和收入分成等功能。

在活动票务领域，Hedera Hashgraph 和 Polygon 网络等区块链正被用于创建基于 NFT 的票务系统。这些系统允许活动组织者创建代表活动门票的 NFT，并可通过区块链安全转移和验证。这有助于减少欺诈，确保只有合法持票人才能参加活动。

最后，您需要选择一个市场来上架和销售您的 NFT。

目前有许多流行的 NFT 市场，例如 OpenSea、Rarible 和 SuperRare，每个都有自己的优缺点。请选择一个最适合您 NFT 特质、且拥有强大用户基础的市场。



## 钱包

加密货币钱包是 NFT 生态系统的关键组成部分，用于存储、管理和转移 NFT。你需要拥有一个支持你计划铸造和交易 NFT 的区块链的加密货币钱包。

市面上存在多种类型的加密货币钱包，包括桌面钱包、移动钱包和硬件钱包。桌面钱包是安装在电脑上的软件程序，适合希望将 NFT 存储在电脑上并方便访问的用户。其优点在于通常免费使用，且配置得当的情况下安全性较高。

但桌面钱包容易受到恶意软件和黑客攻击，可能导致 NFT 丢失。此外，如果电脑崩溃或被盗，你可能无法再访问自己的 NFT。

移动钱包是下载到智能手机或平板电脑上的应用程序，方便用户随时随地访问 NFT。其优点在于通常易于使用且可访问性高。

然而，移动钱包的安全性可能不及桌面钱包或硬件钱包。如果不幸丢失手机或手机被盗，除非有钱包备份，否则可能无法再访问自己的 NFT。

硬件钱包是连接到电脑或智能手机的物理设备，适合希望通过离线存储来保护 NFT 免受黑客攻击和恶意软件侵扰的用户。硬件钱包通常被认为是最安全的数字资产（如 NFT）存储方案，通过物理设备提供了额外的安全层。

硬件钱包的一个缺点是购买成本较高，且可能需要一定的技术知识才能正确设置和使用。此外，硬件钱包可能损坏或被盗，导致无法挽回的损失。

无论选择哪种钱包，务必进行备份并遵循钱包安全的最佳实践。这包括创建强大且唯一的密码、启用双重身份验证，以及写下钱包的助记词或私钥并将其存放在安全位置。

由于钱包通常通过 12 或 24 个单词的助记词进行安全保护，创建者必须将这些助记词写在安全的地方，以防钱包出现问题，同时确保绝不向任何人展示此助记词或以任何方式冒险。不要让任何摄像头拍到，也不要大声读出。

此外，务必只从可信来源下载钱包及其他加密货币相关软件，并警惕可能危及钱包安全的钓鱼诈骗和其他欺诈形式。

在选择钱包时，需考虑安全性、易用性以及与计划使用区块链的兼容性等因素。有些钱包专为特定区块链设计，而有些则支持多个区块链。选择一个能让你自行控制私钥的钱包。

存储和管理 NFT 最流行的选项之一是`MetaMask`钱包。`MetaMask`是一款浏览器扩展，免费使用，支持 Chrome、Firefox、Brave 等主流浏览器，同时也有适用于 Android 和 iOS 设备的移动应用程序。它允许用户连接到包括以太坊和 Polygon 在内的多个区块链上的去中心化应用（dApps），并兼容大量 NFT 市场和平台。

另一个热门选项是`Trust Wallet`，这是一款支持多种区块链的移动钱包。`Trust Wallet`因其易用性和直观的界面而广受好评。

在硬件钱包方面，最流行的选项包括`Ledger Nano`和`Trezor`。虽然它们通常在独立环境中运行，主要作为存储加密货币和 NFT 的离线冷存储解决方案，而非用于日常频繁操作，但在这方面已有技术进步。

例如，`Trezor`与流行的软件钱包`Exodus`实现了集成，使得使用 NFT 更加便捷。同时，尽管`Ledger`长期被视为硬件钱包的黄金标准，但其推出“助记词恢复”服务的争议性举动使其安全性受到质疑。

要创建`MetaMask`钱包，首先前往`MetaMask`网站，点击“获取扩展程序”，根据你的浏览器选择相应选项。安装扩展后，按照提示创建一个新钱包。写下助记词并将其存放在安全的地方。然后，点击`MetaMask`扩展即可将其连接到支持 NFT 交易的各种 dApps。

至于`Trust Wallet`，从 App Store 或 Google Play 商店下载应用并打开。写下助记词并将其存放在安全的地方。至此钱包设置完成，你可以通过内置浏览器将其连接到支持 NFT 交易的各种 dApps。

完成以上步骤后，你就可以开始与 NFT 进行交互了。如果想购买或出售 NFT，需将钱包连接到市场，并通过钱包授权交易。具体过程因 dApp 和交易类型略有不同，但通常只需在钱包应用中按照提示操作，确认交易并支付相关费用。

区块链上的几乎每项活动都需要用户支付交易费。交易费（或称 Gas 费）是大多数区块链的组成部分。区块链网络上的任何交易都需向网络支付费用，以补偿处理交易的节点。费用通常以该链的原生加密货币支付。

费用大小取决于多个因素，如网络拥堵程度、交易复杂性、期望处理时间，以及区块链本身的一些内在特性（如其共识机制）。网络越拥堵，费用越高。矿工和验证者会优先处理费用较高的交易，因为他们从中获得更多奖励。

在交易所购买所需资产，或从其他用户处获取。记得定期监控交易费，避免为交易支付过高费用。



## 市场平台

在用户拥有加密货币钱包并获取了支付交易费用所需的资产之后，参与 NFT 生态系统的下一步便是获取 NFT 本身。许多项目拥有自己的专属 NFT 市场，您可以在其中购买其生态体系中的特定资产，但同时也有几个主流市场销售其支持区块链上的几乎所有 NFT。

`OpenSea`是一个广受欢迎且用户友好的 NFT 市场，支持用户购买、出售和发现 NFT。要在`OpenSea`上创建账户，只需访问网站并点击首页右上角的"注册"按钮。随后系统会提示您使用电子邮件、Google 或 Apple 账户创建账户。账户创建完成后，您就可以开始浏览 NFT、创建藏品集，并与平台上的其他用户互动。

在`OpenSea`上创建账户时，务必牢记需要将加密货币钱包与账户关联，才能购买、出售或创建 NFT。`OpenSea`支持多种钱包，包括`MetaMask`、`WalletConnect`和`Coinbase Wallet`。要将钱包关联到`OpenSea`账户，只需进入账户设置，从下拉菜单中选择"钱包"，然后按照提示输入钱包地址或扫描二维码即可完成连接。

钱包与`OpenSea`账户关联后，您就可以开始购买、出售和创建 NFT 了。

### 买卖基础

购买 NFT 相对简单。

浏览市场并找到您喜欢的 NFT。然后您可以使用加密货币出价或直接购买。`OpenSea`主要运行在以太坊区块链上，这意味着用户通常需要持有以太坊或其他加密货币才能购买 NFT。不过，`OpenSea`已与`MoonPay`合作，在某些情况下支持信用卡支付，通过名为"即时购买"的功能，用户可以使用法币在`OpenSea`上购买特定 NFT。

"即时购买"允许用户使用银行卡购买 NFT，随后由第三方支付处理商将其兑换为以太坊。这使得没有加密货币的用户也能通过传统支付方式购买 NFT。

要在`OpenSea`上使用"即时购买"功能，用户必须先将其信用卡或借记卡关联到`OpenSea`账户。操作方法是导航至`OpenSea`首页的"即时购买"选项卡，并按照提示输入支付信息。

添加支付方式后，用户可以搜索支持"即时购买"的 NFT。这些 NFT 在`OpenSea`市场旁边会显示"即时购买"按钮。点击此按钮将启动购买流程，用户即可使用关联的支付方式购买该 NFT。

并非`OpenSea`上的所有 NFT 都支持"即时购买"功能。此功能仅适用于`OpenSea`团队批准的特定 NFT。此外，"即时购买"交易需支付购买价格 2.5%的费用。出售 NFT 则更为复杂。

您需要先在市场上为其创建列表，包括价格、描述以及任何相关元数据等详细信息。同时，您还必须提前创建好 NFT 本身，并在此过程中仔细考虑您希望集成的所有方面和变量。NFT 上架后，其他用户可以通过前述流程出价或直接购买。

### 选择市场

如前所述，有许多 NFT 市场可供用户选择，每个市场都有其独特的功能和用户体验。例如，`SuperRare`以其精挑细选的高质量 NFT 和强调支持艺术家而闻名。

要在`SuperRare`上创建账户，用户必须先通过验证流程申请批准，该流程包括提交作品样本和个人信息。获批后，用户可以创建个人资料，并开始浏览和竞拍 NFT。

另一个受欢迎的 NFT 市场是`Nifty Gateway`，以其备受瞩目的艺术家合作和限量版发售而闻名。要在`Nifty Gateway`上创建账户，用户只需使用电子邮件地址注册并关联支付方式（如信用卡或银行账户）即可。

最后，`Rarible`是另一个在独立艺术家和创作者中颇受欢迎的 NFT 市场。与`SuperRare`和`Nifty Gateway`不同，`Rarible`允许用户无需第三方验证流程即可创建和出售自己的 NFT。要在`Rarible`上创建账户，用户只需使用电子邮件地址注册并连接其加密货币钱包即可。

这些仅仅是众多支持各类 NFT 的市场中的一小部分。本书附录中提供了更多市场信息。建议探索多个平台，了解它们提供的不同选项和用户体验，以找到最适合您的平台。

还有许多针对特定生态系统或应用场景的小众市场。

其中一个例子是`Axie Infinity Marketplace`，该市场专为购买和出售`Axie` NFT 而设计，这些 NFT 用于`Axie Infinity`发行商`Sky Mavis`开发的游戏中。`Axie`是玩家可以在`Axie Infinity`宇宙中用来战斗并赚取奖励的生物。NFT 用于代表对这些生物的所有权。

要参与`Axie Infinity Marketplace`，用户必须先在`Axie Infinity`平台上创建账户。随后用户可以通过在市场购买、在游戏中赚取，或参与奖学金计划来获取`Axie`。获得`Axie`后，用户可以在`Axie Infinity Marketplace`上将其挂牌出售，或购买其他玩家挂出的`Axie`，不过奖学金获得者不允许出售其`Axie`，因为他们并不拥有这些生物，仅是借用。



#### 交易详情

Axie Infinity 市场采用荷兰式拍卖模式，即 NFT 的价格会逐步降低，直到找到买家。NFT 市场可以采用多种其他拍卖模式，每种模式都各有其优缺点。

英式拍卖是 NFT 市场中最常见的一种拍卖类型。这类拍卖从低价开始，价格逐渐升高，直到找到买家。竞拍者的出价必须高于当前出价，拍卖结束时出价最高的竞拍者赢得该物品。

在维克里拍卖中，竞拍者提交密封出价，出价最高者以第二高的出价赢得该物品。这种拍卖旨在鼓励竞拍者提交其对该物品的真实估价，因为出价低于其认为的物品价值并不会带来任何好处。

密封投标拍卖是指竞拍者将出价放入密封的“信封”中提交，出价最高者以其出价金额赢得该物品。这种拍卖常见于艺术品拍卖，有助于保密出价并防止拍卖狙击。

混合拍卖结合了两种或两种以上的拍卖类型，形成独特的竞价流程。例如，一次拍卖可能以荷兰式拍卖开始，一旦价格达到某个水平，就切换到维克里拍卖。这有助于平衡每种拍卖类型的优缺点，并创建一个更具动态性的竞价过程。

市场支持的拍卖类型是在决定是否要在某个 NFT 市场创建账户时需要考虑的众多因素之一。然而，除了平台的易用性和用户界面之外，还有其他因素需要考虑。

人们还应始终考虑某个市场支持的区块链。这决定了可以在该平台上交易哪些类型的 NFT。一些市场可能只支持在以太坊上铸造的 NFT，而另一些市场则可能支持多种区块链，例如币安智能链、Polygon 或 Flow。

另一个因素是收取的费用。这包括上架 NFT 出售的费用、买卖 NFT 的交易费用，以及支付给 NFT 原创作者的版税。研究和比较不同市场收取的费用，以便找到最符合您需求、性价比最高的市场。

市场是否支持向 NFT 的原创作者支付版税很重要。许多艺术家和创作者依赖版税作为收入来源。一些市场，如 Foundation，优先支持艺术家，并在 NFT 的二次销售中强制收取版税以持续支持创作者。其他市场可能没有这项要求，这一点需要予以考虑。

此外，用户还应考虑每个市场的份额和 NFT 的可用性。一些 NFT 市场拥有更大的用户群和更广泛的 NFT 选择，而另一些市场则更加小众，提供的 NFT 选择也更为有限。在选择买卖 NFT 的平台之前，研究每个市场的受欢迎程度和声誉可能会有所帮助。

在创建账户或买卖 NFT 之前，仔细研究和考虑不同的 NFT 市场非常重要。通过了解每个市场的功能和局限性，用户可以做出明智的决定，从而最大限度地提高在 NFT 领域获得成功和盈利的可能性。

最后，考虑每个 NFT 市场的声誉和社区氛围也是值得的。这包括市场的安全性和透明度历史，以及平台内的活跃度和创新水平等因素。考虑这些因素，用户就可以就加入哪些 NFT 市场、哪些市场最适合自己的需求和兴趣做出明智的决定。

### 出售

一旦您选定了一个市场，对于卖家来说，下一步就是创建藏品。对于买家来说，当决定根据自身动机购买什么时，大部分情况是在浏览作为藏品一部分列出的 NFT。卖家必须创建这些藏品。

创建 NFT 藏品可以让您将一组具有共同主题的 NFT 组合在一起，例如一个艺术品系列或一套集换式卡牌。这可以让买家更容易找到并购买您的 NFT，也可以通过营造专属感和稀缺感来增加其价值。

多数主要的 NFT 市场，如 OpenSea、Rarible 和 Nifty Gateway，都支持藏品。小众市场可能不支持藏品，或者其 NFT 可能被归入开发人员根据 NFT 属性硬编码的特定已有藏品中。

一旦您选择了市场、创建了账户并验证了身份（如适用），您就需要决定 NFT 藏品的主题和设计。这取决于您的 NFT 的属性和变量——如果您在创建自己的 NFT，这些属性是您之前决定好的；如果您是从已有的符合藏品主题的资产中精选，则取决于这些资产。考虑您所选主题的市场需求，以及您资产的质量和独特性，以最大化您 NFT 的价值。

确定主题和资产后，您就可以开始在市场上创建您的 NFT 藏品了。

在一般的 NFT 市场上，系统通常会提示您为藏品输入名称和描述。这有助于买家了解您的藏品内容以及可以期待什么。然后，您需要为藏品中的每个 NFT 上传元数据和数字资产。这通常涉及上传图片或视频文件，以及 NFT 的其他信息，如其名称、描述和属性。之后，请指定您希望在自己的 NFT 被转售时获得的任何版税或佣金比例。

最后，在您的 NFT 藏品上架出售前，请检查并确认其详细信息。这可能涉及查看市场条款和条件，但通常包括为您的 NFT 设定价格和其他细节。

一旦您的 NFT 藏品在市场上线，您就可以开始向潜在买家进行推广，并持续跟踪其表现。这包括在社交媒体或其他在线平台上分享它，参与 NFT 社区和论坛，并联系您所在领域的收藏家或影响者。提高藏品的知名度和兴趣度，会增加销售的可能性，并最大化您 NFT 的价值。

请注意：在大多数情况下，这是一个长达数月的过程，通常需要在创建您的 NFT 之前就提前很久开始。一个有潜力的 NFT 卖家在产生创建和销售藏品的想法之时，就应开始着手建立社区和吸引兴趣。这样，潜在的买家就会在 NFT 被铸造或出售之前很久就在等待它们。

如果您已经拥有一个上传好的 NFT，那么流程通常到此结束。但是，如果您正在创建新的 NFT，它们仍然需要被铸造。大多数市场建议您在 NFT 售出之前不要进行铸造，以节省交易费用，因此这通常是 NFT 在市场交易生命周期中的最后一步。

如果没有铸造，该 NFT 实际上并不存在，它只是其创造者脑海中的一个概念。



#### 铸造与白名单

要铸造 NFT，用户通常需要通过支持 Web3 的钱包与去中心化应用（dApp）中的智能合约进行交互，从而向区块链发送交易。用户必须拥有足够的 Gas 费来支付相关费用。

交易发送后，将由网络处理并打包进下一个区块。此过程通常需要几秒到几分钟，具体取决于区块链和当前的网络拥堵状况。交易一旦得到确认，NFT 就会被铸造出来。根据与智能合约交互所使用的 dApp 不同，该 NFT 可能被发送到创作者的账户，也可能被发送到其他钱包地址。

许多 NFT 创作者通过所谓的“白名单”流程进行销售：他们不直接铸造 NFT，而是将铸造权“白名单”给社区成员，由他们自行铸造。创作者通常会出售这些白名单名额，或将其赠予高活跃度的社区成员，作为对其推动藏品热度增长的奖励。这使得部分社区成员能够以初始价格购买该系列的 NFT。

这通常是一个更复杂的过程，但有助于确保在藏品上架销售之前就已存在买家。不过，也有一些创作者更倾向于直接在市场上架，并利用市场来铸造 NFT。

OpenSea 的 dApp 可以作为一个例子说明其运作方式。要使用 OpenSea 的界面铸造 NFT，需要创建一个新条目，并上传 NFT 的数字资产和元数据。用户还需要在此设置价格和版税比例。

选择“创建”选项，并挑选用于铸造 NFT 的智能合约。OpenSea 支持多种 NFT 智能合约，包括流行的 `ERC-721` 和 `ERC-1155` 标准。然后对交易进行签名并将其发送到区块链，该交易包含创建 NFT 所需的数据。

回想一下，“签名”交易是使用用户加密货币钱包关联的数字签名，来验证拟议活动的真实性。在某些情况下，简单的签名可以免 Gas 费完成，但在创建 NFT 时，用户同时也在发送一笔需要付费的交易。

如果用户选择让 OpenSea 仅在售出时铸造 NFT（而非预先铸造），这意味着 NFT 将在有买家购买时才会被创建出来。在这种情况下，用户无需预先支付铸造 NFT 的交易费，该费用将由买家在购买交易中支付。对于不想预付费用，或不确定其 NFT 是否有足够需求以证明预先铸造成本合理的创作者来说，这是一个有用的选项。

一旦铸造完成，该 NFT 就会作为一枚带有唯一标识符的新代币出现在用户连接的钱包中。随后，用户可以将该 NFT 上架销售、转移到其他钱包，或以任何期望的方式使用。任何涉及该 NFT 的交易都会产生额外费用，并可能需支付版税。

## 总结

选择合适的 NFT 市场是卖家最基础的步骤之一。像 OpenSea、Rarible 和 Nifty Gateway 这样的知名平台通常是首选，尽管一些小众平台也具备独特属性，有时甚至提供硬编码的系列藏品。系列藏品是 NFT 市场不可或缺的组成部分，因此按共享主题组织 NFT 是一个重要的考量。

定义系列藏品需要创作者精心挑选其 NFT 的主题和设计。虽然这可能只是对现有资产进行策展，但系列藏品更常反映创作者打算出售的 NFT 的内在属性。

下一个阶段是上架流程，卖家用标签和描述来推广他们的系列藏品。此过程有助于目标买家识别该系列藏品是否适合自己。后续步骤包括上传元数据、数字资产，以及潜在转售时的版税或佣金比例等细节。在任何藏品上架市场之前，卖家最好仔细检查其资产的各项细节（包括定价），确保一切符合自身目标。

NFT 的成功销售，其营销和推广策略的重要性往往不亚于 NFT 本身的细节和价值。建议卖家在 NFT 铸造之前就早早培养市场期待和需求——通过社交媒体渠道、活跃的论坛以及与知名收藏家的互动来吸引潜在买家，是这一过程的关键部分。

铸造（即在区块链上实际创建 NFT）的环节通常应尽可能推迟，以最大限度地减少不必要的开支。在许多情况下，这是首次创建和销售 NFT 过程中的最后步骤之一。在其他情况下，铸造可能在交易发生前很久就已完成，此时资产只是在易手。白名单流程使社区成员能够直接铸造 NFT，而非由卖家自行铸造，这让他们对所获内容拥有一定程度的控制权，或者完全交由运气决定。无论哪种方式，这都能让实际创作者节省费用。

以 OpenSea 为例，从头到尾的流程是：创作者首先上传数字资产和元数据，确定价格，并决定版税事宜。下一步是选择智能合约，签名并将交易发送到选定的区块链。选择仅在交易售出时铸造，可以让创作者规避初始费用，而由买家承担。随后，NFT 会带着其唯一 ID 被发送到购买者的钱包。此时，许多人可能只想转手 NFT 以获利，但对于具有实际效用的资产而言，这种情况并不常见。那些真正除了通过销售为创作者牟利之外还有其用途的 NFT，会被持有者更长久地保留，并用来实现自己的愿望。
