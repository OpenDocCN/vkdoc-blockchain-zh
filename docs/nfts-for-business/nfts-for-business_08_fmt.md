# NFT 对艺术界的颠覆与创新

长久以来，艺术界一直是创意表达和文化意义的空间，几个世纪以来，实体艺术品被爱好者买卖和收藏。然而，数字时代也带来了变革性的变化，其中最重要的之一便是 NFT 的出现。本章深入探讨了 NFT 在艺术界的应用，对艺术家和收藏家的益处，以及由此引发的争议，并为那些希望成功将 NFT 融入艺术创作的人提供了全面策略。

## NFT 在艺术界的崛起

几个世纪以来，实体艺术的创作者和收藏者会在博物馆甚至自己的客厅等实体空间展示艺术品。这些艺术品的所有权和真实性也易于证明，因为收藏家可以通过独立专家和艺术家本人来认证其真伪。实体艺术与数字艺术的关键区别在于，后者以数字形式存在，表现为虚拟形态。数字艺术可以包括插图、照片、GIF 动图、3D 模型等。然而，数字艺术很容易被复制和展示。例如，若我们在网页上看到喜欢的图片，只需截屏，或是在多数情况下直接将图像以 PDF 或 JPEG 格式保存到自己的电脑上，便可随时随地展示。

`NFT`被用于数字艺术领域，因为它能认证艺术品的真实性，并提供永久的所有权记录。通过将艺术品代币化并记录在区块链上，艺术品的买家可以确信自己拥有的是原作。

此外，`NFT`可以为数字艺术创造稀缺性，使其成为有价值的资产。这正是数字艺术成为`NFT`最早的应用场景之一的原因，也开启了区块链在艺术界复兴的序幕。换句话说，`NFT`作为一种特定资产的数字所有权证书，与流通中的其他任何`NFT`都截然不同。正是凭借这一特性，`NFT`被用于认证数字艺术的所有权。

这项技术还促进了`NFT`的买卖，并将所有权信息存储在区块链上，使得数字资产能够在区块链上进行交换、追踪和管理。

### 早期 NFT 艺术项目及其对艺术界的影响

`NFT`的应用给艺术界带来了显著变化，为创意表达、所有权和交易提供了新的可能性。艺术界最早的一些`NFT`应用是诸如`CryptoPunks`和`CryptoKitties`等项目，为众多艺术家和收藏家探索这一新疆域铺平了道路。

`CryptoPunks`由 Larva Labs 的软件开发者马特·霍尔和约翰·沃特金森创作，于 2017 年 6 月首次亮相，是最早的`NFT`艺术项目之一。该项目包含 10,000 个独特的 24x24 像素艺术角色，每个 Punk 都拥有独特的属性组合，例如不同的发型、面部表情和配饰。最初，任何拥有以太坊钱包的人都可以免费获取这些数字角色。但随着需求增长，这些 Punk 的价值迅速攀升，其中一些的售价高达数十万甚至数百万美元。

`CryptoPunks`的成功昭示了`NFT`作为数字艺术和收藏品媒介的潜力。该项目证明了独特的数字物品可以拥有价值，并能在以太坊等区块链平台上交易。此外，它也凸显了数字艺术世界中可证明的稀缺性和所有权的重要性——由于区块链技术不可篡改的特性，每个 Punk 都无法被复制或伪造，因为每个 Punk 基于其铸造时间的不同而具有不同的价值。

2017 年 11 月，一个名为`CryptoKitties`的新型数字艺术应用向艺术界展示了游戏与艺术如何融合。`CryptoKitties`由 Axiom Zen（现 Dapper Labs）开发，是一款同样基于以太坊网络的虚拟繁殖游戏。用户可以收集、繁殖和交易数字猫咪，每只猫咪都有独特的遗传特征，并可遗传给后代。该项目广受欢迎。在巅峰时期，该游戏甚至占据了以太坊网络交易量的很大一部分，甚至导致网络速度变慢。（这些问题暴露了当时区块链平台潜在的可扩展性问题。）

随着`CryptoPunks`和`CryptoKitties`的成功在艺术界引起反响，艺术家们开始尝试数字艺术形式，探索这种新媒介所提供的可能性。与此同时，社交媒体和在线平台逐渐成为熙熙攘攘的市场，艺术家们可以在那里展示作品，收藏家们则可以寻觅新奇有趣的艺术品。过去，一件艺术品常常需要数年甚至数百年才能获得大众的关注。而借助这些新的社交网络和数字集市，艺术家可以在几天甚至几小时内就使其创作迅速吸引眼球。正是在这些数字集市中，艺术家们开始看到了`NFT`的真正潜力。

其中一位艺术家是迈克·温克尔曼，也被称为“Beeple”。《每一天：前 5000 天》是 Beeple 在 13 年间创作的 5,000 幅图像合集，随后被汇编成一个单一的整体作品。这件艺术品以`NFT`的形式售出。《每一天：前 5000 天》的拍卖是数字艺术史上的里程碑事件，因为它是在主要拍卖行售出的第一件`NFT`艺术品。它以超过 6500 万美元的价格成交，创下了数字艺术品的最高成交价纪录。此次拍卖对艺术界产生了重大影响，赋予了数字艺术——尤其是`NFT`——此前所缺失的合法性。

此后，其他备受瞩目的`NFT`销售接踵而至，包括 Grimes 的《WarNymph》售价超过 600 万美元，以及艺术家 Pak 的作品《Crossroads 2》售价超过 1600 万美元。这些拍卖对艺术界产生了重大影响，并必将塑造未来多年数字艺术的发展方向。

### 知名艺术家和品牌对 NFT 的采用

Beeple 等人的成功所带来的`NFT`在数字艺术领域的合法化，为知名艺术家和品牌通过其艺术品和商品进行变现铺平了道路。

知名品牌和公司已推出限量版和独家产品作为`NFT`。例如，NBA 近期发布了一系列名为`The Association`的`NFT`交易卡，其收藏品的设计会随时间变化。这些`NFT`的外观会根据球员在赛场上的表现而演变。球员在整个赛季中取得的成就越高，其`NFT`的视觉效果变化就越大。除了这个`NFT`项目，NBA 还通过 Dapper Labs 的`NBA Top Shot`极大地推动了体育`NFT`的发展。2022 年，该平台在很大程度上推动了`NFT`体育热潮，带动了近 10 亿美元的销售额。

除了变现，知名艺术家和品牌也采用`NFT`来增加与粉丝、顾客和关注者的互动。例如，音乐艺术家史蒂夫·青木近期发布了一个`NFT`系列，其中包括限量版数字艺术品和商品。其他艺术家和品牌则利用`NFT`开展营销活动，为购买其`NFT`的粉丝提供奖励。

### NFT 对艺术家和收藏家的好处

总体而言，知名艺术家和品牌使用`NFT`正被证明是有益的。`NFT`为艺术家提供了一种变现和认证其艺术品的方式，也为品牌创造了专属产品和基于激励的营销活动。这些知名实体对`NFT`的使用，正在使数字艺术和收藏品得到更广泛的接受。

#### 经济收益

比普尔和史蒂夫·青木的 NFT 藏品案例，已经展示了艺术家和收藏家通过出售原创作品所能获得的潜在经济收益。但除了直接出售这些 NFT 藏品，还有通过提供“二次销售版税”功能的平台来赚取版税的更多机会。利用这些功能，艺术家可以设定二次销售购买价格的一定比例作为自己的版税收入。例如，艺术家可以设定二次销售的 5%版税，这样他们就能从任何二次销售其 NFT 的购买价格中获得 5%的收入。除了设定二次销售的版税比例，艺术家还可以在平台上指定不同的许可类型。这使得艺术家能够控制其艺术作品的用途，除了创造额外的版税收入流之外，还能让他们对自己的创作有更大的控制权。

目前有许多为艺术家提供二次销售版税选项的平台。一些流行的例子包括 `SuperRare`、`MakersPlace`、`Foundation` 和 `OpenSea` 等。这些平台各自提供不同的功能和服务，所以请务必自行研究，找到最适合你的那一个。

除了这些平台，还有一些专门为艺术家打造的区块链项目。这些项目通常提供各种工具来帮助艺术家将作品货币化并加以保护，例如版税支付服务和版权管理。

最后，也有一些传统平台开始提供 NFT 和二次销售版税功能。`eBay` 和 `Patreon` 就是其中的两个例子，相信未来会有更多平台跟进。

#### 来源与认证

来源与认证是艺术界两个至关重要的组成部分。来源指的是艺术品的所有权历史，而认证则是验证艺术品真伪的过程。这两个因素都会显著影响艺术品的价值，因此对于希望出售作品的艺术家来说至关重要。

最近，NFT 已成为艺术家利用来源与认证获益的新工具。NFT 是独特的数字代币，用于代表对数字资产（如一件艺术品）的所有权。当艺术家为其作品创建 NFT 时，他们可以包含有关艺术品来源的信息，例如其前任所有者和展览历史。

通过包含这些信息，艺术家可以向潜在买家展示艺术品的历史和意义，从而提升作品的价值。这有助于确立艺术品的真实性并验证其价值。此外，使用 NFT 可以让艺术家对作品的发行和所有权拥有更大程度的控制，使他们能够保留所有权，并在作品每次售出时都有可能获得版税。

使用 NFT 也有助于保护艺术家免受欺诈和盗窃。由于 NFT 存储在区块链上，它们几乎不可能被伪造或篡改。这使得欺诈者更难将假艺术品冒充为真品，从而有助于保护艺术家和买家双方。

#### 通过民主化为新兴艺术家拓展机会

艺术市场的民主化正在对新兴艺术家的机会产生重大影响，这在很大程度上归功于 NFT。传统艺术市场通常被视为对新兴艺术家封闭且难以进入，画廊和其他渠道往往有很高的准入门槛。然而，随着民主化，越来越多的艺术家找到了无需机构场所就能展示自己作品的机会。通过将艺术品代币化并记录在区块链上，艺术家可以确信自己的作品将得到保护且其真实性得到认证。这使得艺术家能够为其作品创建一个数字市场，赋予他们更多控制权，并为他们接触更广泛的受众创造更多机会。

此外，数字平台的使用也促进了艺术市场的民主化。像 `Instagram`、`Twitter` 和 `YouTube` 这样的平台允许艺术家分享他们的作品并与潜在买家建立联系。这些平台还允许艺术家与观众之间进行更多互动，从而开展有意义的互动并获得反馈。

#### 数字艺术对收藏家的吸引力

得益于数字所有权的出现以及独一无二物品日益增长的吸引力，NFT 也改变了收藏家的市场。有了 NFT，收藏家现在有机会拥有独特、稀有且可验证真伪的数字资产，从而开启了一个全新的投资世界。

对于收藏家来说，数字所有权的一个关键吸引力在于拥有一件独一无二物品所带来的专属感和独特感。对于传统的收藏品，如艺术品或体育纪念品，同一物品可能存在多个副本，导致价值稀释。然而，对于 NFT，每件物品都是独一无二的，拥有自己独特的属性和特征，确保其价值保持高位。

NFT 的出现也为收藏家创造了新的投资机会。对于传统收藏品，物品的价值通常与特定类型物品的更广阔市场挂钩。然而，对于 NFT，其价值完全由供需决定，从而创造了获得显著投资回报的潜力。

此外，NFT 具有高度的可移植性，意味着它们可以在数字市场上轻松买卖，为收藏家提供了更大的流动性和投资灵活性。这导致了一个全新的数字市场和平台生态系统的创建，专门用于 NFT 的交易。

### 围绕 NFT 在艺术界的争议

NFT 的出现彻底改变了艺术界，为艺术家和收藏家都提供了新的机会。然而，NFT 也在艺术界内部引发了激烈的争论。

#### 环境担忧

围绕 NFT 最突出的争议之一是其对环境的影响。在像 `Ethereum` 这样的区块链网络上创建和交易 NFT 需要大量的计算能力，导致高能耗，并因此产生巨大的碳足迹。批评者认为，艺术界对 NFT 的拥抱正在加剧气候变化，破坏了减少碳排放和促进可持续发展的努力。

另一方面，NFT 的支持者指出，许多区块链网络正在向更节能的共识机制过渡，例如 `Ethereum` 从工作量证明向权益证明的转变。此外，一些 NFT 平台正采用环保做法，如碳抵消，以减轻其对环境的影响。

#### 艺术完整性与真实性

另一个争议是关于艺术完整性和真实性的辩论。一些批评者认为，NFT 的数字特性使得艺术家更容易抄袭或歪曲其他创作者的作品。他们认为，实体艺术品的缺失降低了数字艺术的感知价值，并助长了可能忽视知识产权的环境。

相比之下，NFT 支持者声称，区块链技术实际上可以增强数字艺术的真实性和来源。通过去中心化账本，NFT 的所有权和转让历史可以被安全地追踪和验证，使得赝品难以进入市场。他们认为，NFT 为艺术家提供了一个在数字领域保护其作品的机会，而在数字领域，抄袭长期以来一直是一个普遍存在的问题。

#### 艺术与文化的商品化

批评者认为，非同质化代币的兴起导致了艺术的过度商业化，收藏家和投机者更倾向于将数字艺术品视为可交易资产，而非具有意义的创意表达。他们声称，这种转变可能导致人们追逐经济利益，而牺牲艺术价值与文化意义。

然而，非同质化代币的支持者认为，艺术界始终与商业密不可分，而非同质化代币只是艺术家将其作品变现的一种新媒介。他们坚持认为，非同质化代币的经济成功可以使艺术家维持生计，并为未来项目提供资金，从而最终丰富文化景观。

#### 排他性与可及性

最后，排他性与可及性问题一直是 NFT 艺术界争论的关键点。有些人认为，NFT 迎合了富有的收藏家，并为财力较弱的群体设置了准入门槛，从而延续了艺术界的精英主义本质。他们主张，这可能导致代表性不足的艺术家和社区进一步被边缘化。

另一方面，支持者则强调 NFT 在艺术市场民主化方面的作用。他们声称，NFT 平台为新兴艺术家提供了向全球观众展示其作品的机会，绕过了画廊和拍卖行等传统守门人。此外，有人提出通过 NFT 的部分所有权，来让更广泛的收藏家群体更容易接触数字艺术。

### 艺术家与收藏家的策略

随着非同质化代币的出现，数字艺术界经历了变革性的转变，为艺术家和收藏家都带来了新的机遇。随着越来越多人进入 NFT 艺术市场，了解有助于他们在这一快速发展的领域中取得成功的策略至关重要。

以下是艺术家和收藏家在进入 NFT 艺术世界时应考虑的一些关键策略。

1. 艺术家的策略
    1. 建立独特的艺术身份
        1. 形成独特且可识别的风格，使艺术家在 NFT 领域脱颖而出。
        2. 通过数字创作传达一致的艺术愿景或信息。
    2. 建立线上形象
        1. 利用社交媒体平台展示艺术作品，与粉丝互动，并建立社群。
        2. 创建个人网站或作品集来展示和销售数字艺术。这是在 NFT 市场平台上市之外的补充方式。
    3. 拥抱合作与人脉拓展
        1. 与其他艺术家、影响者或品牌合作，创建联合 NFT 项目或相互推广作品。
        2. 参与致力于数字艺术和 NFT 的在线论坛和社群，以拓展专业人脉并向其他艺术家学习。
    4. 选择合适的 NFT 市场
        1. 研究不同的 NFT 市场，找到最符合艺术家风格、目标受众和价值观的平台。
        2. 评估平台费用、用户体验以及对环保实践的支持等因素。
        3. 可供考虑的若干 NFT 市场：
            1. `OpenSea`：`OpenSea`是规模最大、最受欢迎的 NFT 市场之一，提供包括艺术品、收藏品、域名和虚拟商品在内的广泛数字资产。它支持以太坊和 Polygon 区块链，并为创作者和收藏家提供易用的界面。
            2. `Rarible`：`Rarible`是一个去中心化的 NFT 市场，允许用户创建、购买和销售数字艺术与收藏品。它具有用户友好的平台，并提供治理代币（`RARI`），使其社区能够参与决策过程。
            3. `SuperRare`：`SuperRare`是一个策展型市场，专注于高质量的单版数字艺术品。它拥有严格的筛选流程，并以展示 NFT 领域顶尖艺术家的作品而闻名。该平台基于以太坊区块链构建。
            4. `Foundation`：`Foundation`是另一个策展型 NFT 市场，连接数字艺术家与收藏家。它强调创意创新，并支持从新兴人才到知名艺术家的多元化艺术家群体。该平台在以太坊网络上运行。
            5. `Nifty Gateway`：`Nifty Gateway`是一个中心化的 NFT 平台，定期举办来自知名艺术家和品牌的独家限量版数字艺术品的“发售”活动。它提供用户友好的体验并支持信用卡支付，使其对加密社区之外的更广泛受众也具有可及性。
            6. `Async Art`：`Async Art`是一个独特的 NFT 平台，专注于可编程艺术，允许艺术家创作具有可调整图层或元素的数字艺术品。收藏家可以拥有艺术品的单个图层或元素并进行修改，从而产生动态且不断演变的艺术作品。
            7. `KnownOrigin`：`KnownOrigin`是一个策展型 NFT 平台，专注于来自新兴和知名艺术家的高质量数字艺术品。它在以太坊区块链上运行，并提供包括拍卖、版次和艺术家合作在内的一系列功能。
            8. `Hic et Nunc`（`HEN`）：`Hic et Nunc`是一个基于 Tezos 区块链构建的 NFT 平台，以其节能的权益证明共识机制而闻名。它高度关注可持续性，并为艺术家提供铸造和销售数字创作的简单界面。
    5. 这些市场提供不同的功能、费用和体验，因此艺术家和收藏家在进入 NFT 艺术世界之前，应仔细考虑哪个平台最符合他们的需求、偏好和价值观。
    6. 定价与版次
        1. 根据市场需求、制作成本和艺术家声誉等因素，确定数字艺术的适当定价结构。
        2. 考虑创建限量版或独一无二的作品的好处，以提升艺术品的感知价值和排他性。
    7. 营销与推广
        1. 利用社交媒体、电子邮件营销和公共关系来为 NFT 艺术品的发布造势。
        2. 利用数据分析来监控营销工作的有效性，并相应调整策略。

2. 收藏家的策略
    1. 制定收藏策略
        1. 明确收藏家的目标和动机，例如获取经济利益、支持新兴艺术家或保护文化遗产。
        2. 为收藏品确定一个焦点或利基领域，例如特定的艺术风格、主题或艺术家。
    2. 进行充分研究
        1. 在购买数字艺术品之前，调查艺术家的背景和资质。
        2. 分析市场趋势和销售数据，以便就收购哪些 NFT 做出明智的决定。
    3. 融入艺术社群
        1. 参与致力于 NFT 艺术的在线论坛、社交媒体群组和活动，以拓展知识并在社群内建立联系。
        2. 参加虚拟展览、拍卖和艺术家讲座，以了解 NFT 艺术世界的最新发展动态。
    4. 确保 NFT 艺术品的真实性与来源
        1. 在购买前，验证与 NFT 相关的来源、所有权历史和智能合约。
        2. 如有必要，咨询专家或利用第三方认证服务。
    5. 管理和保护 NFT 收藏品
        1. 选择安全的数字钱包来存储 NFT 并保护私钥。
        2. 定期监控收藏品的价值，并考虑保险或资产保护方案。

## 总结

NFT 的出现无疑颠覆了艺术界，在为艺术家和收藏家带来新机遇的同时，也引发了重大的争议和讨论。

随着 NFT 的普及度持续增长，利益相关者持续探讨这项技术的影响及其对艺术界的作用至关重要。NFT 的采用者也需要深思熟虑的策略——艺术家和收藏家需精心挑选合适的 NFT 市场，并制定恰当的目标受众和营销策略。艺术界唯有在拥抱 NFT 带来的益处的同时，正视围绕它的争议，才能在这场数字革命的复杂格局中稳步前行。

对于艺术家和收藏家来说，进入 NFT 艺术世界既令人兴奋，也充满挑战。通过采纳本章概述的策略，艺术家和收藏家可以更好地应对 NFT 市场的复杂性，最大化自身成功的可能性，并为数字艺术的持续发展与演变做出贡献。
