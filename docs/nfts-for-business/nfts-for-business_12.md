# 8. NFT 的未来

NFT 的未来与网络的未来紧密交织。NFT 并非凭空出现，也不是极客们在他们与世隔绝的黑暗屏幕实验室里偶然创造的古怪产物。NFT 的出现是为了解决数字世界中一个长期存在的问题——非稀缺性，并在此过程中揭示出，它们能够通过引入商品约束，让我们在数字世界中重建现实世界的基本构建模块（实体砖瓦经济），从而提供的远不止是对“现实生活”的模拟。我们发现，NFT 实际上可以帮助我们实现最初在 20 世纪 90 年代初随着网络诞生而萌生的那个愿景：一个分布式、民主、开放、人人可访问且安全可靠的网络。而就目前而言，30 年过去了，我们距离那个我们向往的理想之地仍然非常遥远。



## 三个网络时代的故事

那个梦想的追求似乎在 2000 年 3 月的股市崩盘中戛然而止，据称这场崩盘冲刷掉了所有的“愚蠢”和“疯狂”——当我们的想象力飞向令人振奋的新高度和深渊时，这些泡沫与酝酿已久的东西便被宣告终结。“派对结束了”，人们宣布，“非理性繁荣”不再，清算随即开始，将那个长达十年的实验的碎片扫入某个无名垃圾箱，为下一阶段让路：那个“清醒”、“克制”且“有序”的网络。

然而，我们被安慰道，并非一切都丢失了。地下的管道依然存在，连同各种基础设施，以及那疯狂、古怪、狂野的 20 世纪 90 年代留下的经验教训、实验和失败案例——正是这些管道使我们现在习以为常的一切成为可能：从智能手机到视频，再到智能音箱、`AirPods`，以及那些没有就无法入睡的播客。

许多像我们一样积极参与那场奇妙疯狂的人，在 NFT 和其他新兴 Web3 工具面前，正经历着一种发自肺腑的*既视感*。夸大的主张与质疑者、令人兴奋的前景与黑暗的反愿景、醉人的梦想与现实者的嘲讽——这些都不是什么新鲜事。但与 20 世纪 90 年代中期以实体为基础不同，我们当前的基线是一个成熟的、被 Web 2.0 浸透的生活世界，这个世界早已摸索出如何从中赚得盆满钵满。而不仅仅是赚钱，它还关乎我们如何度过日常生活，以及作为有感知的人类如何存在。“脱离网络”如今已是一种如此奇怪的状态，以至于你需要提前几天甚至几周向他人宣布。即便如此，我们也从未真正脱离网络。

因此，挑战在于：从无网时代到 Web 1.0，或从 Web 1.0 到 Web 2.0 的转变是显著的，甚至显而易见且相对容易理解，但从 Web 2.0 到 Web 3.0 显然并非如此。

但请稍作停顿，快速定义这三个网络迭代的一般含义。

**Web 1.0**可以被描述为这样一个网络状态：绝大多数上网冲浪者是被动的读者，而极少数人是内容的创造者。Web1 的使命是让那些有资源的人（通常是大学）能够创建内容，并让那些有途径的人（例如通过`Yahoo`、`AltaVista`）能够搜索到这些内容，并使用一个只需点击链接、或许偶尔填写表单（例如`Mosaic`、`Netscape`）的界面来找到它。

随着**Web 2.0**的到来，我们从少数被动读者查找和阅读文档、以及更少的内容创造者，转变为不仅读者数量大增，创造者数量也大幅增长。无畏者们通过发布博客和网站（例如`Blogger`）是首批 Web2 行为，随之而来的是内容策展、播客和社交媒体。正是在这个阶段，应用程序编程接口及其支持的协议诞生了（如`REST`、`RPC`、`SOAP`、`XML`、`JSON`等）。

因此，从无网到 Web 1.0 再到 Web 2.0 的跳跃是容易理解的——至少事后看来如此。我们从一个极少数人创造和分享信息的时代，走向了创造与分发全面民主化和主流化的时代。然而，从 Web 2.0 到 Web 3.0 所谓的非线性跳跃，显然更难以描述。

但果真如此吗？因为我们都知道，当人们回顾过去时，很容易看到各个阶段和里程碑。而在当时，当我们身处“演进”的过程中时，网络还是一件新鲜事物，绝大多数人并未认真对待它，更不用说预见到它后来所引发的巨大变革了。这甚至包括比尔·盖茨，他在 1995 年出版的《未来之路》一书中完全未提及网络，后来匆忙在第二版中补写了一章关于网络的内容，以修正这个严重的失误。

我们是否预测到我们会使用网络订购几乎所有物品？会使用网络进行银行业务？会使用网络彼此交流？会使用网络进行自我治疗？会使用网络学习？会使用网络找到任何问题的解决方案？这包括通过观看某人发布的 YouTube 视频来学习如何取出断掉的钥匙——谁知道原因呢？本质上，就是生活在线上？

不，我们没有。完全没有。

但事实是，未来早已存在，就像未来总是存在一样，只不过，正如威廉·吉布森那句名言所说，“它只是尚未均匀分布”——而已。

我怀疑今天同样的事情正在发生。

我发现，想要开始理解 Web 3.0 究竟是什么，深入钻研那些声称实现它的技术——区块链、NFT、DAO、DeFi 等——是错误的方法。这样做就像钻研`TCP/IP`、`HTTP`、`FTP`、`SMTP`或`XML`来理解 Web1 或 Web2 是什么一样。相反，也许更富有成效的方式是列举这个即将到来的阶段所蕴含的价值、原则和目标，以及我们希望这个新迭代能为我们解决的问题。

简而言之，在与二十多位 Web 3.0 专家和实践者通过电话、Zoom 对话以及电子邮件交流后，我逐渐理解了 Web 3.0 究竟关乎什么。其中，Web 3.0 涉及去中心化、无摩擦交易、集体与社区基础创建、所有权、获取意义而非仅仅信息、创造意义而非仅仅信息，以及安全与隐私。



### 回到未来

与我们交谈过的 Web 3.0 专家和实践者让我感到惊讶的是，他们都如此反感使用夸大其词的表述。我的惊讶显然反映了整个 Web 3.0 项目所处的普遍怀疑氛围，甚至连我这样对 Web 3.0 抱有同情态度的人，也受到了这种氛围的影响。

“人们被炒作和噪音劝退了，” Identity Review 的联合创始人 Shannon Wu 在一次 Zoom 访谈中告诉我。“但当我分享许多关于早年最早的 Web3 公司刚刚起步时的故事时，”她补充道，“我想人们会看到这与互联网早期围绕个人权利、可扩展性、隐私和用户权利发展时有许多相似之处。”

同样来自 Identity Review 的技术编辑兼 Web3 创新奖学金项目总监 Olivia Baker 则指出，“如果你剥开表面的泥垢，真正给[人们]提供事实……利用这些具体证据来说服他们其实相当容易……让他们相信，以这种特定方式来看待系统确实有其合理性。”她指出，这甚至“在大学环境中也有效，那里很多人……天生就对区块链、Web3 以及所有那些起初听起来非常模糊的概念持怀疑态度……”

当被问及如何看待威廉·吉布森那句关于“未来已来，只是分布不均”的名言在 Web3 背景下的意义时，与我交谈的几位专家都表示赞同，并提供了一些富有见地且有趣的例子和观察。

ARRIVANT 的区块链工程师 Drew Riester 提出了一个令人惊叹的观察：我们正在目睹“货币与国家的分离”的出现，并补充说，如果成功，这种分离可能“像马丁·路德在 1512 年开始的政教分离一样重要”。

Viker 的产品负责人 Manuel Urrego 提到了 NFL 的绿湾包装工队，这是唯一一支由球迷拥有的运动队。Consensys 的 DeFi 经济学家 David Shuttleworth 表示赞同，并指出包装工队的例子“代表了可以通过 NFT 系统实现的碎片化所有权的潜力”。

然而，真正让我豁然开朗的是 ZAG Communications 的创始人、Swirlds Labs 的高级副总裁 Zenobia Godschalk，她将话题引向了我一直认为是网络世界光明山丘上的一个奇迹的产品：维基百科。

这个网站的内容由所有人创建，却不属于任何人；没有中央监督者掌控一切；价值被免费、无差别地提供给每一个人。这样一个奇迹最初是如何被构想出来的，更不用说被构建、维护、培育成一个繁荣的生态系统并持续二十多年（现在我们不仅有维基百科，还有维基媒体基金会），在我这个身处深度资本主义社会和经济中的人看来，几乎是无法解释的——在这个社会里，如果你不能将其变现，它就会消亡。

然而，我必须承认，尽管我自 2001 年初首次使用维基百科起就一直钦佩它，但我从未向这个奇迹捐款过，即使在被要求捐款时也没有。真是惭愧。Web3 能帮助我更忠实于自己的价值观吗？

“每次你访问维基百科，页面顶部都会有一个横幅，上面在向你‘讨钱’，” Godschalk 说道，我则 vigorously 点头表示同意。她补充说：“如果你应用 Web3 元素，就可以说，‘太好了，现在我可以每次阅读文章时支付微支付’，这有助于支持运行维基百科的分布式基础设施……”

同样来自 Swirlds Labs、担任营销与 Web3 生态系统战略总监的 Brady Gentile 则指出了网络的另一个奇迹：开源。“开源一直存在，”他在谈话中指出，“许多开源软件已经融入我们今天使用的操作系统和应用程序中。”

但是，他补充道，“从历史上看，围绕开源模式创建业务一直极其困难。大多数成功的公司不得不采用服务模式，红帽公司可能是少数几家成功的纯开源公司之一。今天，开源无处不在，在 Web3 中也是如此——它可以是 Layer 1 网络、DeFi 协议、交易所、市场……不同之处在于，现在我们能够拥有与这些开源项目相关联的资产，因此贡献者有动力，因为他们热衷于构建并为这些开源项目做贡献，但除此之外，还有一个激励模型：他们通常是该项目相关代币的持有者，随着他们构建并为其创造价值，他们也能获得收益。”

UppticWeb3 的 Web3 增长经理 Ian Brent 与我分享了一个非常好的例子，用每个人都能理解的语言解释了“不可变区块链”这个神秘的概念，以及追踪某个 NFT 可能经历的每一笔交易意味着什么：谷歌文档！ “你可以与所有队友协作，并可以审查所有修订版本，从而获得一份完整的、涵盖文档所有细节的历史记录。”

我接触的 Web 3.0 专家和实践者提到的其他例子包括：

1.  **在线支付**：当然不是去中心化的，但现在我们可以以数字速度进行交易：我可以使用 PayPal 付款或安全地输入我的信用卡信息，并立即激活我的账户。

2.  **生成式 AI 和聊天机器人**：这些代表了语义网的开始。我现在可以问诸如“玉米片为什么被发明？”或“吃野生洋葱安全吗？”这样的问题，并得到简洁的答案，而无需费力浏览大量文档。

3.  **Signal、Telegram**：这些软件让我能够安全地通信。

4.  **付费墙**：为我想要消费的价值付费（《华盛顿邮报》、《纽约时报》、Spotify、Netflix），而不是通过广告出售我的信息。

当然，所有这些例子，以其目前的形态，都未能完全实现 Web3 的全部，甚至大部分理想与辉煌。它们仍然依赖于 Web2 的堆栈，其底层架构（服务器、私有权力的中心、惊人的能耗）显然不足以满足我们对更多、更可扩展、更可持续的需求。

尽管如此，Web3 的承诺本身是激动人心的，它清晰地勾勒出了我们期望奔赴的地平线轮廓。但是，正如 Mimo 的区块链工程师 Raunak Singh 所指出的：“我同意威廉·吉布森的说法，但我认为未来分布不均的原因在于，新开发的技术在解决某些问题方面远比解决其他问题更出色。”

加密货币是最深入渗透主流意识的 Web3 技术，无疑会继续存在。即使是迄今为止对该技术最坚定的批评者也已经开始软化态度，而那些你可能会想象会因一个比过去二十年我们所习惯的更加去中心化的技术生态系统出现而感到不安的科技巨头们，也在拥抱 Web3。

正如谷歌云 Web3 和数字资产部门负责人 Bertrand Portier 在回答我提出的“用你自己的话说，Web3 的承诺是什么？”这一问题时写道：“Web3 的承诺是一个去中心化且透明的世界，它允许个人和组织通过掌控自己的数字资产和属性来繁荣发展。”

但除了让加密货币成为主流之外，另一项必将与我们同在的技术是智能合约。



Consensys 的大卫·舒特沃斯指出：“尽管智能合约本质上很复杂，但即使在最基础的层面，能够实现公开审计、完全透明且无需第三方的直接点对点交互，这一前景本身就是一个极具吸引力的命题。而这正是智能合约所能实现的。”

### 超越炒作

就目前情况而言，区块链及其之上的加密货币、CeFi 和 DeFi 堆栈、dApp 和 DAO 平台，以及它们的智能合约和 NFT，都处于摇摇欲坠的不完美状态，充满了巨大的漏洞，有时甚至近乎荒谬的不理性（当人们把自己的 NFT 托管在 OpenSea 或 Rarible 的旗帜下时，如何能真正谈论摆脱中心化范式？这些平台与支撑当前 Web 格局的任何市场一样，都是中心化且受控的）。但毫无疑问，有某种事物正在发生——我深信这种事物既美妙又危险。

#### 艺术与收藏品

NFT 已经向我们展示了这项技术能对艺术界产生多么深远的影响。NFT 使艺术家能够将其数字创作代币化，并直接出售给那些对其艺术感兴趣并希望收藏的人。只要有艺术家愿意创作并通过艺术获利，只要有买家相信他们所追随的艺术家，这一趋势就会持续。事实胜于雄辩，只要拥抱 NFT 能让艺术家绕过传统中间商、将作品变现并对其知识产权获得更大控制权，NFT 的影响力就会增长，并迟早会成为主流。

#### 音乐与娱乐

音乐行业同样见证了 NFT 日益增长的好处，音乐人现在不仅可以将歌曲和专辑代币化，还可以通过铸造代表其专辑、EP 和单曲特别版的限量版 NFT 来发行限量版作品。通常，此类 NFT 会包含附赠曲目和幕后花絮内容。

#### 游戏与虚拟世界

随着虚拟物品和游戏内资产的代币化，NFT 已经在游戏行业获得了显著的发展势头。这一趋势很可能会加剧，因为 NFT 实现了数字物品在多个游戏平台上的真实、可验证所有权和可转移性。此外，区块链技术与虚拟现实（VR）和增强现实（AR）体验的集成，预计将创造出 NFT 在其中扮演重要角色的沉浸式体验。

#### 分式所有权与流动性

除了直接、可验证的所有权之外，NFT 还可以实现分式所有权，允许多人共同拥有某个高价值数字资产的一部分。这使得获取稀有收藏品、昂贵虚拟房地产等昂贵物品的机会大众化。此外，通过实现无缝交易和即时转移，NFT 增强了流动性，从而释放了投资和经济参与的新可能性。

#### 数字身份与个人品牌

NFT 已经开始在建立和验证数字身份方面发挥关键作用。得益于其代币化，独特的数字资产现在可以作为所有权、专业知识和成就的可验证证明。一个引人注目的例子是社交代币化：将 NFT 与社交代币（代表个人社群或社交资本的代币）结合使用。这种代币允许个人直接奖励其支持者或粉丝。通过将 NFT 与社交代币结合，个人可以为其社群支持者提供独特的福利、访问独家社群的权限，或对某些决策的投票权，从而强化个人品牌并培养忠实的追随者。

### 挑战与潜在解决方案

但前方的道路仍然充满挑战，NFT 的普及速度将取决于这些挑战被克服的速度。以下是未来的主要挑战。

#### 缺乏认知与教育

截至撰写本文时，基本现实仍然是，绝大多数潜在用户和创作者仍然不了解什么是 NFT 以及如何使用它们。就大多数人对 NFT 的认知程度而言，他们已将其等同于虚假的加密货币骗局。NFT 普及的关键在于教育个人了解什么是 NFT，更重要的是，了解它们带来的具体好处、它们解决的实际问题，同时清醒地列举与 NFT 相关的潜在风险。

#### 信任与验证

验证 NFT 的真实性和所有权可能具有挑战性，尤其是在数字艺术和收藏品领域。由于真实性和出处（来源）在数字资产世界中至关重要，因此制定标准化的验证流程、声誉系统和基于区块链的出处记录，将在建立信任和增强 NFT 可信度方面大有裨益。

#### 市场投机与波动性

过去几年 NFT 的迅速崛起吸引了投机投资者，导致价格虚高，明显脱离了数字资产的内在价值。仅仅是一张图片的 NFT 就能卖到数千万美元，再加上人们几乎不理解 NFT 是什么以及它们如何运作，这自然会引起对操纵（如果不是公然违规行为）的怀疑。这反过来又毫不意外地阻止了创作者和买家参与 NFT 市场。因此，可持续增长的关键在于建立市场稳定性、负责任的实践以及确定 NFT 真实价值的机制。

#### 技术复杂性

NFT 运行在区块链技术之上，这对于非技术用户来说理解和操作起来可能颇具挑战。目前，创建、购买和出售 NFT 的过程通常涉及与加密货币钱包、去中心化交易所和区块链网络进行交互等多种操作。克服技术障碍的关键在于通过创建任何人都可以使用的用户友好型平台来简化用户体验。

#### 可扩展性与网络拥堵

区块链网络，尤其是像以太坊这样采用工作量证明共识算法的网络，持续面临可扩展性问题。高昂的交易费用和网络拥堵可能会给 NFT 的采用造成障碍，尤其是对于低价值交易或频繁交易而言。除非 NFT 的铸造、购买和销售能够实现可扩展性，否则 NFT 市场将仍是一个小众兴趣领域。潜在的解决方案包括：（1）探索替代共识机制，如权益证明，与工作量证明网络相比，其能耗显著降低，并且可以更有效地处理交易；（2）采用二层解决方案并开发更具可扩展性的区块链；（3）建立互操作性协议，如 Polkadot 和 Cosmos，以实现不同区块链网络之间的无缝通信和资产转移。这些解决方案有助于促进 NFT 在多个链之间的移动，从而减少任何单一网络上的拥堵。

#### 版权与知识产权挑战

尽管 NFT 在帮助创作者保护作品并从中受益方面前景广阔，但它们也给我们带来了几个复杂的法律和知识产权问题。NFT 的数字特性以及数字资产易于复制的特点，引发了关于版权侵权和所有权纠纷的担忧。制定健全的法律框架、去中心化版权注册机构和透明的许可机制，对于提供必要的保护并确保公平补偿至关重要。



### 环境问题

以太坊等采用工作量证明机制的区块链，在挖矿和交易验证过程中需要消耗大量的算力和电力，导致巨额碳排放。从高能耗的工作量证明共识机制转向权益证明等更环保的替代方案，是显著减少 NFT 交易碳足迹的明确举措之一。同样重要的是推新法规，要求 NFT 平台披露其运营的能源消耗情况，创作者也要提供铸造 NFT 所涉及能耗的详细信息。建立 NFT 平台、创作者、环保组织和研究人员之间的紧密合作也至关重要，共同投资研发以探索创新解决方案，例如绿色挖矿技术、节能共识算法或可持续的区块链基础设施。

## 摘要

NFT 是数字领域长期存在的非稀缺性问题的解决方案，其潜力在于能够在数字世界重塑现实世界的经济概念。本章概述了从 Web 早期（Web 1.0）到民主化、交互式的 Web 2.0 格局的演进过程，进而引出 Web 3.0 的诞生及其核心价值：去中心化、无摩擦交易、集体创作、所有权、意义获取以及安全/隐私。我们探讨了理解从 Web 2.0 到 Web 3.0 跃迁的挑战，并将其与 Web 发展史上的转型进行类比。通过与专家的对话，我们探讨了 Web 3.0 的显著特征，包括去中心化、透明性以及新型经济模式的潜力。本章还涵盖了 NFT 在艺术、音乐、游戏和个人品牌等不同领域的应用，同时也指出了存在的挑战，如认知不足、信任问题、市场投机、技术复杂性、可扩展性、版权问题以及环境影响。尽管挑战重重，作者仍对 NFT 在不断演进的网络大格局中所蕴含的变革潜力持乐观态度。
