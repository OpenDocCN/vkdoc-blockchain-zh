# 第五章：“你的数字行动计划”

是时候为你自己的数字化转型做准备了。这最后一章为我们关于数字化转型及赋能技术的广泛讨论画上句号，并为你提供一个基于八个核心维度、可操作且全面的框架，用于规划和执行你自己的数字化转型。本章穿插了大量非常成功且世界领先组织的例子，我们将特别讨论如何开发数字业务和运营模式、如何基于云计算选择合适的技术栈，以及如何数字化一个组织的核心。你还将学习如何识别有前景的数字试点项目、赋能管理层和员工，以及塑造组织结构。最后且同样重要的是，我们将强调建立开放创新文化、通过接触客户及其他重要受益者来利用你的生态系统的重要性。

除了延伸阅读部分，你还会在每章末尾找到关键点的总结。此外，第二章、第三章和第四章将为你提供易于使用的框架，帮助你判断某项数字技术是否适用于你自己的创意和用例。本书末尾附有与数字化转型和数字技术相关的常用术语综合词汇表，这将为你自己的数字化转型之旅提供进一步的指导。

从本书大量的参考文献中，你完全可以推断出，量子计算、区块链技术和人工智能是极具活力的领域，几乎每天都有新发现和新参与者出现。尽管本书中讨论的众多例子无疑会变得过时，但所提出的概念、基本准则和框架却不会。随着数字技术不断改进和演变，这些宝贵的见解将继续适用。

由于涵盖的数字主题种类繁多，*《数字素养》* 提供了三条不同的快速学习路径。如果你特别想从商业角度学习数字化和数字化转型，你可能只需阅读第一章和第五章的选定章节。如果你侧重于技术并希望更好地理解量子计算、区块链技术或人工智能，你可能只需阅读第一章至第四章的选定章节。另一方面，如果你熟悉数字化转型及其赋能技术的运作原理，我建议你按照下表阅读第二、三、四章的应用章节。
