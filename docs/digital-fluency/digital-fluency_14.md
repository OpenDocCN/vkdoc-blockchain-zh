# 3. 区块链技术

合同和交易是我们现代社会的核心。其纸质和数字记录通过设定相关的监管边界，定义了行政、经济、法律、政治和社会体系中最重要的结构。它们规范着全球范围内国家、公共和私人组织、社区以及个人之间的互动 [1]。正因如此，它们经常成为数字滥用和欺诈的目标，这也使得网络安全在数字时代比以往任何时候都更为重要。在此背景下，区块链技术有望发挥关键作用，因为它对数据完整性和记录的影响力，可能如同互联网对数据访问和交换的影响力一样具有颠覆性。

就其公众接受度而言，区块链技术与 TCP/IP（传输控制协议/互联网协议）有一些相似之处，后者是互联网最重要的使能技术之一。在 TCP/IP 出现之前，电路交换是电信架构的基础，当时双方之间的连接必须通过手动将插头插入电话交换台上的相应插座来建立。因此，多年来，电信服务提供商在通信设备和大型企业及私人住宅之间的专用（私有）线路上投资了数十亿美元。

随着 TCP/IP 协议的引入，通信行为开始发生彻底改变。TCP/IP 协议于 1972 年推出，实现了通过电子邮件的双向消息传递，以及以标准化数据格式交换数字信息。正如 TCP/IP 实现了双向消息传递一样，区块链技术实现了双向交易和价值转移。两种协议都是开放、分布式和共享的网络，没有中央监管机构负责其维护和改进。此外，两项技术最初都是在被更广泛采用之前，先在由研究人员和软件开发人员组成的一个热情但规模不大的社区中流行起来的。虽然 TCP/IP 通过降低（双向）连接和通信的成本释放了新的经济价值，但正如我们将在本章中看到的，区块链技术可以极大地降低交易成本。例如，法国商业咨询公司凯捷估计，消费者通过使用基于区块链的应用，可以在银行和保险费用上节省高达 160 亿美元 [2]。

区块链技术的起源与*比特币*密不可分，比特币是有史以来第一个通过密码学保护的数字货币（或称*加密货币*），也是迄今为止区块链技术最流行的应用 [3]。但在 2017 年围绕比特币的风靡一时（注：传奇美国投资者沃伦·巴菲特曾将比特币称为“赌博工具”[5]）之后不久，人们意识到区块链远不止是数字货币的使能技术。他们意识到它可以应用于包括金融服务、汽车和医疗保健行业在内的各行各业的一系列应用。
