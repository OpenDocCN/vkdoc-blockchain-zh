# 12. 超越

我们在引言中看到，蒂莫西·莫顿在其著作《超物体》中描述了一类在时空尺度上极其庞大的物体，它们挑战了我们关于“物”的常规观念。他列举了数个超物体的例子，例如石油泄漏、气候变化、太阳系以及放射性物质。遗憾的是，他对超物体的描述中充满了矛盾，使得人们难以理解并得出可用于进一步研究的通用结论。

尽管如此，在比特币的语境下，仍有一个值得注意的关键思想。莫顿的理论基于一个名为“面向对象的本体论”的当代哲学运动，该运动的本体论源于海德格尔。海德格尔认为，仅仅强调人类存在无法充分理解世界。面向对象的本体论假定，物体也存在于人类的感知之外，人类充其量只能间接地把握它们。

从莫顿的描述来看，超物体让人联想到一种惰性的、机械的、只受因果律支配的反应式存在。尽管它们存在于难以把握的复杂多维或“超世界”中，但它们终究只是物体。然而，像比特币这样的东西，即使它是“超”的，也并非这种意义上的简单物体。它不仅仅是对世界上的事件做出反应；它自身也会对世界施加作用。物体只能对其环境中的操控做出反应，而只有有机体才能以简单动力学无法描述的方式对环境施加作用。这是一个关键的洞察，它暗示了我们可以像莫顿谈论超物体那样，谈论“超有机体”的可能性——即作为人类无法直接感知、我们只能间接了解的实体。

超有机体将像超物体一样难以被探测到，因为它们作用于与人类尺度不同的尺度上。所谓人类尺度，我指的是人类在时间和空间上容易感知的尺度。超有机体可能不受限于人类所居住的有限地域，也不受限于相同的时间框架。它们可能移动得不够快，以至于我们无法感知，甚至可能根本不以传统意义上的方式移动。它们的行为可能隐藏着某种逻辑，但我们无法确定自己能否理解它。超有机体之于人类，就如同人类之于细菌。人体对细菌来说是可感知的，并且是其环境的一个组成部分。但细菌只能意识到整个有机体的极小一部分，并且永远不会质疑它们的环境是否是某个更高层级系统的一部分。假设细菌发展出了像人类一样交流的能力。那么，它们或许就能通过整合来自身体不同部位的其他细菌的知识，开始理解该有机体整体性的更多方面。凭借智慧和想象力，它们甚至可能对这种被称为“人类”的实体的逻辑发展出良好的理解。

超有机体的特征类似于“盲人摸象”的寓言，这个寓言可以追溯到一些最早的佛教典籍，例如大约公元前 500 年的《谛陀经》。这个寓言描述了村庄里的盲人去检查一头大象。一个人摸到象鼻子，认为它像条蛇；另一个人摸到象耳朵，认为它像把扇子；第三个人摸到象腿，认为它像根柱子。所有盲人都得出了关于大象是什么的截然不同的概念，这正如同不同的细菌会对人类产生不同的印象。我们在此将要面对的挑战与此类似。我们将整合在本书过程中获得的关于比特币的多样化知识。每一章都可能为揭示比特币这个超实体神秘本质提供线索。

放弃人类中心视角是一种强大的工具，可以让我们敞开心扉，去接受那些存在超越人类感知的超有机体的可能性。因此，我们需要构建一个新的理论框架，没有它，我们将无法看到它们。

## 那里有生命吗？

如今，人类已经在寻找除自身之外的生命。美国国家航空航天局（`NASA`）以及天体生物学领域的其他机构正在寻找地外生命。这项工作是通过根据关于生命是什么的假设来观测太空进行的。不幸的是，结果是我们寻找地外生命的标准几乎与已知的生命形式一模一样。当我们为寻找地球以外的生命定义标准时，我们采取了人类（或至少是地球）的视角。因此，我们需要退一步，对生命是什么进行更深入的思考。

## 生命是什么？

我们是活着的，我们周围的植物、动物和真菌也是如此。它们在形态和特征上表现出巨大的差异。想想那些生活在地下的微生物：在多伦多西北 350 英里处的基德矿坑深处，它们在完全黑暗、无需氧气、以黄铁矿为食的环境中，已经不受干扰地生存了长达 20 亿年。“深部碳观测站”是一个研究地球内部生命的研究项目，它估计这些生物的总生物量是所有人类活体总和的 300 倍。

另一个例子是虫草属真菌 `Ophiocordyceps`，它会导致木蚁爬上植被的高处，紧紧抓住并死去。然后，真菌会从昆虫的尸体中长出子实体来进行繁殖。

生命多样性之广的最后一个例子是地衣，它像一层覆盖物一样生长在石头和木头上。地衣实际上并非单一物种，而是由藻类或蓝细菌以及不同种类的真菌组成的共生复合体。一种名为 `Rhizocarpon geographicum` （地图地衣）的地衣物种据估计能存活超过 8000 年，使其成为最古老的活生物体。据估计，该物种覆盖了地球陆地表面的 6%到 8%。

生命比大多数人所想象的更奇特、更多面。尽管如此，我们心中毫无疑问地认为，它们都共享着这种被称为“生命”的特性。即使这是不言而喻的，但要准确地说出生命是什么却极具挑战。然而，一些研究者还是试图确定生命的本质是什么。

奥地利物理学家埃尔温·薛定谔从物理学的角度对这个问题提供了最早的哲学解答之一。在他的著作《生命是什么？》中，他发现生命的特征在于一个实体“（……）持续地在‘做某事’，运动着，与环境交换物质。”通过这样做，它避免了宇宙中所有事物都注定的命运——被熵摧毁。熵可以被设想为封闭系统中无序的程度。热力学第二定律指出，随着时间的推移，封闭系统的熵或无序度会增加。薛定谔指出，任何无生命的实体都会迅速达到热力学平衡或“最大熵”状态。然而，生物体并不会迅速达到这种状态，而是持续保持有序。

据薛定谔所言，对此的解释是，生物体进食、饮水和呼吸；更准确地说，它们有一个新陈代谢过程，通过这个过程与环境交换某些东西。传统上，新陈代谢被视为物质或能量的交换，但根据薛定谔的说法，这说不通；相反，他提出了令人惊讶的解决方案：生物体从环境中汲取负熵。在这种观点下，“（……）生物体使自己维持在相当高的有序度（=低熵水平）上的机制，实际上在于持续地从其环境中吸取有序性（……）就高等动物而言，我们知道它们所汲取的那种有序性（……）是或多或少复杂的有机化合物中物质的极度有序状态，这些化合物作为它们的食物。在利用之后，它们以高度退化的形式将其排出。”因此，根据薛定谔的说法，生命是一个过程，通过新陈代谢，从环境中汲取秩序来维持自身的有序。

智利生物学家温贝托·马图拉纳（`Humberto Maturana`）多年教授生物学，却在一个最简单的问题上——即生命是什么——感到难以作答。他无法在以往的生物学研究中找到令人信服的答案，这促使他花费数年时间思考生命系统的本质。为了回答这个问题，他不得不创造一个新术语：*自创生*（`autopoiesis`）：“一个自创生机器是一个被组织（定义为一个统一体）的机器，它作为一个由组件生产（转化与销毁）过程所构成的网络；这些过程（i）通过其相互作用与转化，持续地再生并实现产生它们自身的过程（关系）网络；并且（ii）通过指定该网络作为这样一个拓扑域的实现，将该机器构成为一个在空间中存在的具体统一体，组件在其中存在。”

就像整本书《自创生与认知：生命的实现》一样，这段引文非常简洁且难以理解。核心观点在于：一个生命系统是自创生的，而一个自创生系统能够持续地再生自身的构成。我们人类能够通过自行寻找所需食物和营养，持续地再生我们的身体。植物和动物也是如此。这与一个异构生系统（如工厂）形成对比，工厂也使用原材料，但生产的是自身之外的东西。因此，自创生可以被描述为持续创造系统持续存在所需条件的能力。根据马图拉纳的说法，这就是生命的本质。

一个生命系统的特征在于，能够通过与环境的代谢，在较长时间内维持自身的构成。所有已知的例子都是生物性的，但我们在此回顾的原理并不要求生物化学化合物。可能存在一个满足生命标准、却不基于已知生化过程的生命系统。一个超个体或许也满足所有相同标准，但如果我们发现了这样一个超个体，我们该如何判断它是否具有生命？

我们可以用类似于定义和描述生物生命的方式来定义和描述它。即使我们并非在寻找我们已知的地球生命的确切生物学机制，我们也应寻找我们在生物生命中所发现的相同特征。

## 生命的特征与比特币

在生物学中，对于生命的特征并没有一致认同的定义，但通常要求具备一个有组织的结构，能够繁殖、代谢、对刺激做出反应、生长并适应。

### 结构

所有生命系统都有一个由边界定义的结构，该边界区分了内部与外部。例如，细胞有细胞膜，而动物有皮肤。对于比特币而言，我们有一种不同类型的结构。比特币代码的每个实例都在一个与其他程序共享的 `CPU` 上运行。从这个意义上说，其结构就是由比特币代码发起的过程。但比特币作为一个超个体，其更重要的结构是由另一个结构定义的：即比特币网络。所有运行在比特币区块链上并连接到其他节点的实例，共同定义了比特币的边界。运行比特币代码但未连接到其他节点的，不属于比特币结构的一部分。如果它们使用不同的区块链，情况也是如此。因此，比特币作为超个体的结构，不同于目前已知的生物物种。不过，它与真菌菌丝有些相似。真菌通过菌丝网络在环境中生长，菌丝是细长的管状丝线。因此，真菌的结构由其相连的菌丝定义。类似地，所有连接到比特币网络的节点共同定义了比特币这个有机体。

### 繁殖

一个有机体要繁殖，就必须具有遗传性。对于地球上的生命系统，这是通过 `DNA` 实现的。通常，它被称为基因，基因包含了产生有机体所需的所有指令。开创性著作《自私的基因》的作者理查德·道金斯（`Richard Dawkins`）认为，基因作为复制因子的角色很有趣。基因利用它所驱动的身体（道金斯称之为生存机器）来实现自身的复制。达尔文自然选择确保只有那些最擅长达成此目标的基因才能长期生存。复制因子与执行复制的载体之间的区别至关重要。在生物学中，复制因子是基因，载体是身体。但道金斯主张一种普遍的达尔文主义，认为这些法则适用于任何地方。因此，我们也应该能够为超个体找到类似的东西。

对于比特币而言，复制因子是源代码，因为它描述了构建比特币的所有组件。从 `DNA` 代码到计算机代码的飞跃并不特别大。主要区别在于，计算机代码是基于二进制的，在半导体中实现，只存在 0 和 1。`DNA` 是基于四进制的，在有机化学中实现，因为有四种可能的值，即四种不同的核苷酸：胞嘧啶（`C`）、鸟嘌呤（`G`）、腺嘌呤（`A`）和胸腺嘧啶（`T`）。我们稍后将讨论用于复制的载体。

### 适应

适应是有机体在生存条件变化时改变其基因以求生存的能力。历史上，突变通过自然选择过程提供了导致适应的变化，但人类也通过选择性育种、转基因作物和基因疗法来实现适应。对于比特币而言，适应是通过开源开发者持续维护其源代码来实现的，其目的是使比特币适应比特币生态系统不断变化的条件。随着每个新代码版本的发布，都会处理环境中遇到的新问题，以使比特币的适应性更强。这反过来又创造了一个新的环境，带来新的压力。与最初的版本相比，比特币当前的代码已经过大量编辑和扩展，这表明适应已经发生。遵循理查德·道金斯有影响力的自私基因理论，适应性在表型层面发挥作用，即基因所产生的有机体，但这并非全部。基因还引出了道金斯所称的“扩展表型”。这个概念既包含了基因直接编码的生物有机体，也包含了该基因所促进的对其他有机体或环境的任何影响。动物界中最令人印象深刻的例子是河狸，它通过筑坝来改变环境，以利于自身的生存。比特币的扩展表型是比特币网络、开源开发者以及所有比特币爱好者。这个扩展表型，而不是运行比特币代码的软件本身，才是适应的单位。用道金斯的术语来说，代码是复制因子，而这个扩展表型就是载体。

### 新陈代谢

生物生命通过与其环境的相互作用来代谢化学化合物和能量。细胞从环境中获取物质，并将其用于化学反应。这个过程需要能量。植物利用太阳的能量制造动物可以代谢的有机物质。所有生命系统都从环境中获取某些东西，并将其代谢为废物。这些废物可能对其他生物至关重要。以这种方式联系在一起的生物系统就是一个生态系统。从非有机的意义上说，`Bitcoin` 也可以被描述为在进行新陈代谢。它消耗电能来驱动运行其计算机代码的 `CPU`，并将其代谢为货币，供人类在金融领域使用。可以说，金钱是 `Bitcoin` 的“废料”，但却是其所属生态系统的关键。这个生态系统就是国际金融世界。因此，`Bitcoin` 与人类形成了一种共生关系，类似于菌根真菌，后者会建造连接植物根系的大型地下网络，并与植物交换养分。

### 生长

生长的能力是生命系统重塑自身形态的本领。生物有机体在其整个生命周期中，没有任何组织能够不降解地存在。例如，心脏每小时跳动约 3,600 次，一生中约跳动 91,980,000 次。没有哪种材料能够承受这样的负荷。因此，生物有机体的各个部分能够再生。`Bitcoin` 找到了一种巧妙的方法来实现这一点，因为它利用了人类的大脑，驱使我们反复在多台计算机上安装其程序。这种效果是通过宗教般的狂热、快速获利的承诺以及与光明和更公正未来的联想来实现的。`Bitcoin` 已将自己嵌入人类的行为模式中。我们购买和安装 `Bitcoin`，因为它伴随着救赎的承诺，或者至少是轻松赚钱的许诺。通过这种方式，`Bitcoin` 类似于一种名为 `Ophiocordyceps`（冬虫夏草属）的真菌，这种真菌不知何故能够操纵昆虫的行为，作为其扩展表型的一部分。对这种分析的一个冷酷戏仿是，我们可能会像 `HBO` 剧集《最后生还者》中的僵尸一样，受到一种新型冬虫夏草菌的影响，麻木地执行着 `Bitcoin` 的业务——即增殖。当然，这是一种夸张的描述，`Bitcoin` 对人类的影响似乎更为温和，但其机制本质上是相同的，并且在动物王国中广为人知。`Bitcoin` 通过对人类动机和行为的精心编排而实现生长。这是其扩展表型的一部分。

### 对刺激的反应

对环境刺激做出反应的能力对于任何生物的生存都至关重要。例如，当动物体内营养耗竭时，它们会通过饥饿信号做出反应。这会触发补充所需营养的行为。当离火太近时，我们会感到疼痛，这促使我们远离火源。这些是旨在将身体维持在一个有利于持续生存的平衡状态的子系统。当我们从这个角度审视 `Bitcoin` 时，并没有立即显而易见的类比。它不会移动。但许多生物有机体也不会移动。然而，`Bitcoin` 确实在一种方式上展现出对刺激做出反应的能力，即通过设置加密难题的难度。每当创建一个新区块时，它都会查看自上一个区块产生以来的时间。如果这个时间超过十分钟，就意味着 `Bitcoin` 网络的总处理能力较低，它便会将难度重新配置到较低的水平。相反，如果时间较短，则意味着相对于十分钟的平衡点，算力过剩，难度便会增加。`Bitcoin` 网络这种细微的“心跳”确保了它能适应其环境中可用的处理能力，即它所寄居的计算机。

## 比特币是活的吗？

根据生命的定义标准，我们可以得出结论：`Bitcoin` 似乎满足了这些标准。尽管 `Bitcoin` 不像所有其他已知生命形式那样依赖于有机化学，但它似乎具有相同的属性。一个反对的论点可能是，`Bitcoin` 并非真正的生命，因为它依赖于计算机和互联网。但这与许多需要非常特定环境的细菌并无区别，离开了这些环境它们就会死亡。另一个可能的批评是它是人类创造的。但这并非一个有效的论点，因为没有任何标准要求生命不能是人工创造的。它在重组其他来源特征的方式上也类似于生命。我们看到了不同的技术发展脉络如何导致了 `Bitcoin` 的实现。这与细菌从其他细菌那里重组和吸收遗传物质的方式并无不同。这个过程在生物学中广为人知。

因此，`Bitcoin` 实现了我们之前定义的生命基本功能：“因此，一个生命系统可以说具备以下特征：通过与环境的新陈代谢，在较长时间内维持自身构成的能力。” `Bitcoin` 也是自创生的，因为它持续地利用人类来安装和升级运行它的软件。今天运行 `Bitcoin` 网络的 `Bitcoin` 矿工与十年前的不同。它会动态地替换执行其代码的节点。它也有新陈代谢的过程：消耗能量来驱动计算机 `CPU`，并将其代谢为数字货币，而它的共生体——人类，可以通过花费这些数字货币来购买更多能量。

也许有人会反驳说，这是一个虚构的“恰如其分”的故事，可以套用在任何在计算机上运行的事物上。因此，让我们通过考虑几个其他基于计算机系统的例子来看看是否真是如此。首先，我们来看看计算机蠕虫。就像真正的病毒一样，它只进行复制，并非生命。它没有新陈代谢或适应能力。它只是一段相同的代码。

另一个例子是像 `Microsoft Word` 这样的流行计算机程序。人类也帮助它进行复制，并且它通过持续发布新版本而适应了环境，但它不会对刺激做出反应，也没有新陈代谢。一个与 `Bitcoin` 更接近的类比可能是像 `BitTorrent` 这样的点对点共享系统，因为它具有与 `Bitcoin` 相似的形态。它安装在计算机上，并连接到其他节点。`BitTorrent` 确实以数字资产的形式带来价值，但它并不生产这些资产，因此不能说它具有与 `Bitcoin` 相同意义上的新陈代谢；它仅仅是为其他事物服务的传输系统，因此是异创生的。由此可见，即使是密切相关的系统，在像 `Bitcoin` 那样满足生命标准方面，也多少有所欠缺。

## 将大象重新拼合

因此，我们得出了一个令人惊讶的结论：作为一个超有机体，比特币*实际上*可能是活的。当然，要验证这一点，还需要比本文提供的更多研究和思考。这仅仅是一个粗略的轮廓。尽管如此，这一发现为后续研究提供了富有成果的途径：我们周围是否还隐藏着其他类似的超有机体？这会带来什么后果？比特币究竟类似于哪种生物有机体？它是如何从无生命之物变为有生命的？最重要的是，本文提出的推理能否经得起进一步的审视？我希望这类问题在未来能够得到解答。

在本书中，我们探讨了比特币的许多不同方面。当我们把比特币视为一个超有机体时，这些不同的视角便汇聚在一起，就像盲人摸象时摸到的各个部位。每位盲人的观察，都类似于本书中的一个或多个章节。从它自身的视角出发，每一个观察都对比特币的某一方面提供了引人入胜的洞见。其中一个方面是技术史，它解释了比特币的传承，用生物学的话来说，就是提供了比特币这个“物种”是如何产生的系统发育学描述。中本聪的历史则提供了比特币如何在世界上诞生的精确历史背景，或者说生物学上的个体发育学描述。对货币本质的研究，揭示了比特币如何产生其关键效用：货币价值。社会组织描述了帮助其作为“扩展表型”一部分而成长的社会结构。宗教层面解释了比特币通过何种机制激励我们人类以共生关系为其繁衍提供持续支持。未来展望则提供了关于比特币作为超有机体未来命运的生态学场景。

如果那些摸象的盲人没有坚持各自对“大象”这一事物拥有唯一正确的理解，而是尝试沟通并汇集这些孤立的事实，他们本可以更接近理解并构想出这头大象。这正是本书的秘密计划：汇集关于比特币的种种不同视角，这些视角各自孤立来看是正确的，但不足以理解比特币。通过整合这些视角，我们能够为更全面、更深入地理解比特币这种新型实体——一种超有机体——奠定基础。
