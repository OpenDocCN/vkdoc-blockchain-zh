# 加密货币如何成为货币

密码学曾是一个两千年来鲜有变化的边缘学科，主要局限于军事和政府外交领域。如果有人在 20 世纪 60 年代初提出，密码学将成为全球数字通信网络和一种新型货币的核心，得到的反应很可能是轻蔑一笑和翻个白眼。

但这一切在 20 世纪 70 年代发生了改变。现代计算机的出现以及数学理论更聚焦的应用，引发了一场重大变革，彻底改变了密码学和数字通信。公钥密码学和哈希算法的发现，为如今支撑互联网信息基础设施的诸多创新铺平了道路。它也奠定了坚实基础，让数字货币的梦想成为现实——以至于这类货币后来被称为**加密货币**。

然而，这还远远不够。正如我们所知，这类加密货币在世纪之交就已有人提出，但一些问题依然存在，最显著的是双重支付问题，即如何证明某个代币或币种尚未被花掉。另一个问题是中心化，因为所有提出的模型都或多或少依赖于一个需要信任的中心化第三方。这是一个重大缺陷，密码朋克们对此心知肚明，也不打算轻描淡写，因为这类似于令人畏惧的银行作为你资金中央看门人的角色。要兑现真正自由虚拟货币的全部承诺，还需要解决几个关键难题。我们现在就来关注这些难题。
