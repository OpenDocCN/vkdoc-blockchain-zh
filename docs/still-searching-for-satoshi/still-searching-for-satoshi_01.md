# 前言

我第一次认真关注比特币和区块链是在 2015 年左右。当时我正在金融领域设计反洗钱系统，立刻意识到比特币在这方面会非常方便，因为所有交易都已经公开可见。诚然，这很可能不是大多数比特币爱好者的第一反应。经过深入研究后，我看到了巨大潜力，并开始小试牛刀，购买了一小部分比特币进行尝试。我还购买了相当数量、据信是首个允许散户投资者接触比特币的交易所交易基金（ETF）。不久之后，出现了一些关于比特币的负面消息。大概是开源开发团队的一位核心开发者离开了。我不记得具体细节了。人们都说这个东西快要完蛋了。我清楚地知道，这项投资是我投资组合中真正有相当大概率归零的资产之一，加上比特币自去年以来已经上涨了超过 100%（因此很可能上行空间有限，对吧？），我做了许多比特币持有者（无论此前还是此后）都做过的事；我相信其专业术语是“吓坏了”，然后卖掉了所有比特币。我当时并不知道，关于比特币即将崩溃的恐吓故事，就像我家乡丹麦的下雨天一样常见。

几年后，我身在美国，恰逢比特币又一次繁荣期。我突然想起，当初为了测试而购买的比特币，在价格约 800 美元时我忘记卖掉了。如今各大新闻，甚至纽约地铁的信息屏幕上，都在报道比特币即将突破 20,000 美元大关。我当时觉得这简直疯了，比特币可能再也涨不了那么高了，于是我找到旧钱包，将其套现。我用这笔钱给妻子买了一枚钻戒，这算是一次小型技术实验带来的不错回报。我希望这样的机会还能再多一些。

在那之后，我对比特币的看法变得负面，认为它已经退化为一个庞氏骗局，从未真正认真尝试去实现其应有的潜力。有时我会在 Quora 或推特上对比特币发表批评意见，结果却被大量喷子淹没，他们嘲笑我，说我有多蠢。自然而然，我失去了兴趣。

如今回想起来，我注意到，如果当初我没有惊慌失措，而是持有最初对比特币 ETF 的投资，我实际上本可以买下一辆比特币社区梦寐以求的兰博基尼。我的故事对于许多普通人——或者比特币社区所称的“圈外人”——来说并不罕见。有些人幸运地中了头奖并发了财。大多数人则没有。

在我写下这些文字时，加密市场已经回落。我既未直接也未间接持有比特币。我经历了对比特币的热情、鄙夷和冷漠的完整光谱。然而，我仍然觉得这是一项如此迷人且独特的技术发展，需要有人中立且客观地讲述它的故事。关于比特币或由专家们向你全面解读的书籍并不少见。问题在于，他们无一例外地在这上面投入了大量精力，因此也极力正面描绘它。这并不意味着没有批评者，但他们完全置身事外。他们通常很早就断定比特币是个骗局，并认为不需要做超过表面的研究。这为那些好奇、开放的非技术读者留下了一个巨大的空白，他们只想了解这一切喧嚣背后的原因，并能够对比特币和区块链这件事形成自己的判断。

当前，比特币面临监管压力和丑闻等重重阻力，但即使你认为比特币已经见顶，它仍然是现代世界前所未有的杰出技术成就，值得认真关注。

如果你尚未对比特币着迷，但对它是什么以及它可能成为什么感到好奇，那么这本书就是为你准备的。我希望能够引导你了解比特币和区块链背后那些引人入胜的思想和历史，阐明比特币所针对的人类文明中的种种问题，并提供一个关于区块链未来可能发展方向的视角。即使你已经是比特币的“老手”，我希望你也能从中找到新的信息和视角。我本意是提供一个平衡的叙述，既不草率否定比特币，也不追捧关于其神奇之处的最夸张说辞。
