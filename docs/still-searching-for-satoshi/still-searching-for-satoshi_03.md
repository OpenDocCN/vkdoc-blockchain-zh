# 本书大纲

本书第一部分是一部技术史，梳理了比特币所依托的关键发展历程。这些发展或多或少是独立进行的，最终融合成了比特币。在解释清楚这些之后，我们才能更深入地阐述比特币的工作原理。对这些原理的理解将贯穿本书的其余部分。

我们将考察比特币出现之前的电子货币或虚拟货币的历史。摆脱货币物理形态的梦想由来已久，在比特币之前已出现过几次失败的尝试。通过消除货币的物理属性，伪造变得不可能，或者至少转变为一种电子层面的行为。但这同时也带来了新的问题和监管挑战。

二十世纪密码学的关键发展为比特币铺平了道路。密码学的目的是通过维护信息的机密性和完整性来保护其私密性。密码学历史悠久，但直到最近才发展成一个专门的学术领域。电子计算机的出现只是加剧了已有的趋势，这促使了旨在缓解互联世界问题的新技术的开发，并从根本上为千年难题带来了全新解决方案。

基于我们对历史前身所获得的认识，我们将逐步拼凑出比特币和区块链的工作原理。我们将看到钱包、区块链、挖矿和比特币代币等核心概念如何在比特币系统中运作，从而提供一种全球性的虚拟货币。

在完成一般性的历史考察之后，我们将把注意力转向围绕比特币出现的特定历史方面。我们将尝试对它的发展进行历史记述，并审视现有资料能告诉我们关于其神秘发明者——中本聪——的哪些信息。

我们将尝试提供一些关于历史上中本聪究竟是谁的见解。此前许多关于比特币的文章都集中于探讨中本聪背后的历史人物或团体究竟是谁，但我们将以历史批判的视角来审视这个问题，并筛选现有证据，以便更清晰地理解我们能够和不能够就中本聪的身份说些什么。

自 2011 年以来，既有资深记者，也有业余博主，对此进行了大量调查和猜测。但资料本身究竟说了什么？通过聚焦于这些资料，我们能够提取出几个需要关注的关键特征。这将使我们能够构建出中本聪的侧写画像。以一手资料为基础，我们将列出一份可能是中本聪的历史人物名单，并将他们与侧写画像进行匹配。事实上，我们会发现，只有一个明确匹配的人选。

本书的最后一部分将尝试将比特币置于更广阔的背景下，考察如何在人类文明的宏大图景中看待比特币，因为它并非存在于技术真空中、仅解决技术问题，而是牢固地植根于一系列对人类文明至关重要、且历来如此的主题网络之中。

自史前时代起，人类就从事交换活动。这种活动在从物物交换，到经由中介（如货贝、金银币），再到纯粹象征性交换手段的连续光谱上，呈现出许多不同的形态。我们将考察货币的历史与本质，并了解比特币如何与之契合。

社会组织对于理解文化现象至关重要。因此，我们将更深入地研究与比特币相关的不同类型的社会组织。我们将审视密码朋克、自由意志主义者和开源开发社区，并探究比特币的本质和愿景如何与其产生、使用的社会组织紧密相连。

一个鲜少被讨论的话题是比特币及加密运动的宗教层面。但就人类文化而言，这些层面并非新鲜事物。理解这一点，有助于解释比特币信徒许多看似奇怪的行为，而无需声称比特币是一种真正的宗教。我们可以清晰地看到其中的相似之处：先知中本聪；神圣经文——比特币白皮书；真信徒与异教徒的划分；以及无处不在的千禧年主义特征。

接着，我们将审视比特币可能的未来。基于几种未来发展的情景，我们将推演比特币如何可能与之契合。根据我们在本书过程中所收集的见解，我们可以假设性地构建比特币在不同环境下可能的发展模式。

在最后一章，我们将超越人类视角，尝试从另一个角度来看待比特币。我们会提出一个疑问：比特币是否可能是活的？我们将通过将其与普通生物生命的评判标准进行比较来探讨。我们会发现，出人意料的是，它竟是一个独立存在的、显示出生命迹象的超有机体。
