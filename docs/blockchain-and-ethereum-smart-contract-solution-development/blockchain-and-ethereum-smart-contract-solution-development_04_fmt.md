# 第 1 章：区块链的商业与经济动机

在我们的社会中，很难想象有哪个业务流程不是由计算机硬件、软件和网络的组合来支持、中介或完全自动化的。在本书中，计算机硬件、软件、网络以及数据与流程分析、用户体验设计和系统设计等相关学科统称为信息技术。已有研究证实信息技术对商业生产力具有广泛而积极的影响（Loveman, 1993; Brynjolfsson, 1991）。在本章中，我们首先列举自 20 世纪 60 年代中期以来推动商业生产力提升的信息技术创新，目的是理解那些尚未受到影响的商业方面，因为这些正是区块链有潜力发挥作用的领域。
