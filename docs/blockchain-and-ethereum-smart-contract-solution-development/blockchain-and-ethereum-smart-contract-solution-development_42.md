# 第 4 章 区块链商业应用

区块链在医疗领域的应用重点在于：减少与“三 R”相关的经济低效问题、提升运营透明度，以及赋予消费者数据掌控权。这些应用均不旨在减少或消除第三方管理者的角色——因为若造成此类颠覆，很可能阻碍其被成功采纳。事实上，许多应用正由第三方服务商联盟主导。Hashed Health（[`hashedhealth.com/`](https://hashedhealth.com/)）已成为凝聚医疗行业利益相关方、共同探索分布式账本应用的重要催化剂。区块链在医疗领域的应用包括以下方面：

- **理赔处理**
  通过智能合约提供理赔处理状态透明度及处理逻辑的透明化，对于减少返工、对账及欺诈导致的经济低效问题至关重要。这还能缩短所有理赔的结算时间，尤其是目前可能需要数月才能结案的 20%的理赔。对于健康保险公司而言，理赔处理代表着庞大的遗留运营与技术体系。该领域的任何颠覆都伴随着风险。

- **医疗服务提供者资质认证**
  临床服务提供者必须向医院系统提交资质证明，方能获得在特定医院系统内的执业权限。资质认证在医疗生态系统中造成了大量重复性工作。区块链资质认证应用将减少此类重复工作以及因资质维护不当导致的返工所带来的经济低效问题，同时具备减少欺诈的潜力。此类应用还可能被集成到当前遗留技术环境中，从而拥有更高的成功采纳潜力。

- **预授权处理**
  基于区块链的预授权处理是另一个可集成至医疗遗留技术体系的应用场景。通过智能合约实现的自动化能够减少经济低效问题和医疗差错。基于区块链的系统还能保护隐私，因为患者数据无需在服务提供者与第三方管理者之外进行共享。
