# 第 4 章 区块链业务应用

## • 临床试验

美国的临床试验效率低下，原因在于难以招募到合适的患者、难以在试验过程中跟踪患者，并且确保向`FDA`报告时数据来源的准确性既缓慢又依赖于人工操作。这个用例是另一个适合基于区块链的应用场景（`Zhuang et al., 2020`）。

## • 患者数据主权

此外还有一些区块链用例，如果我们能将数据交到患者手中，由患者掌控并成为数据的主权者，我们就可以确保这些数据在医疗系统的适当节点可用，这同样会带来效率提升。

尽管在医疗领域有许多潜在的区块链应用，但医疗行业在技术采用方面通常落后于其他行业。历史上，在医疗行业成功实施技术项目一直颇具挑战性，因此我们建议优先考虑那些能够集成到现有遗留技术和运营体系中的用例。这也是一个尽管消费者普遍感到沮丧，但仍然尊重现有行业巨头权力的领域。所以我们的建议是谨慎而深思熟虑地推进。
