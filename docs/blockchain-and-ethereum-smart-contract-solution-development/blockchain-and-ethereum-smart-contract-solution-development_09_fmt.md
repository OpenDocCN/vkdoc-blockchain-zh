## 第一章 区块链的商业与经济动机

在本节中，我们看到了价值交换如何成为我们经济的基础。为了使经济能够在互不信任的企业之间运转，第三方机构和政府法规便层出不穷。第三方机构在未增加相应价值的情况下收取费用，拖慢流程，并成为拥有过度影响力的权力中心。我们还阐述了数据的作用，特别是价值交换中互不信任的各方开始创建的有关交换的专有数据。

在下一节中，我们将深入探讨当前*运作方式*所导致的经济低效问题。在本章结尾，我们将从宏观层面描述区块链为何以及如何能够减少当前的经济低效问题。

### 当前的经济低效问题

本章开篇我们便断言，催生了 ERP 软件和互联网的技术创新弧线为企业带来了非凡的生产力提升。这些提升是通过标准化、集中化、数字化和全球化实现的。数据的捕获和利用在这些提升中发挥了关键作用。我们当前的经济低效问题源于这样一个事实：当前的技术创新并未解决价值交换各方之间缺乏信任的问题。我们的许多经济低效问题直接源于我们为在缺乏信任的商业环境中运作而制定的流程、机构、协议及其他机制。我们识别出七种经济低效来源，汇总于表 1-2，并在本节中进行详细阐述。

**表 1-2.** 七种经济低效来源

| 经济低效类别 | 简要描述 |
| --- | --- |
| 1. 交易结算时间 | 在当前商业环境中，交易可能需要数天、数周甚至数月才能结算。 |
| 2. 支付给第三方的费用 | 银行和信用卡处理机构等金融机构会收取处理货币兑换交易的手续费。 |
| 3. 数据相关的重复工作、返工与对账工作（三个 R） | 由于业务涉及的各方均维护专有数据，他们在创建、维护和对账这些数据时会产生不必要的成本。 |
| 4. 政府法规及非政府组织规则的约束 | 表面上旨在提供帮助的政府法规，反而导致不必要的成本增加。在某些行业，企业自愿遵守非政府行业团体强加的规则，这些规则也同样会增加成本并扼杀创新。 |
| 5. 欺诈行为 | 当前价值交换系统中的复杂性以及透明度的缺失为欺诈提供了可乘之机。 |
| 6. 隐私权衡 | 要在当前系统中进行交易，个人必须提供可识别身份的数据，这损害了隐私。 |
