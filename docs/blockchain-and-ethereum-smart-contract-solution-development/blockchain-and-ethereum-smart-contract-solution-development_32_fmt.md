# 代币分类

## 原生代币与非原生代币

第一种分类将代币划分为 `原生代币` 与 `非原生代币`。原生代币由区块链网络的共识机制生成，它们本身就是数字资产的代表形式。其价值由市场驱动，基于认知和情绪，不一定与任何其他实体或数字资产挂钩。

### 原生代币的估值

目前有人提出借鉴公司估值的方法来对原生代币进行估值。总体而言，存在两种估值方法论：`技术分析` 与 `基本面分析`。简单来说，`技术分析` 方法依赖趋势、历史估值以及市场对事件的历史反应来评估公司（或公司股票）。`基本面分析` 方法则依赖每股收益、收入、利润率、增长率以及人均收入等指标来评估公司。需要注意的是，这些方法并非精确科学，不同的人可能使用相同的基础指标和趋势得出不同的估值结果。

运用 `基本面` 方法论，原生代币（加密货币）的估值基于交易数量、用户数量以及执行共识机制的节点集中度/分散度等指标。我们认为，这些指标与互联网公司在互联网泡沫破裂前早期的估值指标相似。我们建议应对此类估值方法持一定程度的怀疑态度。由于交易的匿名性，这些方法所依赖的许多指标很容易被操纵。加密资产或加密货币的估值仍处于非常初级的阶段。

## 非原生代币

非原生代币来源于原生代币。非原生代币映射到实体资产或其他数字资产，其价值源于所映射的资产。当开发企业级区块链应用时，我们关注的是将要映射到资产或交换支付货币的非原生代币。非原生支付代币的一个具体例子是 `稳定币`。`稳定币` 的价值映射到一种或多种国家法定货币。
